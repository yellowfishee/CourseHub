## 内容

### 一、集合（Set）

**什么是集合？**

一组**不重复**的元素，没有顺序。

**类比**：
```
集合 = 一袋玻璃珠
每个颜色的球只有一个（不重复）
倒出来顺序是随机的（无序）
```

**创建集合**：

```python
# 方式1：直接创建
fruits = {"apple", "banana", "orange"}

# 方式2：从列表创建
fruits = set(["apple", "banana", "apple", "orange"])
print(fruits)  # {"apple", "banana", "orange"} 自动去重！

# 方式3：空集合
empty = set()  # 注意：{} 是空字典，不是空集合！
```

**集合的特点**：
- 元素不重复
- 元素必须不可变（int、str、tuple可以，list不行）
- 无序（不能用索引访问）

---

### 二、集合的基本操作

**添加元素**：

```python
fruits = {"apple", "banana"}
fruits.add("orange")  # 添加一个
fruits.update(["grape", "kiwi"])  # 添加多个

print(fruits)
```

**删除元素**：

```python
fruits = {"apple", "banana", "orange"}
fruits.remove("apple")   # 删除（不存在会报错）
fruits.discard("kiwi")   # 删除（不存在不报错）
fruits.pop()             # 随机删除一个
```

**检查是否存在**：

```python
fruits = {"apple", "banana"}
print("apple" in fruits)   # True
print("orange" in fruits)  # False
```

### 三、集合运算

**交集（&）**：两个集合共有的元素

```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a & b)  # {3, 4}
print(a.intersection(b))  # 同上
```

**并集（|）**：两个集合所有元素（去重）

```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a | b)  # {1, 2, 3, 4, 5, 6}
print(a.union(b))  # 同上
```

**差集（-）**：在a但不在b的元素

```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a - b)  # {1, 2}
print(a.difference(b))  # 同上
```

**对称差集（^）**：只在a或只在b的元素

```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a ^ b)  # {1, 2, 5, 6}
```

**可视化**：
```
交集（&）：    并集（|）：    差集（-）：
  ┌───┐        ┌───┐        ┌───┐
  │ a │        │ a │        │ a │
  └─┬─┘        └─┬─┘        └─┬─┘
    │∩│          │∪│          │-│
  ┌─┴─┐        ┌─┴─┐        ┌─┴─┐
  │ b │        │ b │        │ b │
  └───┘        └───┘        └───┘
```

### 四、例题：去重

**问题**：输入一串数字，去掉重复的，输出唯一值

**输入**：`1 2 3 2 1 4 5 4`

**输出**：`1 2 3 4 5`

**代码**：

```python
nums = [1, 2, 3, 2, 1, 4, 5, 4]
unique = set(nums)  # 直接转成集合，自动去重
print(unique)  # {1, 2, 3, 4, 5}

# 如果要保持顺序
nums = [1, 2, 3, 2, 1, 4, 5, 4]
unique = []
for x in nums:
    if x not in unique:
        unique.append(x)
print(unique)  # [1, 2, 3, 4, 5]
```

**Python简洁写法**：

```python
nums = [1, 2, 3, 2, 1, 4, 5, 4]
unique = list(dict.fromkeys(nums))  # 保持顺序去重
print(unique)  # [1, 2, 3, 4, 5]
```

### 五、元组（Tuple）

**什么是元组？**

**不可变**的列表，创建后不能修改。

**类比**：
```
列表 = 白板（可以擦了重写）
元组 = 刻在石头上（一旦刻好就不能改）
```

**创建元组**：

```python
# 方式1：直接创建
coords = (3, 5)

# 方式2：从列表创建
coords = tuple([3, 5])

# 注意：单个元素要加逗号
single = (1,)  # 这是元组
not_tuple = (1)  # 这只是数字1
```

**访问元素**：

```python
coords = (3, 5)
print(coords[0])  # 3
print(coords[1])  # 5
print(coords[-1]) # 5（最后一个）
```

**元组不能修改**：

```python
coords = (3, 5)
coords[0] = 10  # 错误！TypeError
```

### 六、元组的应用场景

**场景1：函数返回多个值**

```python
def get_min_max(nums):
    return min(nums), max(nums)

result = get_min_max([1, 5, 3, 2])
print(result)  # (1, 5)

# 解包
min_val, max_val = get_min_max([1, 5, 3, 2])
print(min_val, max_val)  # 1 5
```

**场景2：字典的键**

```python
# 列表不能做键，但元组可以
locations = {}
locations[(0, 0)] = "起点"
locations[(1, 0)] = "终点"

print(locations[(0, 0)])  # 起点
```

**场景3：安全传递数据**

```python
def process(data):
    # data是元组，不会意外修改
    pass

coords = (3, 5)
process(coords)  # coords不会被修改
```

### 七、集合 vs 元组 vs 列表

| 特点 | 列表 | 元组 | 集合 |
|------|------|------|------|
| 可变性 | 可变 | 不可变 | 可变 |
| 重复性 | 可重复 | 可重复 | 不重复 |
| 有序性 | 有序 | 有序 | 无序 |
| 索引访问 | 可以 | 可以 | 不可以 |
| 做字典键 | 不行 | 可以 | 不行 |

**何时用哪个？**
- 需要修改 → 列表
- 不需要修改，保护数据 → 元组
- 需要去重 → 集合
- 需要快速查找 → 集合

---

### 八、常见错误

**错误1：用{}创建空集合**

```python
empty = {}  # 这是空字典！
empty = set()  # 这才是空集合
```

**错误2：修改元组**

```python
coords = (3, 5)
coords[0] = 10  # TypeError!
```

**正确**：创建新元组

```python
coords = (3, 5)
coords = (10, coords[1])  # 创建新元组
```

**错误3：集合包含可变元素**

```python
bad_set = {[1, 2], [3, 4]}  # TypeError!
```

**正确**：用元组代替列表

```python
good_set = {(1, 2), (3, 4)}  # OK
```

## 总结

| 概念 | 要点 |
|------|------|
| 集合 | 不重复、无序、可变 |
| 交集 & | 共有的元素 |
| 并集 \| | 所有元素 |
| 差集 - | 在a不在b |
| 元组 | 不可变的列表 |

**记住**：
- 集合自动去重
- 元组创建后不能修改
- 用元组保护数据不被意外修改
