集合与元组

不重复与不可变

🎭

集合

不重复、无序

🔒

元组

不可变的列表

🎭 集合运算

a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

a & b # 交集 {3, 4}
a | b # 并集 {1,2,3,4,5,6}
a - b # 差集 {1, 2}

🔒 元组特点

coords = (3, 5)

coords[0] # 3
coords[0] = 10 # TypeError! 不能修改

元组创建后不能修改，可以保护数据
