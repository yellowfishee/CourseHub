方法代码作用

⚠️ 字符串不可变！

❌ 常见错误

索引越界

nums = [1, 2, 3]
nums[3] # IndexError!

字符串不能修改

s = "Hello"
s[0] = 'h' # TypeError!

📝 练习：列表去重

nums = [1, 2, 3, 2, 1, 4]

# 方法：转成集合
unique = list(set(nums))
print(unique) # [1, 2, 3, 4]
