## 内容

### 一、知识回顾

**列表（List）**

列表就像一排编好号的储物柜，每个"柜子"存一个数据，编号从 0 开始。

```python
nums = [1, 3, 5, 7, 9]  # 创建列表
print(nums[0])  # 1（第一个元素）
print(nums[-1])  # 9（最后一个元素）
print(len(nums))  # 5（列表长度）
```

**常用操作**：

| 操作 | 代码 | 作用 |
|------|------|------|
| 取值 | `nums[0]` | 取第1个 |
| 修改 | `nums[0] = 10` | 改第1个 |
| 添加 | `nums.append(4)` | 添加到末尾 |
| 删除 | `nums.pop(0)` | 删除第1个 |
| 长度 | `len(nums)` | 元素个数 |
| 最大 | `max(nums)` | 最大值 |
| 最小 | `min(nums)` | 最小值 |
| 求和 | `sum(nums)` | 总和 |
| 排序 | `nums.sort()` | 从小到大排序 |

---

**字符串（String）**

字符串是一串文字，每个字符就像一颗珠子，索引也从 0 开始。

```python
s = "Hello"
print(s[0])  # 'H'（第一个字符）
print(s[-1])  # 'o'（最后一个字符）
print(len(s))  # 5（字符个数）
print(s.count('l'))  # 2（'l'出现的次数）
```

**常用操作**：

| 操作 | 代码 | 作用 |
|------|------|------|
| 长度 | `len(s)` | 字符个数 |
| 查找 | `s.find("l")` | 找到第一个'l'的位置 |
| 计数 | `s.count("l")` | 'l'出现的次数 |
| 替换 | `s.replace("l", "L")` | 把'l'换成'L' |
| 分割 | `s.split(" ")` | 用空格分割成列表 |
| 连接 | `" ".join(list)` | 用空格连接列表 |
| 大写 | `s.upper()` | 全部大写 |
| 小写 | `s.lower()` | 全部小写 |
| 去空格 | `s.strip()` | 去掉两端空格 |

---

**列表 vs 字符串**

| 特点 | 列表 | 字符串 |
|------|------|------|
| 创建 | `[1, 2, 3]` | `"hello"` |
| 索引 | 从 0 开始 | 从 0 开始 |
| 切片 | 支持 | 支持 |
| 可修改 | 可以 | 不可以 |
| 存储内容 | 任意类型 | 只能是字符 |

### 二、常见错误分析

**错误1：索引越界**

```python
nums = [1, 2, 3]
print(nums[3])  # 错误！索引从0开始，最大是nums[2]
```

**正确**：
```python
nums = [1, 2, 3]
print(nums[2])  # 3（第三个元素）
# 或使用负索引
print(nums[-1])  # 3（最后一个元素）
```

---

**错误2：字符串不能修改单个字符**

```python
s = "Hello"
s[0] = 'h'  # 错误！字符串是不可变的
```

**正确**：
```python
s = "Hello"
# 方法1：转成列表修改
chars = list(s)
chars[0] = 'h'
s = "".join(chars)  # "hello"

# 方法2：用切片替换
s = 'h' + s[1:]  # "hello"
```

---

**错误3：append() 返回 None**

```python
nums = [1, 2, 3]
nums = nums.append(4)  # 错误！append() 返回 None
print(nums)  # None
```

**正确**：
```python
nums = [1, 2, 3]
nums.append(4)  # append() 直接修改原列表
print(nums)  # [1, 2, 3, 4]
```

---

**错误4：split() 的结果没保存**

```python
s = "1 2 3 4 5"
s.split()  # 没保存结果！
print(s)  # 还是 "1 2 3 4 5"（字符串没变）
```

**正确**：
```python
s = "1 2 3 4 5"
nums = s.split()  # 保存结果
print(nums)  # ['1', '2', '3', '4', '5']
# 如果需要数字，再转换
nums = list(map(int, nums))  # [1, 2, 3, 4, 5]
```

---

**错误5：忘记类型转换**

```python
nums = input().split()  # 得到的是字符串列表！
print(nums[0] + nums[1])  # "1" + "2" = "12"（字符串拼接）
```

**正确**：
```python
nums = list(map(int, input().split()))  # 转成整数列表
print(nums[0] + nums[1])  # 1 + 2 = 3（数值相加）
```

---

**错误6：遍历时修改列表**

```python
nums = [1, 2, 3, 4, 5]
for x in nums:
    if x % 2 == 0:
        nums.remove(x)  # 危险！遍历时修改列表
print(nums)  # [1, 3, 5] 可能不正确
```

**正确方法1：创建新列表**
```python
nums = [1, 2, 3, 4, 5]
new_nums = [x for x in nums if x % 2 != 0]
print(new_nums)  # [1, 3, 5]
```

**正确方法2：倒序遍历**
```python
nums = [1, 2, 3, 4, 5]
for i in range(len(nums) - 1, -1, -1):
    if nums[i] % 2 == 0:
        nums.pop(i)
print(nums)  # [1, 3, 5]
```

### 三、练习题

**练习1：列表求和与平均值**

输入 n 个数字，输出它们的和与平均值

```python
nums = list(map(int, input().split()))
total = sum(nums)
avg = total / len(nums)
print(f"总和：{total}")
print(f"平均值：{avg}")
```

---

**练习2：找最大值和最小值**

输入 n 个数字，输出最大值和最小值

```python
nums = list(map(int, input().split()))
print(f"最大值：{max(nums)}")
print(f"最小值：{min(nums)}")
```

**手动实现**：
```python
nums = list(map(int, input().split()))
max_val = nums[0]
min_val = nums[0]
for x in nums:
    if x > max_val:
        max_val = x
    if x < min_val:
        min_val = x
print(f"最大值：{max_val}")
print(f"最小值：{min_val}")
```

**练习3：统计字符出现次数**

输入一个字符串，统计每个元音字母出现的次数

```python
s = input().lower()
vowels = 'aeiou'
for v in vowels:
    count = s.count(v)
    print(f"{v}: {count}次")
```

---

**练习4：字符串反转**

输入一个字符串，输出它的反转

```python
s = input()
print(s[::-1])  # 切片方法
```

**手动实现**：
```python
s = input()
rev = ""
for ch in s:
    rev = ch + rev
print(rev)
```

**练习5：删除列表中的奇数**

输入 n 个数字，删除所有奇数，输出剩余的偶数

```python
nums = list(map(int, input().split()))
evens = [x for x in nums if x % 2 == 0]
print(evens)
```

**练习6：判断回文字符串**

输入一个字符串，判断它是否是回文（正读反读一样）

```python
s = input()
if s == s[::-1]:
    print("是回文")
else:
    print("不是回文")
```

---

**练习7：列表去重**

输入 n 个数字，去除重复的数字后输出

```python
nums = list(map(int, input().split()))
unique = list(set(nums))  # 用集合去重
unique.sort()  # 排序（可选）
print(unique)
```

**保持顺序的去重**：
```python
nums = list(map(int, input().split()))
unique = []
for x in nums:
    if x not in unique:
        unique.append(x)
print(unique)
```

### 四、调试技巧

**1. 用 print() 查看中间结果**

```python
nums = list(map(int, input().split()))
print(f"输入的列表：{nums}")  # 调试输出
print(f"列表长度：{len(nums)}")
```

**2. 检查索引范围**

```python
nums = [1, 2, 3]
print(f"有效索引范围：0 到 {len(nums)-1}")
```

**3. 检查类型**

```python
x = input()
print(f"类型：{type(x)}")  # <class 'str'>
```

## 总结

| 知识点 | 要点 |
|--------|------|
| 列表索引 | 从 0 开始，最大是 len-1 |
| 字符串索引 | 与列表相同，从 0 开始 |
| 字符串不可变 | 不能直接修改单个字符 |
| append() | 直接修改原列表，返回 None |
| split() | 返回新列表，原字符串不变 |
| 类型转换 | input() 得到字符串，需用 int() 转换 |
| 切片 | nums[1:4] 取索引 1 到 3，不包括 4 |

**记住**：列表和字符串都从索引 0 开始，但字符串不能直接修改！
