## 内容

### 一、引入：什么是循环？

**生活例子**：
```
运动员跑操场：
一圈 → 一圈 → 一圈 → ... → 直到跑了10圈
```

**计算机中的循环**：
```
print("Hello!")  → 第1次
print("Hello!")  → 第2次
print("Hello!")  → 第3次
...一直重复，直到完成任务
```

**类比**：
```
循环 = 跑圈
跑10圈 = 循环执行10次
```

---

### 二、for循环的基本结构

**Python 代码**：

```python
for i in range(10):
    print("Hello!")
```

**拆解**：
```
for 变量 in 范围:
    循环体（要重复执行的代码）
```

| 部分 | 例子 | 含义 |
|------|------|------|
| 变量 | i | 循环变量，记录当前次数 |
| 范围 | range(10) | 循环 10 次（0~9） |
| 循环体 | print("Hello!") | 每次循环做的事 |

**运行结果**：
```
Hello!
Hello!
Hello!
Hello!
Hello!
Hello!
Hello!
Hello!
Hello!
Hello!
```

### 三、range() 函数详解

**range() 是什么？**

range() 产生一个"范围"，告诉循环从哪开始、到哪结束。

**三种用法**：

| 写法 | 含义 | 产生的数 |
|------|------|---------|
| range(5) | 0~4（共5个） | 0, 1, 2, 3, 4 |
| range(1, 5) | 1~4（不包括5） | 1, 2, 3, 4 |
| range(1, 10, 2) | 1~9，步长为2 | 1, 3, 5, 7, 9 |

**类比**：
```
range(5) = 从0开始，数5个数
range(1, 5) = 从1开始，数到4（不包括5）
range(1, 10, 2) = 从1开始，每次跳2个数
```

**实验**：

```python
# 实验1：默认从0开始
for i in range(5):
    print(i)
# 输出：0 1 2 3 4

# 实验2：指定起始值
for i in range(1, 5):
    print(i)
# 输出：1 2 3 4

# 实验3：带步长
for i in range(1, 10, 2):
    print(i)
# 输出：1 3 5 7 9
```

### 四、循环变量 i

**i 是什么？**

i 是一个"计数器"，记录循环执行到第几次了。

```python
for i in range(1, 6):
    print(f"第 {i} 次")
```

**输出**：
```
第 1 次
第 2 次
第 3 次
第 4 次
第 5 次
```

> ⚠️ **注意**：Python 的 range(1, 6) 包含 1~5，不包括 6！

### 五、例题：求 1+2+3+...+100

**问题**：计算 1 + 2 + 3 + ... + 100 的和

**思路**：
```
1. 准备一个"盒子"存总和，初始为 0
2. 每次循环，把当前的 i 加到总和中
3. 循环结束后，输出总和
```

**代码**：

```python
total = 0
for i in range(1, 101):  # 注意：101 不包括在内！
    total = total + i
print(total)
```

**结果**：5050

**类比**：
```
total = 一个空盒子
每次循环，往盒子里放一个数字
最后看看盒子里有多少
```

> ⚠️ **Python最佳实践**：不要用 `sum` 做变量名，因为 `sum()` 是Python内置函数！

### 六、循环的三个关键

1. **起始值**：从哪开始？
2. **结束值**：到什么为止？
3. **步长**：每次增加多少？

```
           开始
             ↓
    ┌─────────────────┐
    │  i = 起始值      │  ← range() 设置
    └─────────────────┘
             ↓
    ┌─────────────────┐
    │  i < 结束值 ?    │  ← 判断条件
    └─────────────────┘
         ↓ Yes
    ┌─────────────────┐
    │  执行循环体      │
    └─────────────────┘
             ↓
    ┌─────────────────┐
    │  i += 步长       │  ← 自动增加
    └─────────────────┘
             ↓
         回到条件判断
```

### 七、循环嵌套

**问题**：输出 3×3 的星号方阵

```python
for row in range(3):
    for col in range(3):
        print("*", end=" ")
    print()  # 换行
```

**输出**：
```
* * *
* * *
* * *
```

**解释**：
```
外层循环（row）：控制行数，循环3次
内层循环（col）：控制每行的星星，循环3次
print("*", end=" ")：打印星星但不换行
print()：每行结束后换行
```

### 八、常见错误

**错误 1**：结束值写错

```python
# 想：循环 1~100
for i in range(1, 100):   # 错！只会循环到 99
for i in range(1, 101):   # 对！会循环到 100
```

**错误 2**：忘记冒号

```python
for i in range(5)          # 错！少了冒号
    print(i)

for i in range(5):         # 对！冒号不能少
    print(i)
```

**错误 3**：缩进错误

```python
for i in range(5):
print(i)   # 错！缩进必须对齐

for i in range(5):
    print(i)  # 对！缩进4个空格
```

> ⚠️ **提示**：Python 用缩进（空格）来区分代码块，必须保持一致！

### 九、Python vs C++ 循环对比

| 特点 | Python | C++ |
|------|--------|-----|
| 循环语法 | for i in range(5): | for (int i=0; i<5; i++) |
| 范围 | range() 函数 | 直接写条件 |
| 步长 | range(start, end, step) | i += step |
| 缩进 | 必须用缩进 | 用大括号 {} |

**Python 的优势**：
- 写法更简洁
- range() 自动处理范围
- 不容易写错条件

## 总结

| 概念 | Python 写法 | 作用 |
|------|-------------|------|
| for循环 | for i in range(5): | 循环执行5次 |
| range(n) | range(5) | 产生 0~4 的范围 |
| range(start, end) | range(1, 5) | 产生 1~4 的范围 |
| range(start, end, step) | range(0, 10, 2) | 产生 0,2,4,6,8 |
| 循环变量 | i | 记录当前循环次数 |

**记住**：range(1, 101) 才能循环到 100！Python 的 range 不包括结束值！
