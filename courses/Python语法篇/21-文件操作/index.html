🤔 为什么需要文件？

❌ 没有文件

score = 100
print(score) # 100

# 程序结束...
# 数据消失了！

程序关闭，数据就没了

✓ 有文件

# 保存分数到文件
f = open("score.txt", "w")
f.write("100")
f.close()

# 下次还能读取！

数据永久保存

文件 = 数据的永久存储

📂 open() 函数

打开文件的第一步

f = open("文件名", "模式")
r

读取

read

w

写入

write（覆盖）

a

追加

append（添加）

✍️ 写入文件

# 1. 打开文件（写入模式）
f = open("hello.txt", "w")

# 2. 写入内容
f.write("Hello, World!\n")
f.write("你好，世界！")

# 3. 关闭文件（重要！）
f.close()

1. open

打开文件

2. write

写入内容

3. close

关闭文件

⚠️ "w"模式会覆盖原有内容！

📖 读取文件

方式1：read()

f = open("hello.txt", "r")
content = f.read()
print(content)
f.close()

读取全部内容

方式2：逐行读取

f = open("hello.txt", "r")
for line in f:
    print(line)
f.close()

每次读取一行

readline() 读取一行

readlines() 返回所有行的列表

🔒 with 语句

自动关闭文件，不用担心忘记close()

❌ 手动关闭

f = open("test.txt", "r")
content = f.read()
# 忘记 close()？
# 可能导致问题！

✓ with 自动关闭

with open("test.txt", "r") as f:
    content = f.read()
# 自动关闭，安全！
推荐：始终使用 with 语句！

📋 文件模式总结

模式 说明 特点
r 读取（默认） 文件不存在则报错
w 写入 覆盖原有内容
a 追加 在文件末尾添加
r+ 读写 文件不存在则报错
w+ 读写 创建新文件

📝 例题：保存成绩

把学生成绩保存到文件

# 写入成绩
scores = [("小明", 90), ("小红", 85)]

with open("scores.txt", "w") as f:
    for name, score in scores:
        f.write(f"{name}: {score}\n")

# 读取成绩
with open("scores.txt", "r") as f:
    for line in f:
        print(line.strip())

输出：

小明: 90

小红: 85
