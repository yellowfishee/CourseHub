## 内容

### 一、引入：为什么要操作文件？

**问题**：程序的数据怎么保存？

**答案**：存到文件里！

**类比**：
```
变量 = 写在便利贴上（程序结束就没了）
文件 = 写在本子上（永久保存）
```

---

### 二、打开和关闭文件

**基本流程**：
```
1. 打开文件
2. 读写操作
3. 关闭文件
```

**传统写法**：

```python
f = open("data.txt", "r")  # 打开文件
content = f.read()         # 读取内容
f.close()                  # 关闭文件（容易忘记！）
```

**推荐写法（with语句）**：

```python
with open("data.txt", "r") as f:
    content = f.read()
# 自动关闭文件，即使出错也会关闭
```

> ⚠️ **推荐用 `with` 语句**：自动处理文件关闭，避免资源泄露！

---

### 三、文件打开模式

| 模式 | 含义 | 说明 |
|------|------|------|
| "r" | 读取 | 文件必须存在 |
| "w" | 写入 | 文件不存在会创建，存在会覆盖 |
| "a" | 追加 | 写入到文件末尾 |
| "x" | 创建 | 文件已存在会报错 |

**例子**：

```python
# 读取文件
with open("data.txt", "r") as f:
    content = f.read()

# 写入文件（覆盖）
with open("data.txt", "w") as f:
    f.write("Hello World")

# 追加写入
with open("data.txt", "a") as f:
    f.write("\nNew line")
```

### 四、读取文件

**方法1：read() - 读取全部**

```python
with open("data.txt", "r") as f:
    content = f.read()  # 读取全部内容，返回字符串
    print(content)
```

**方法2：readline() - 读取一行**

```python
with open("data.txt", "r") as f:
    line = f.readline()  # 读取一行
    print(line)
```

**方法3：readlines() - 读取所有行**

```python
with open("data.txt", "r") as f:
    lines = f.readlines()  # 返回列表，每行一个元素
    for line in lines:
        print(line.strip())  # strip()去掉换行符
```

**方法4：遍历文件（推荐）**

```python
with open("data.txt", "r") as f:
    for line in f:  # 直接遍历文件对象
        print(line.strip())
```

---

### 五、写入文件

**写入字符串**：

```python
with open("output.txt", "w") as f:
    f.write("Hello\n")
    f.write("World\n")
```

**写入多行**：

```python
lines = ["Line 1\n", "Line 2\n", "Line 3\n"]
with open("output.txt", "w") as f:
    f.writelines(lines)
```

> ⚠️ **注意**：`write()` 不会自动加换行符，需要手动加 `\n`

### 六、例题：统计文件行数

**问题**：读取文件，统计有多少行

**代码**：

```python
count = 0
with open("data.txt", "r") as f:
    for line in f:
        count += 1

print(f"文件共有 {count} 行")
```

**Python简洁写法**：

```python
with open("data.txt", "r") as f:
    count = len(f.readlines())

print(f"文件共有 {count} 行")
```

---

### 七、例题：复制文件

**问题**：将一个文件的内容复制到另一个文件

**代码**：

```python
with open("source.txt", "r") as src:
    content = src.read()

with open("dest.txt", "w") as dst:
    dst.write(content)

print("复制完成")
```

### 八、文件路径

**相对路径**：从当前目录开始

```python
with open("data.txt", "r") as f:      # 当前目录
with open("data/info.txt", "r") as f: # 子目录
with open("../data.txt", "r") as f:   # 上级目录
```

**绝对路径**：从根目录开始

```python
with open("/Users/mac/data.txt", "r") as f:
    pass
```

**安全处理路径**：

```python
import os

# 拼接路径（跨平台兼容）
file_path = os.path.join("data", "info.txt")
# Windows: data\info.txt
# Mac/Linux: data/info.txt

with open(file_path, "r") as f:
    pass
```

### 九、常见错误

**错误1：文件不存在**

```python
with open("not_exist.txt", "r") as f:
    pass
# FileNotFoundError!
```

**解决**：先检查或用异常处理

```python
import os

if os.path.exists("data.txt"):
    with open("data.txt", "r") as f:
        pass
else:
    print("文件不存在")
```

**错误2：忘记关闭文件**

```python
f = open("data.txt", "r")
content = f.read()
# 忘记 f.close()
```

**解决**：用 `with` 语句

**错误3：写入模式覆盖文件**

```python
with open("data.txt", "w") as f:  # "w"会覆盖！
    f.write("New content")
# 原来的内容没了！
```

**解决**：用 `"a"` 追加模式

## 总结

| 概念 | 要点 |
|------|------|
| with语句 | 自动关闭文件 |
| "r"模式 | 读取 |
| "w"模式 | 写入（覆盖） |
| "a"模式 | 追加 |
| read() | 读取全部 |
| readline() | 读取一行 |
| readlines() | 读取所有行（列表） |

**记住**：
- 用 `with` 语句操作文件
- "w" 模式会覆盖，"a" 模式追加
- 文件不存在用异常处理
