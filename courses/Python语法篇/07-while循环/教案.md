## 内容

### 一、引入：当...的时候

**生活例子**：
```
while 饿着:
    吃一口饭
    
翻译：当（while）还饿着的时候，就继续吃
```

**与 for 的区别**：
```
for：知道要做多少次（跑10圈）
while：不知道要做多少次，只知道什么时候停下（吃饱为止）
```

---

### 二、while循环的基本结构

**Python 代码**：

```python
count = 1
while count <= 5:
    print(f"第 {count} 次")
    count += 1
```

**输出**：
```
第 1 次
第 2 次
第 3 次
第 4 次
第 5 次
```

**拆解**：
```
while 条件:
    循环体（条件成立时执行）
```

| 部分 | 含义 |
|------|------|
| while | "当...的时候" |
| 条件 | 判断是否继续循环 |
| 循环体 | 条件成立时执行的代码 |

**类比**：
```
while 就像一个"门卫"
每次循环前检查条件
条件成立 → 放行，继续循环
条件不成立 → 关门，结束循环
```

---

### 三、while vs for：什么时候用哪个？

| 场景 | 推荐使用 | 原因 |
|------|---------|------|
| 已知循环次数 | for | range() 很方便 |
| 未知循环次数 | while | 更灵活 |
| 读取输入直到结束 | while | 不知道要读多少次 |
| 游戏主循环 | while | 玩家没退出就继续 |

**例子1：已知次数 → 用 for**

```python
# 输出 1~10
for i in range(1, 11):
    print(i)
```

**例子2：未知次数 → 用 while**

```python
# 猜数字，直到猜对为止
answer = 7
guess = int(input("猜一个数字："))
while guess != answer:
    print("猜错了，再试一次！")
    guess = int(input("猜一个数字："))
print("猜对了！")
```

### 四、例题：计算阶乘

**问题**：用 while 循环计算 5! = 5×4×3×2×1

**代码**：

```python
n = 5
result = 1
while n > 0:
    result = result * n
    n -= 1
print(result)  # 输出 120
```

**执行过程**：
```
第1次：n=5, result=1×5=5, n变成4
第2次：n=4, result=5×4=20, n变成3
第3次：n=3, result=20×3=60, n变成2
第4次：n=2, result=60×2=120, n变成1
第5次：n=1, result=120×1=120, n变成0
n=0, 条件不成立，退出循环
```

### 五、死循环问题

**什么是死循环？**

条件永远为真，循环永远不会停止。

```python
# 死循环！
while True:
    print("我停不下来啦！")
```

**为什么危险？**
- 程序卡住不动
- 电脑 CPU 飙升
- 需要强制关闭

**如何避免？**

确保循环条件最终会变成 False：

```python
count = 1
while count <= 5:
    print(count)
    count += 1  # 关键：让 count 变大，最终条件不成立
```

**什么时候需要死循环？**

有些场景故意用死循环（比如游戏主循环）：

```python
while True:
    command = input("输入命令（输入 quit 退出）：")
    if command == "quit":
        break  # 用 break 跳出循环
    print(f"执行命令：{command}")
```

### 六、break 和 continue

**break：跳出整个循环**

```python
for i in range(10):
    if i == 5:
        break  # i=5 时直接跳出循环
    print(i)
# 输出：0 1 2 3 4
```

**continue：跳过这一次，继续下一次**

```python
for i in range(10):
    if i == 5:
        continue  # i=5 时跳过，继续下一次
    print(i)
# 输出：0 1 2 3 4 6 7 8 9（没有5）
```

**类比**：
```
break = 提前下课（整个课都不上了）
continue = 去厕所（这节课跳过，下节课继续）
```

### 七、例题：猜数字游戏

**问题**：让用户猜数字，猜对为止，最多猜5次

**代码**：

```python
answer = 42
attempts = 0

while attempts < 5:
    guess = int(input("猜一个数字（1-100）："))
    attempts += 1
    
    if guess == answer:
        print(f"恭喜！你用了 {attempts} 次猜对了！")
        break
    elif guess < answer:
        print("太小了！")
    else:
        print("太大了！")
else:
    print(f"游戏结束！答案是 {answer}")
```

**注意**：while 的 else 分支在循环正常结束时执行（不是 break 跳出的情况）。

### 八、常见错误

**错误 1**：忘记更新条件

```python
count = 1
while count <= 5:
    print(count)
    # 忘了 count += 1 → 死循环！
```

**错误 2**：条件写反

```python
count = 1
while count >= 5:  # 错！一开始条件就不成立
    print(count)
    count += 1
# 什么都不输出
```

**错误 3**：缩进错误

```python
count = 1
while count <= 5:
print(count)   # 错！缩进必须对齐
count += 1
```

### 九、while 循环流程图

```
        开始
          ↓
    ┌─────────────┐
    │ 条件成立？   │
    └─────────────┘
      ↓ Yes      ↓ No
┌───────────┐    结束
│ 执行循环体 │
└───────────┘
      ↓
  回到条件判断
```

## 总结

| 概念 | 写法 | 作用 |
|------|------|------|
| while循环 | while 条件: | 条件成立时重复执行 |
| break | break | 跳出整个循环 |
| continue | continue | 跳过这一次，继续下一次 |
| 死循环 | while True: | 永远循环（需要 break 跳出） |

**选择建议**：
- 知道循环次数 → for
- 不知道循环次数 → while
- 永远循环 → while True + break
