⚔️ for vs while：什么时候用哪个？

✅ 用 for
  • ✓ 已知循环次数
  • ✓ 输出 1~100
  • ✓ 遍历列表
for i in range(10):
    print(i)
🎯 用 while
  • ✓ 未知循环次数
  • ✓ 猜数字游戏
  • ✓ 游戏主循环
while guess != answer:
    guess = int(input())
VS

🚪 break 和 continue

💥 break：跳出整个循环

for i in range(10):
    if i == 5:
        break
    print(i)
输出： 0 1 2 3 4

⏭️ continue：跳过这一次

for i in range(10):
    if i == 5:
        continue
    print(i)
输出： 0 1 2 3 4 6 7 8 9 （没有5）
break = 提前下课（整个课都不上了） | continue = 去厕所（这节课跳过，下节课继续）

☠️ 死循环问题

⚠️

什么是死循环？

条件永远为真，循环永远不会停止！

while True:
    print("我停不下来啦！")
😱

危险

  • • 程序卡住不动
  • • CPU 飙升
  • • 需要强制关闭
✅

如何避免

  • • 确保条件会变
  • • 用 break 跳出
  • • 加计数器限制

🎮 例题：猜数字游戏

📝 代码

answer = 42
attempts = 0

while attempts < 5:
    guess = int(input("猜："))
    attempts += 1
    if guess == answer:
        print("猜对了！")
        break
    elif guess < answer:
        print("太小！")
    else:
        print("太大！")

🎯 运行示例

猜：50
太大！
猜：30
太小！
猜：42
猜对了！

❌ 常见错误

错误1：忘记更新条件 → 死循环！

❌ count = 1; while count <= 5: print(count)
✓ count += 1 在循环体内

错误2：条件写反

❌ while count >= 5（一开始就不成立）
✓ while count <= 5

错误3：缩进错误

❌ print(count) 没有缩进
✓     print(count) 缩进4格
