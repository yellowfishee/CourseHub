函数调用与参数传递

理解Python的变量传递

🔒

不可变对象

int, str, tuple

🔓

可变对象

list, dict

⚠️

注意陷阱

🤔 函数会改变参数吗？

整数参数

def add_one(n):
  n = n + 1

x = 5
add_one(x)
print(x) # 还是5!

列表参数

def add_item(lst):
  lst.append(100)

nums = [1,2,3]
add_item(nums)
print(nums) # [1,2,3,100]

🔒 不可变对象

类型例子特点
int5, 100整数不能改
str"hello"字符串不能改
tuple(1, 2, 3)元组不能改

类比：刻在石头上的字，想改只能重新刻

🔓 可变对象

类型例子特点
list[1, 2, 3]列表可以改
dict{"a": 1}字典可以改

类比：写在白板上，可以随时擦掉重写

🧪 小测验

哪些是不可变对象？

A. 字符串 "hello"
B. 列表 [1, 2, 3]
C. 元组 (1, 2, 3)
D. 字典 {"a": 1}

📋 总结

核心记住：

整数、字符串传入函数不会被修改

列表、字典传入函数可能被修改

用 .copy() 可以避免意外修改
