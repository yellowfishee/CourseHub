## 参考答案

**选择题：** 1-B, 2-B, 3-B, 4-B, 5-B

**填空题：**
6. 上、下
7. 整数、保留2位小数的浮点数

**编程题（参考答案）：**

```python
# 题目1
n = int(input())
bai = n // 100
shi = n // 10 % 10
ge = n % 10
print(f"百位：{bai}")
print(f"十位：{shi}")
print(f"个位：{ge}")

# 题目2
n = int(input())
bai = n // 100
shi = n // 10 % 10
ge = n % 10
result = ge * 100 + shi * 10 + bai
print(result)

# 题目3
a = 10
b = 20
t = a
a = b
b = t
print(f"a = {a}")
print(f"b = {b}")

# 习题1-2 温度
f = float(input())
c = 5.0 * (f - 32.0) / 9.0
print(f"{c:.3f}")

# 习题1-3 连续和
n = int(input())
print(sum(range(1, n + 1)))

# 习题1-5 打折
n = int(input())
total = n * 95.0
if total >= 300.0:
    total *= 0.85
print(f"{total:.2f}")

# 习题1-4 正弦和余弦
import math
n = int(input())
rad = n * math.pi / 180.0
print(f"{math.sin(rad):.2f} {math.cos(rad):.2f}")
```
