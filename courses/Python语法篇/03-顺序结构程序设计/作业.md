## 编程题

### 简单题

**题目1：** 三位数反转

**描述：** 输入一个三位数，反转后输出。

```
示例输入：127
示例输出：721
```

**提示：** 分离各位数字后重新组合

---

**题目2：** 变量交换

**描述：** 已知 a = 10, b = 20，交换它们的值后输出。

```
示例输出：
a = 20
b = 10
```

---

**题目3：** 计算球体积

**描述：** 输入球的半径 r，输出球的体积。
公式：体积 = 4/3 * π * r³
要求保留3位小数。

```
示例输入：3
示例输出：113.097
```

**提示：** 用 import math 和 math.pi

---

### 中等题

**题目4：** 三位数各位求和

**描述：** 输入一个三位数，输出它的各位数字之和。
例如：127 -> 1 + 2 + 7 = 10

```
示例输入：127
示例输出：10
```

**提示：** 分离各位数字后相加

## 参考答案

### 选择题答案
1-B, 2-B, 3-B, 4-C, 5-B

### 编程题参考答案

```python
# 题目1：三位数反转
n = int(input())
bai = n // 100
shi = n // 10 % 10
ge = n % 10
result = ge * 100 + shi * 10 + bai
print(result)
```

```python
# 题目2：变量交换
a = 10
b = 20
a, b = b, a
print(f"a = {a}")
print(f"b = {b}")
```

```python
# 题目3：计算球体积
import math
r = float(input())
volume = 4.0 / 3.0 * math.pi * r ** 3
print(f"{volume:.3f}")
```

```python
# 题目4：三位数各位求和
n = int(input())
bai = n // 100
shi = n // 10 % 10
ge = n % 10
total = bai + shi + ge
print(total)
```
