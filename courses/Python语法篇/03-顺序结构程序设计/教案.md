## 内容

### 一、引入：做菜的顺序

**提问**：做一道菜（比如番茄炒蛋），步骤是什么？

**回答**：
1. 打鸡蛋
2. 切番茄
3. 开火炒菜
4. 装盘

**类比**：程序执行也是一样，从上往下**依次执行**，这就是"顺序结构"。

---

### 二、顺序执行

**Python 代码的执行顺序**：

```python
a = 5      # 第1步
b = 3      # 第2步
c = a + b  # 第3步
print(c)   # 第4步
```

**执行顺序**：
```
从上往下，一步一步执行
第1步 → 第2步 → 第3步 → 第4步
```

---

### 三、格式化输出

**问题**：如何控制输出的格式？

**方法1：f-string（推荐）**

```python
a = 5
print(f"a = {a}")        # a = 5
print(f"{a:.2f}")        # 5.00
```

**方法2：% 格式化**

```python
print("%d" % 5)          # 5
print("%.3f" % 3.14159)  # 3.142
print("%03d" % 5)        # 005
```

**格式化符号说明**：

| 符号 | 含义 | 例子 |
|------|------|------|
| %d | 整数 | 5 |
| %.nf | 浮点数，保留n位小数 | %.3f → 3.142 |
| %0nd | 宽度n，不足前面补0 | %03d → 005 |
| %s | 字符串 | "hello" |

### 四、例题：三位数反转

**题目**：输入一个三位数，反转后输出

```
输入：127
输出：721
```

**分析**：
- 百位 = n // 100
- 十位 = n // 10 % 10
- 个位 = n % 10

**Python 代码**：

```python
n = int(input())
bai = n // 100
shi = n // 10 % 10
ge = n % 10
print(bai, shi, ge, sep='')
```

**拆解问题**：
```
127 // 100 = 1（百位）
127 // 10 = 12，12 % 10 = 2（十位）
127 % 10 = 7（个位）

反转：721
```

### 五、变量交换复习

**问题**：如何交换两个变量的值？

**方法1：三变量法（推荐）**

```python
a = 3
b = 5
t = a    # t = 3
a = b    # a = 5
b = t    # b = 3
```

**方法2：不借变量（数学技巧）**

```python
a = 3
b = 5
a = a + b   # a = 8
b = a - b   # b = 3
a = a - b   # a = 5
```

**注意**：方法2只适用于数字，不够通用。

### 六、竞赛题目格式

**正规竞赛题目的组成**：

1. **题目描述**：要解决什么问题
2. **输入格式**：数据怎么给
3. **输出格式**：结果怎么输出
4. **样例输入/输出**：示例

**例子**：

```
题目：计算 a + b

输入：一行两个整数 a 和 b
输出：a + b 的值

样例输入：3 5
样例输出：8
```

---

### 七、竞赛规则提醒

**注意点**：

1. **不要加提示语**
   - ❌ `print("请输入a:")`
   - ✅ 直接 `input()`

2. **输出格式要严格**
   - 题目说输出整数，就输出整数
   - 题目说保留3位小数，就用 `%.3f`

3. **每行末尾要有换行**
   - Python 的 `print()` 自动带换行

4. **不要等待用户按键**
   - 不要用 `input("按回车继续")`

## 总结

今天我们学了：

| 概念 | Python 写法 | 说明 |
|------|------------|------|
| 顺序执行 | 一行一行 | 从上往下依次执行 |
| 整数格式化 | %d 或 f"{x}" | 控制输出整数 |
| 浮点格式化 | %.nf | 保留n位小数 |
| 宽度格式化 | %0nd | 宽度n，不足补0 |
| 三位数反转 | 百十个分解 | 用 // 和 % |

**记住**：程序是"一步一步"执行的，要养成拆解问题的好习惯！
