❌ 常见错误：变量名拼写

❌ 错误代码

name = "小明"
print(nmae) # 拼写错误！

NameError: name 'nmae' is not defined

✓ 正确代码

name = "小明"
print(name) # 拼写正确

输出：小明

❌ 常见错误：类型错误

❌ 错误代码

a = input() # 得到字符串
print(a + 1) # 字符串+数字？

TypeError: can only concatenate str to str

✓ 正确代码

a = int(input()) # 转整数
print(a + 1)

输入 5 → 输出 6

❌ 常见错误：比较运算符

❌ 错误代码

if a = 5: # = 是赋值！
    print("等于5")

SyntaxError: invalid syntax

✓ 正确代码

if a == 5: # == 是比较
    print("等于5")

正确判断是否等于5

❌ 常见错误：忘记冒号和缩进

❌ 错误代码

if a > 5 # 少了冒号！
print("大于5") # 没有缩进！

✓ 正确代码

if a > 5: # 有冒号
    print("大于5") # 缩进4格
💡 记住：if/elif/else 后面都要加冒号，代码块要缩进！

✏️ 练习题：判断奇偶

输入一个整数，判断它是奇数还是偶数

n = int(input())

if n % 2 == 0:
    print("偶数")
else:
    print("奇数")
输入： 6
输出：偶数
输入： 7
输出：奇数

📝 总结

知识点 要点
变量 先定义后使用，注意拼写
输入 input() 得到字符串，需要转换
比较 用 == 不是 =
分支 别忘冒号，缩进一致
调试 看报错信息，用 print()
