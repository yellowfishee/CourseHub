### 二、常见错误分析

**错误1：变量名拼写错误**

```python
name = "小明"
print(nmae)  # 错误！拼写错误
```

**正确**：
```python
name = "小明"
print(name)  # 正确
```

---

**错误2：类型错误**

```python
a = input()  # 得到的是字符串！
print(a + 1)  # 错误！字符串不能加数字
```

**正确**：
```python
a = int(input())  # 转换为整数
print(a + 1)
```

---

**错误3：比较运算符写错**

```python
if a = 5:  # 错误！= 是赋值，不是比较
    print("等于5")
```

**正确**：
```python
if a == 5:  # == 才是比较
    print("等于5")
```

---

**错误4：忘记冒号**

```python
if a > 5  # 错误！少了冒号
    print("大于5")
```

**正确**：
```python
if a > 5:  # 冒号不能少
    print("大于5")
```

---

**错误5：缩进不一致**

```python
if a > 5:
print("大于5")  # 错误！没有缩进
    print("真的大于5")  # 缩进不一致
```

**正确**：
```python
if a > 5:
    print("大于5")      # 缩进4格
    print("真的大于5")  # 缩进一致
```

**练习3：三个数排序**

输入三个整数 a, b, c，按从小到大的顺序输出

**思路**：
1. 比较三次，找出最小、中间、最大
2. 或者用 Python 的排序功能

```python
a = int(input())
b = int(input())
c = int(input())

# 方法1：手动比较
if a <= b and a <= c:
    min_val = a
    if b <= c:
        mid, max_val = b, c
    else:
        mid, max_val = c, b
elif b <= a and b <= c:
    min_val = b
    if a <= c:
        mid, max_val = a, c
    else:
        mid, max_val = c, a
else:
    min_val = c
    if a <= b:
        mid, max_val = a, b
    else:
        mid, max_val = b, a

print(min_val, mid, max_val)

# 方法2：Python 简洁写法
nums = [a, b, c]
nums.sort()
print(nums[0], nums[1], nums[2])
```

**练习5：判断闰年**

输入年份，判断是否为闰年

**规则**：
- 能被4整除但不能被100整除，或者能被400整除

```python
year = int(input())

if (year % 4 == 0 and year % 100 != 0) or year % 400 == 0:
    print("闰年")
else:
    print("平年")
```

### 四、调试技巧

**1. 看报错信息**

Python 会告诉你哪一行出错了：

```
Traceback (most recent call last):
  File "test.py", line 3, in <module>
    print(nmae)
NameError: name 'nmae' is not defined
```

**解读**：第3行，变量 `nmae` 未定义（可能是拼写错误）

**2. 用 print() 调试**

```python
a = int(input())
b = int(input())
print(f"a = {a}, b = {b}")  # 调试输出
print(a + b)
```

**3. 分步执行**

把复杂的程序拆成小步骤，一步一步验证。
