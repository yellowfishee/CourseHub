## 内容

### 一、引入：为什么需要字典？

**问题**：如何存储学生的成绩？

**用列表？**
```python
names = ["小明", "小红", "小刚"]
scores = [90, 85, 88]

# 要找小明的成绩，要先找索引
idx = names.index("小明")
print(scores[idx])  # 90
```

**麻烦！** 要用两个列表，还要找索引。

**用字典！**
```python
scores = {"小明": 90, "小红": 85, "小刚": 88}

print(scores["小明"])  # 直接用名字找！
```

**类比**：
```
字典 = 真正的字典
查"apple" → 找到"苹果"
用"键"（apple）找"值"（苹果）
```

---

### 二、字典的基本概念

**键值对（Key-Value Pair）**：
```
键 → 唯一的标识符（不能重复）
值 → 对应的数据（可以重复）
```

**创建字典**：

```python
# 方式1：直接创建
scores = {"小明": 90, "小红": 85}

# 方式2：空字典，然后添加
scores = {}
scores["小明"] = 90
scores["小红"] = 85

# 方式3：用dict函数
scores = dict(小明=90, 小红=85)
```

**访问值**：

```python
scores = {"小明": 90, "小红": 85}

print(scores["小明"])  # 90
print(scores["小红"])  # 85
```

**类比**：
```
列表：用编号找 → nums[0]
字典：用名字找 → scores["小明"]
```

---

### 三、字典的操作

**添加/修改**：

```python
scores = {"小明": 90}

# 添加新键值对
scores["小红"] = 85

# 修改已有键的值
scores["小明"] = 95

print(scores)  # {"小明": 95, "小红": 85}
```

**删除**：

```python
scores = {"小明": 90, "小红": 85}

del scores["小明"]
print(scores)  # {"小红": 85}
```

**检查键是否存在**：

```python
scores = {"小明": 90}

if "小明" in scores:
    print("小明存在")

if "小刚" in scores:
    print("小刚存在")  # 不打印
```

### 四、字典的方法

**常用方法**：

| 方法 | 作用 | 例子 |
|------|------|------|
| `get(key)` | 获取值（不存在返回None） | `scores.get("小明")` |
| `keys()` | 获取所有键 | `scores.keys()` |
| `values()` | 获取所有值 | `scores.values()` |
| `items()` | 获取所有键值对 | `scores.items()` |
| `pop(key)` | 删除并返回值 | `scores.pop("小明")` |
| `clear()` | 清空字典 | `scores.clear()` |

**例子**：

```python
scores = {"小明": 90, "小红": 85, "小刚": 88}

# 获取所有键
print(scores.keys())   # dict_keys(['小明', '小红', '小刚'])

# 获取所有值
print(scores.values()) # dict_values([90, 85, 88])

# 获取所有键值对
print(scores.items())  # dict_items([('小明', 90), ('小红', 85), ('小刚', 88)])

# 安全获取（避免KeyError）
print(scores.get("小刚"))     # 88
print(scores.get("小李"))     # None（不会报错）
print(scores.get("小李", 0))  # 0（不存在返回默认值）
```

### 五、遍历字典

**方式1：遍历键**

```python
scores = {"小明": 90, "小红": 85}

for name in scores:
    print(name, scores[name])
```

**方式2：遍历键值对**

```python
scores = {"小明": 90, "小红": 85}

for name, score in scores.items():
    print(f"{name}: {score}")
```

**输出**：
```
小明: 90
小红: 85
```

### 六、例题：单词计数

**问题**：统计一段文字中每个单词出现的次数

**输入**：
```
"apple banana apple orange banana apple"
```

**输出**：
```
apple: 3
banana: 2
orange: 1
```

**代码**：

```python
text = "apple banana apple orange banana apple"
words = text.split()  # 分割成单词列表

# 用字典计数
count = {}
for word in words:
    if word in count:
        count[word] += 1  # 已存在，加1
    else:
        count[word] = 1   # 不存在，设为1

# 输出结果
for word, num in count.items():
    print(f"{word}: {num}")
```

**Python简洁写法**：

```python
from collections import Counter

text = "apple banana apple orange banana apple"
count = Counter(text.split())

print(count)  # Counter({'apple': 3, 'banana': 2, 'orange': 1})
```

### 七、字典 vs 列表

| 特点 | 列表 | 字典 |
|------|------|------|
| 索引方式 | 数字编号（0,1,2...） | 键（任意不可变类型） |
| 查找速度 | 按顺序找，慢 | 直接找，快 |
| 是否有序 | 有序 | Python3.7+有序 |
| 适用场景 | 存储一组数据 | 存储映射关系 |

**何时用字典？**
- 需要用"名字"而不是"编号"找数据
- 需要快速查找
- 存储键值对关系

### 八、常见错误

**错误1：键不存在**

```python
scores = {"小明": 90}
print(scores["小红"])  # KeyError！
```

**正确**：用 `get()` 方法

```python
scores = {"小明": 90}
print(scores.get("小红"))  # None（不报错）
```

**错误2：键重复**

```python
scores = {"小明": 90, "小明": 85}
print(scores)  # {"小明": 85} 后面的覆盖前面的
```

**错误3：用可变对象做键**

```python
bad_dict = {[1, 2]: "value"}  # 错误！列表不能做键
```

**正确**：键必须是不可变对象（int、str、tuple）

```python
good_dict = {(1, 2): "value"}  # 元组可以做键
```

## 总结

| 概念 | 要点 |
|------|------|
| 字典 | 键值对的映射表 |
| 键 | 唯一标识符，不可变 |
| 值 | 对应的数据，可变 |
| get() | 安全获取值 |
| items() | 获取所有键值对 |

**记住**：
- 字典用键找值，比列表更方便
- 用 `get()` 避免键不存在报错
- 键必须是不可变对象
