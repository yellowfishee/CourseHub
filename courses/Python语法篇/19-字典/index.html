📖 什么是字典？

❌ 用列表存储成绩

names = ["小明", "小红"]
scores = [90, 85]

# 要找小明的成绩
idx = names.index("小明")
print(scores[idx])

麻烦！要用两个列表

✓ 用字典存储成绩

scores = {
  "小明": 90,
  "小红": 85
}

print(scores["小明"])

直接用名字找！

⚙️ 字典操作

scores = {"小明": 90}

# 添加
scores["小红"] = 85

# 修改
scores["小明"] = 95

# 删除
del scores["小红"]

🛠️ 常用方法

方法作用例子
get(key)获取值scores.get("小明")
keys()所有键scores.keys()
values()所有值scores.values()
items()所有键值对scores.items()
pop(key)删除并返回scores.pop("小明")

🔄 遍历字典

scores = {"小明": 90, "小红": 85}

# 遍历键值对
for name, score in scores.items():
    print(f"{name}: {score}")

输出：

小明: 90

小红: 85

📝 例题：单词计数

text = "apple banana apple orange"
words = text.split()

count = {}
for word in words:
    count[word] = count.get(word, 0) + 1

print(count)
# {'apple': 2, 'banana': 1, 'orange': 1}
