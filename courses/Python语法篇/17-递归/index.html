⚠️ 递归的两个条件

1️⃣ 基准情形

必须有停止点

if n == 1:
  return 1

2️⃣ 递归调用

每次缩小问题

return n * factorial(n-1)

📝 阶乘递归

def factorial(n):
  # 基准情形
  if n == 1:
    return 1

  # 递归调用
  return n * factorial(n - 1)

print(factorial(5)) # 120
