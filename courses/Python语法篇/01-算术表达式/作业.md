## 编程题

### 简单题

**题目1：** 温度转换

**描述：** 输入华氏温度 f，输出摄氏温度 c，保留3位小数。
公式：c = 5*(f-32)/9

```
示例输入：100
示例输出：37.778
```

**提示：** 用浮点数运算

---

**题目2：** 计算总和

**描述：** 输入三个整数，输出它们的和与平均值，平均值保留2位小数。

```
示例输入：10 20 30
示例输出：
60
20.00
```

**提示：** 用 sum() 或直接相加

---

**题目3：** 矩形面积

**描述：** 输入矩形的宽和高，输出矩形面积。

```
示例输入：5 8
示例输出：40
```

---

### 中等题

**题目4：** 分段计算

**描述：** 输入一个整数 x，计算下面的分段函数值：
- 如果 x < 0：y = x + 10
- 如果 0 <= x < 10：y = x * 2
- 如果 x >= 10：y = x - 10

```
示例输入1：-5
示例输出1：5

示例输入2：3
示例输出2：6

示例输入3：15
示例输出3：5
```

**提示：** 用 if-elif-else 判断

## 参考答案

### 选择题答案
1-B, 2-B, 3-B, 4-A, 5-B

### 编程题参考答案

```python
# 题目1：温度转换
f = float(input())
c = 5.0 * (f - 32.0) / 9.0
print(f"{c:.3f}")
```

```python
# 题目2：计算总和与平均值
a = int(input())
b = int(input())
c = int(input())
total = a + b + c
average = total / 3
print(total)
print(f"{average:.2f}")
```

```python
# 题目3：矩形面积
width = float(input())
height = float(input())
area = width * height
print(area)
```

```python
# 题目4：分段计算
x = int(input())
if x < 0:
    y = x + 10
elif x < 10:
    y = x * 2
else:
    y = x - 10
print(y)
```
