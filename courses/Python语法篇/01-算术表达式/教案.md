### 一、引入：计算机最擅长做计算

**提问**：计算机最擅长什么？

**回答**：计算！速度超级快。

**类比**：就像计算器，但比计算器更厉害——可以做很复杂的计算。

**生活例子**：
```
你去餐厅吃饭，点完菜后：
服务员记下你的菜名（输入）→ 厨房做菜（计算）→ 服务员端菜给你（输出）
计算机也一样：
接收数据（输入）→ 计算处理 → 显示结果（输出）
```

### 二、第一个 Python 命令：print()

**问题**：怎么让计算机显示一个数字？

**Python 命令**：

```python
print(1 + 2)
```

**解释**：
- `print()` 是一个"命令"，意思是"显示出来"
- 就像打印机把文字印在纸上，计算机会把结果"印"在屏幕上

**类比**：
```
print() = 屏幕上的"打印机"
把内容"打印"到屏幕上
```

**动手试试**：
```python
print(1 + 2)      # 显示 3
print(100 - 50)   # 显示 50
print(3 * 4)      # 显示 12
print(10 / 2)     # 显示 5.0
```

### 三、加减乘除：四个基本运算

**提问**：1 + 2 等于几？ 10 - 3 呢？ 3 × 4 呢？ 15 ÷ 3 呢？

**在 Python 中**：

| 数学符号 | Python 写法 | 例子 | 结果 |
|---------|------------|------|------|
| + | + | 1 + 2 | 3 |
| - | - | 10 - 3 | 7 |
| × | * | 3 * 4 | 12 |
| ÷ | / | 15 / 3 | 5.0 |

**重点**：乘号是 `*`（星号），除号是 `/`（斜杠）——键盘上找不到数学符号，用它们代替！

**小任务**：计算 123 + 456 的结果

### 四、重要发现：整数除法和浮点数除法

**实验1**：运行下面代码

```python
print(8 / 5)
```

**结果**：显示 `1.6`？

**再实验**：用整数试试

```python
print(8 / 5)   # 结果是 1.6（浮点数）
print(8 // 5)  # 结果是 1（整数除法，向下取整）
```

**规则**：

| 运算 | Python 写法 | 结果 | 类型 |
|------|------------|------|------|
| 浮点数除法 | 8 / 5 | 1.6 | 浮点数 |
| 整数除法 | 8 // 5 | 1 | 整数 |
| 取余数（取余） | 8 % 5 | 3 | 整数 |

**生活类比**：
```
8 ÷ 5 = 1 ... 3
     ↓
商是 1，余数是 3
     ↓
8 // 5 = 1（得到商）
8 % 5 = 3（得到余数）
```

**提示**：整数之间的运算，结果也是整数（除了 `/` 以外）

### 五、数学函数：sqrt() 求平方根

**问题**：怎么计算 √3（3的平方根）？

**Python 代码**：

```python
import math
print(math.sqrt(3))
```

**结果**：大约是 1.732...

**解释**：
- `math.sqrt()` 是一个"数学函数"
- `sqrt` 是 "square root"（平方根）的缩写
- `math.sqrt(x)` 的作用是计算 x 的平方根

**类比**：
```
math.sqrt() = 计算器上的 √ 按钮
math.sqrt(9) = 3，因为 3 × 3 = 9
```

**实验**：验证 √9 = 3

```python
import math
print(math.sqrt(9))    # 3.0
print(math.sqrt(2))    # 1.4142135623730951
print(math.sqrt(100))  # 10.0
```

### 六、复杂表达式：混合运算

**问题**：计算 `1 + 2 × √3 ÷ (5 - 0.1)`

**Python 代码**：

```python
import math
print(1 + 2 * math.sqrt(3) / (5 - 0.1))
```

**拆解问题**：

```
1 + 2 * math.sqrt(3) / (5 - 0.1)
       ↓
一步一步来：
1. 先算 5 - 0.1 = 4.9
2. 再算 math.sqrt(3) ≈ 1.732
3. 再算 2 * 1.732 = 3.464
4. 再算 3.464 / 4.9 ≈ 0.707
5. 最后算 1 + 0.707 ≈ 1.707
```

**运算符优先级**（先算什么）：
```
1. 括号 ()
2. 乘除 * / // %
3. 加减 + -
4. 从左到右
```

**类比**：
```
就像数学课学的：先乘除，后加减，有括号先算括号里的
```

### 七、常见错误和调试

**错误1**：忘了加引号或括号

```python
print(Hello)      # 错误！Hello 是变量名，不是字符串
print("Hello")    # 正确！用引号包围才是字符串
print(1 + 2)      # 正确！数字不需要引号
```

**错误2**：除号用错了

```python
print(8 \ 5)      # 错误！反斜杠不是除号
print(8 / 5)      # 正确！用斜杠
```

**错误3**：中文括号

```python
print（1 + 2）    # 错误！用了中文括号
print(1 + 2)      # 正确！用英文括号
```

**提示**：看报错信息很重要！Python 会告诉你哪里出了问题。
