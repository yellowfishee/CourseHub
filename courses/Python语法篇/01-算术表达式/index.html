数学函数：sqrt() 求平方根

√
import math
print(math.sqrt(9))

输出：3.0

sqrt = square root = 平方根

math.sqrt(9) = 3，因为 3 × 3 = 9

常见错误 ⚠️

错误 1：忘了加引号

print(Hello) # 错！
print("Hello") # 对！

错误 2：用错了除号

print(8 \ 5) # 错！
print(8 / 5) # 对！

错误 3：用中文括号

print（1 + 2） # 错！
print(1 + 2) # 对！
