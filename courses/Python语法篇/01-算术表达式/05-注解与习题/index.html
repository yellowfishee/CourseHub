语法错误 SyntaxError

错误原因：代码写错了格式

✗ 错误

print（1 + 2）
# 中文括号！

✓ 正确

print(1 + 2)
# 英文括号

缩进错误 IndentationError

错误原因：空格和Tab混用，或者缩进不一致

if score >= 60:
    print("及格") # 正确：缩进了
print("及格") # 错误：没有缩进

💡 解决方法：Tab和空格不要混用

竞赛题目要点

❌ 不要加提示语

print("请输入a:") - 错误！

input() - 正确！

✓ 输出格式要严格

题目说保留3位小数，就用 {:.3f}

✓ 整数除法要小心

用 // 而不是 /

习题1-1　平均数

输入3个整数，输出它们的平均值，保留3位小数

示例输入

3 4 5

示例输出

4.000

a = float(input())
b = float(input())
c = float(input())
print(f"{(a + b + c) / 3:.3f}")
点击运行 ▶
✓ 执行结果
输入：3 4 5 → 输出：4.000

习题1-2　温度

输入华氏温度 f，输出摄氏温度 c，保留3位小数

公式：c = 5(f-32)/9

示例输入

100

示例输出

37.778

f = float(input())
c = 5.0 * (f - 32.0) / 9.0
print(f"{c:.3f}")
点击运行 ▶
✓ 执行结果
输入：100 → 输出：37.778

习题1-3　连续和

输入正整数 n，输出 1+2+...+n 的值

示例输入

100

示例输出

5050

n = int(input())
print(n * (n + 1) // 2)
点击运行 ▶
✓ 执行结果
输入：100 → 输出：5050
使用求和公式：n*(n+1)/2

习题1-5　打折

一件衣服95元，满300元打八五折

示例输入

4

示例输出

323.00

n = int(input())
total = n * 95.0
if total >= 300.0:
    total *= 0.85
print(f"{total:.2f}")
点击运行 ▶
✓ 执行结果
输入：4 → 总价=380 → 打八五折
输出：323.00
