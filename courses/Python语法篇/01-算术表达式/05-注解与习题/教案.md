### 常见错误分类

#### 1. 语法错误（SyntaxError）

**错误原因：** 代码写错了格式

```
print(1 + 2)    # 正确：括号是英文括号
print（1 + 2）   # 错误：括号是中文括号
```

#### 2. 缩进错误（IndentationError）

**错误原因：** 空格和Tab混用，或者缩进不一致

```python
if score >= 60:
print("及格")    # 错误：没有缩进
    print("及格")  # 正确：缩进了
```

#### 3. 引号错误

```python
print("Hello")   # 正确：英文引号
print（"Hello"）  # 错误：括号是中文
print("Hello）   # 错误：引号不配对
```

#### 4. 除法类型错误

```python
print(8 / 5)    # 正确：浮点数除法，结果是1.6
print(8 // 5)   # 整数除法，结果是1（取整）
print(8 % 5)    # 取余数，结果是3
```

### 调试技巧

#### 看报错信息的三个步骤

1. **找到错误在哪一行** - 报错会告诉你是第几行
2. **看清是什么类型的错误** - SyntaxError? NameError?
3. **检查那一行的代码** - 引号、括号、缩进是否正确

```python
# 示例报错
File "test.py", line 3
    print("Hello
                  ^
SyntaxError: EOL while scanning string literal
```

这个报错告诉我们：
- 第3行有错误
- 错误类型：字符串没有正常结束
- 问题：引号没有配对

### 竞赛题目要点

1. **不要加提示语** - 直接 input()，不要 print("请输入")
2. **输出格式要严格** - 题目说保留3位小数，就用 {:.3f}
3. **整数除法要小心** - 用 // 而不是 /

## 习题讲解

### 题目1：平均数（average）

**题目描述：** 输入3个整数，输出它们的平均值，保留3位小数。

**思路分析：**
1. 读取3个数字
2. 计算它们的和
3. 除以3得到平均值
4. 保留3位小数输出

**参考答案：**
```python
a = float(input())
b = float(input())
c = float(input())
print(f"{(a + b + c) / 3:.3f}")
```

### 题目2：温度（temperature）

**题目描述：** 输入华氏温度 f，输出对应的摄氏温度 c，保留3位小数。
**公式：** c = 5(f-32)/9

**思路分析：**
1. 读取华氏温度
2. 用公式计算摄氏温度
3. 保留3位小数输出

**参考答案：**
```python
f = float(input())
c = 5.0 * (f - 32.0) / 9.0
print(f"{c:.3f}")
```

### 题目3：连续和（sum）

**题目描述：** 输入正整数 n，输出 1+2+...+n 的值。

**思路分析：**
1. 读取 n
2. 使用循环累加，或者用求和公式 n*(n+1)/2

**参考答案：**
```python
n = int(input())
print(n * (n + 1) // 2)  # 使用公式
```

### 题目4：打折（discount）

**题目描述：** 一件衣服95元，若消费满300元，可打八五折。输入购买衣服件数，输出需要支付的金额（单位：元），保留两位小数。

**思路分析：**
1. 计算总价 = 件数 × 95
2. 如果总价 >= 300，打八五折
3. 保留两位小数输出

**参考答案：**
```python
n = int(input())
total = n * 95.0
if total >= 300.0:
    total *= 0.85
print(f"{total:.2f}")
```
