## 编程题

### 基础题

**题目1：** 平均数

输入3个整数，输出它们的平均值，保留3位小数。

```
示例输入：3 4 5
示例输出：4.000
```

---

**题目2：** 温度转换

输入华氏温度 f，输出摄氏温度 c，保留3位小数。
公式：c = 5(f-32)/9

```
示例输入：100
示例输出：37.778
```

---

**题目3：** 连续和

输入正整数 n，输出 1+2+...+n 的值。

```
示例输入：100
示例输出：5050
```

---

### 挑战题

**题目4：** 打折

一件衣服95元，若消费满300元，可打八五折。输入购买衣服件数，输出需要支付的金额（单位：元），保留两位小数。

```
示例输入：4
示例输出：323.00
```

**题目5（选做）：** 猜数字

计算机想一个1-100之间的数字，你输入猜测，程序告诉你太大了还是太小了，直到猜对为止。

```
示例交互：
请输入你的猜测：50
太大了！请继续猜：25
太小了！请继续猜：37
恭喜你猜对了！答案是37
```

## 参考答案

**选择题：** 1-B, 2-B, 3-B, 4-B, 5-B

**填空题：**
6. 语法
7. 英文
8. n*(n+1)/2
9. 格式
10. 3.142

**编程题参考答案：**

```python
# 题目1：平均数
a = float(input())
b = float(input())
c = float(input())
print(f"{(a + b + c) / 3:.3f}")

# 题目2：温度转换
f = float(input())
c = 5.0 * (f - 32.0) / 9.0
print(f"{c:.3f}")

# 题目3：连续和
n = int(input())
print(n * (n + 1) // 2)

# 题目4：打折
n = int(input())
total = n * 95.0
if total >= 300.0:
    total *= 0.85
print(f"{total:.2f}")
```
