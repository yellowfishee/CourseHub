## 内容

### 一、竞赛题目特点

**竞赛题目结构**：
```
1. 题目描述：要解决什么问题
2. 输入格式：数据怎么给
3. 输出格式：结果怎么输出
4. 样例输入/输出：示例
5. 时间/空间限制：效率要求
```

**解题步骤**：
```
1. 理解题目（读三遍！）
2. 分析输入输出
3. 设计算法
4. 写代码
5. 测试样例
6. 调试优化
```

### 二、例题1：三位数反转

**题目**：输入一个三位数，反转后输出。

**样例**：
```
输入：123
输出：321

输入：100
输出：1（注意：去掉前导零）
```

**方法1：数学方法**

```python
n = int(input())
a = n // 100        # 百位
b = (n // 10) % 10  # 十位
c = n % 10          # 个位
result = c * 100 + b * 10 + a
print(result)
```

**方法2：字符串方法**

```python
n = input()
result = int(n[::-1])  # 反转后转整数，自动去掉前导零
print(result)
```

### 三、例题2：鸡兔同笼

**题目**：已知鸡和兔的总数量为n，总腿数为m。输出鸡和兔的数量，无解输出"No answer"。

**样例**：
```
输入：14 32
输出：12 2（12只鸡，2只兔）

输入：10 16
输出：No answer
```

**分析**：
```
设鸡x只，兔y只
x + y = n
2x + 4y = m

解方程：
y = (m - 2n) / 2
x = n - y

条件：
1. y >= 0
2. x >= 0
3. m - 2n 是偶数
```

**代码**：

```python
n, m = map(int, input().split())

y = (m - 2 * n) // 2  # 兔子数量
x = n - y              # 鸡的数量

# 判断是否有效
if y >= 0 and x >= 0 and (m - 2 * n) % 2 == 0:
    print(x, y)
else:
    print("No answer")
```

### 四、例题3：最大值和最小值

**题目**：输入n个数字，输出最大值和最小值。

**代码**：

```python
n = int(input())
nums = list(map(int, input().split()))

print(max(nums), min(nums))
```

**手动实现**：

```python
n = int(input())
nums = list(map(int, input().split()))

max_val = nums[0]
min_val = nums[0]

for num in nums:
    if num > max_val:
        max_val = num
    if num < min_val:
        min_val = num

print(max_val, min_val)
```

### 五、例题4：统计数字

**题目**：输入一个字符串，统计其中数字的个数。

**代码**：

```python
s = input()
count = 0
for ch in s:
    if ch.isdigit():
        count += 1
print(count)
```

### 六、例题5：判断素数

**题目**：判断一个数是否是素数。

**代码**：

```python
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

n = int(input())
if is_prime(n):
    print("Yes")
else:
    print("No")
```

## 总结

| 步骤 | 要点 |
|------|------|
| 理解题目 | 读三遍，明确输入输出 |
| 设计算法 | 先想思路，再写代码 |
| 测试验证 | 用样例测试 |
| 调试优化 | 看错误信息，逐步修复 |

**记住**：
- 先理解题目，再动手写代码
- 分步实现，不要一次性写完
- 用函数封装复杂逻辑
