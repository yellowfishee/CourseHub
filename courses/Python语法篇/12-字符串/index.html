s = 'Hello'

✂️ 字符串切片

取一部分字符串

s = "Hello World"

print(s[0:5]) # "Hello"
print(s[6:11]) # "World"
print(s[:5]) # "Hello"（前5个）
print(s[6:]) # "World"（从6开始）

🔧 字符串常用操作

操作 代码 作用
长度 len(s) 字符个数
计数 s.count('l') 'l'出现的次数
查找 s.find('l') 第一个'l'的位置
替换 s.replace('l','L') 把'l'换成'L'
大写 s.upper() 全部大写
小写 s.lower() 全部小写

🔄 遍历字符串

方式1：用索引

s = "Hello"
for i in range(len(s)):
    print(s[i])

方式2：直接遍历

s = "Hello"
for ch in s:
    print(ch)

📊 例题：字符计数

统计字符串中某个字符出现的次数

s = "Hello World"

count = 0
for ch in s:
    if ch == 'l':
        count += 1

print(f"'l' 出现了 {count} 次") # 3次

手动实现

count = 3

Python 内置

s.count('l') # 直接得3

⚠️ 字符串不可修改

❌ 错误做法

s = "Hello"
s[0] = 'h' # 错误！

TypeError: 字符串不能修改

✓ 正确做法

s = "Hello"
chars = list(s)
chars[0] = 'h'
s = "".join(chars) # "hello"
字符串与列表的区别： 字符串不可修改，列表可以

📝 总结

概念 要点
字符串 一串文字，索引从0开始
切片 s[0:5] 取索引0到4的字符
长度 len(s) 返回字符个数
遍历 for ch in s 或 for i in range(len(s))
计数 s.count('l') 统计出现次数
区别 字符串不可修改，列表可以
字符串的索引和切片与列表完全一样！
