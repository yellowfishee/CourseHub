## 内容

### 一、引入：什么是字符串？

**生活例子**：
```
一串珠子：🟢🟡🔴🔵🟢
每个珠子是一个字符，串在一起就是字符串
```

**Python 的字符串**：

```python
s = "Hello"
# 或者用单引号
s = 'Hello'
```

**类比**：
```
字符串 = 一串文字
每个字符就像一颗珠子
可以用索引找到单个字符
```

---

### 二、字符串索引

**与列表类似，索引从 0 开始**：

```python
s = "Hello"

print(s[0])  # 'H'（第1个字符）
print(s[1])  # 'e'（第2个字符）
print(s[4])  # 'o'（第5个字符）
print(s[-1]) # 'o'（最后一个字符）
```

**可视化**：
```
字符串： H  e  l  l  o
索引：   0  1  2  3  4
负索引：-5 -4 -3 -2 -1
```

---

### 三、字符串切片

**取一部分字符串**：

```python
s = "Hello World"

print(s[0:5])   # "Hello"（索引0到4）
print(s[6:11])  # "World"（索引6到10）
print(s[:5])    # "Hello"（前5个）
print(s[6:])    # "World"（从索引6开始）
```

**类比**：
```
切片就像"剪一段珠子"
s[0:5] = 从位置0剪到位置5（不包括5）
```

---

### 四、字符串常用操作

| 操作 | 代码 | 作用 |
|------|------|------|
| 长度 | `len(s)` | 字符个数 |
| 查找 | `s.find("l")` | 找到第一个'l'的位置 |
| 计数 | `s.count("l")` | 'l'出现的次数 |
| 替换 | `s.replace("l", "L")` | 把'l'换成'L' |
| 分割 | `s.split(" ")` | 用空格分割成列表 |
| 连接 | `" ".join(list)` | 用空格连接列表 |
| 大写 | `s.upper()` | 全部大写 |
| 小写 | `s.lower()` | 全部小写 |
| 去空格 | `s.strip()` | 去掉两端空格 |

---

### 五、遍历字符串

**方式1：用索引**

```python
s = "Hello"
for i in range(len(s)):
    print(s[i])
```

**方式2：直接遍历**

```python
s = "Hello"
for ch in s:
    print(ch)
```

**输出**：
```
H
e
l
l
o
```

### 六、例题：字符计数

**问题**：统计字符串中每个字母出现的次数

**代码**：

```python
s = "Hello World"

# 统计 'l' 出现的次数
count = 0
for ch in s:
    if ch == 'l':
        count += 1
print(f"'l' 出现了 {count} 次")  # 3次
```

**Python 简洁写法**：

```python
s = "Hello World"
print(s.count('l'))  # 3
```

### 七、字符串与列表的区别

**区别1：字符串不可修改**

```python
s = "Hello"
s[0] = 'h'  # 错误！字符串不能修改单个字符
```

**解决方法**：先转成列表，再转回字符串

```python
s = "Hello"
chars = list(s)  # ['H', 'e', 'l', 'l', 'o']
chars[0] = 'h'   # 可以修改
s = "".join(chars)  # "hello"
```

**区别2：字符串只能存字符**

```python
# 列表可以存各种类型
nums = [1, "hello", 3.14]

# 字符串只能存字符
s = "hello"  # 只是一串文字
```

---

### 八、例题：反转字符串

**问题**：把字符串倒过来

**方法1：切片**

```python
s = "Hello"
rev = s[::-1]  # 步长为-1，倒序
print(rev)  # "olleH"
```

**方法2：循环**

```python
s = "Hello"
rev = ""
for ch in s:
    rev = ch + rev  # 每次把字符放到前面
print(rev)  # "olleH"
```

### 九、字符串输入输出

**输入字符串**：

```python
s = input()  # 直接读取一行
# 输入：Hello World
# s = "Hello World"
```

**输入单个字符**：

```python
ch = input()  # 读一行
ch = ch[0]    # 取第一个字符
```

**输出字符串**：

```python
print(s)           # 直接输出
print(f"结果是：{s}")  # 格式化输出
```

## 总结

| 概念 | 要点 |
|------|------|
| 字符串 | 一串文字，索引从0开始 |
| 切片 | s[0:5] 取索引0到4的字符 |
| 长度 | len(s) 返回字符个数 |
| 遍历 | for ch in s 或 for i in range(len(s)) |
| 计数 | s.count('l') 统计出现次数 |
| 区别 | 字符串不可修改，列表可以 |

**记住**：字符串的索引和切片与列表完全一样！
