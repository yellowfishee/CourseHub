🏪 什么是列表？

🚪 储物柜类比

📦 Python 列表

nums = [1, 3, 5, 7, 9]

每个"柜子"存一个数据

编号从 0 开始！（不是从1）

📝 创建列表

方式1：直接创建

nums = [1, 2, 3, 4, 5]
names = ["小明", "小红"]

方式2：空列表

nums = []
nums.append(1)
nums.append(2)

方式3：从输入创建

nums = list(map(int, input().split()))
# 输入：1 2 3 4 5 → nums = [1, 2, 3, 4, 5]

🔢 索引：找到柜子里的东西

索引就是"柜子的编号"，用来找到列表中的某个元素

10
[0]
20
[1]
30
[2]
40
[3]
50
[4]
nums = [10, 20, 30, 40, 50]

print(nums[0]) # 10（第1个）
print(nums[1]) # 20（第2个）
print(nums[4]) # 50（第5个）
⚠️ 重要：Python 索引从 0 开始！第1个元素是 nums[0]

🔢 负数索引

10
[0] / [-5]
20
[1] / [-4]
30
[2] / [-3]
40
[3] / [-2]
50
[4] / [-1]
nums = [10, 20, 30, 40, 50]

print(nums[-1]) # 50（最后一个）
print(nums[-2]) # 40（倒数第二个）

🔄 遍历列表

方式1：用索引

nums = [10, 20, 30]
for i in range(len(nums)):
    print(nums[i])

方式2：直接遍历

nums = [10, 20, 30]
for x in nums:
    print(x)
len(nums) = 列表的长度（元素个数）

📊 例题：找最大值

nums = [3, 7, 2, 9, 5]

max_val = nums[0] # 假设第一个最大
for x in nums:
    if x > max_val:
        max_val = x

print(max_val) # 9

手动实现

max_val = 9

Python 内置

max(nums) # 直接得9

🔧 常用操作

操作 代码 作用
取值 nums[0] 取第1个元素
修改 nums[0] = 10 改第1个元素
添加 nums.append(4) 添加到末尾
长度 len(nums) 元素个数
最大 max(nums) 最大值
求和 sum(nums) 总和

🧪 小测验

nums = [10, 20, 30, 40]，nums[1] 是多少？

A 10（第1个元素）
B 20（第2个元素，索引从0开始）
C 30（第3个元素）
D 索引1不存在

📝 总结

概念 要点
列表 一排编好号的"储物柜"
索引 从 0 开始！不是从 1
遍历 for x in nums 或 for i in range(len(nums))
常用函数 max(), min(), sum(), len()
第1个元素是 nums[0]，第5个元素是 nums[4]！
