## 内容

### 一、引入：什么是列表？

**生活例子**：
```
学校储物柜：
柜1号 → 柜2号 → 柜3号 → 柜4号 → 棍5号 → ...
每个柜子可以放一样东西，编号从1开始
```

**Python 的列表**：
```python
nums = [1, 3, 5, 7, 9]
```

**类比**：
```
列表 = 一排编好号的储物柜
每个"柜子"存一个数据
编号从 0 开始！（不是从1）
```

**为什么叫"列表"而不是"数组"？**
```
Python 里的叫法是"列表"（list）
其他语言（如 C++）叫"数组"（array）
概念相似，但 Python 的列表更灵活：
- 可以随时添加/删除元素
- 可以存放不同类型的数据
```

### 二、创建列表

**方式1：直接创建**

```python
nums = [1, 2, 3, 4, 5]
names = ["小明", "小红", "小刚"]
mixed = [1, "hello", 3.14]  # 可以混合不同类型
```

**方式2：空列表**

```python
nums = []  # 空列表
nums.append(1)  # 添加元素
nums.append(2)
print(nums)  # [1, 2]
```

**方式3：从输入创建**

```python
nums = list(map(int, input().split()))
# 输入：1 2 3 4 5
# nums = [1, 2, 3, 4, 5]
```

---

### 三、索引：找到柜子里的东西

**索引是什么？**

索引就是"柜子的编号"，用来找到列表中的某个元素。

**重要**：Python 的索引从 **0** 开始！

```python
nums = [10, 20, 30, 40, 50]

print(nums[0])  # 10（第1个元素）
print(nums[1])  # 20（第2个元素）
print(nums[4])  # 50（第5个元素）
```

**类比**：
```
列表：[10, 20, 30, 40, 50]
位置：  0   1   2   3   4

编号从0开始，就像楼层从地面开始数
```

**负数索引**：

```python
nums = [10, 20, 30, 40, 50]

print(nums[-1])  # 50（最后一个）
print(nums[-2])  # 40（倒数第二个）
```

---

### 四、修改列表

**修改某个元素**：

```python
nums = [1, 2, 3, 4, 5]
nums[0] = 100  # 把第一个改成100
print(nums)  # [100, 2, 3, 4, 5]
```

**添加元素**：

```python
nums = [1, 2, 3]
nums.append(4)  # 添加到末尾
print(nums)  # [1, 2, 3, 4]
```

**删除元素**：

```python
nums = [1, 2, 3, 4, 5]
nums.remove(3)  # 删除值为3的元素
print(nums)  # [1, 2, 4, 5]

# 或用索引删除
nums.pop(0)  # 删除第0个元素
print(nums)  # [2, 4, 5]
```

---

### 五、遍历列表

**方式1：用索引**

```python
nums = [10, 20, 30, 40, 50]
for i in range(len(nums)):
    print(nums[i])
```

**方式2：直接遍历**

```python
nums = [10, 20, 30, 40, 50]
for x in nums:
    print(x)
```

**len() 函数**：

```python
nums = [1, 2, 3, 4, 5]
print(len(nums))  # 5（列表长度）
```

### 六、例题：找最大值

**问题**：找出列表中的最大值

**思路**：
```
1. 假设第一个是最大值
2. 挨个比较，如果更大的就更新
3. 最后得到真正的最大值
```

**代码**：

```python
nums = [3, 7, 2, 9, 5]

max_val = nums[0]  # 假设第一个最大
for x in nums:
    if x > max_val:
        max_val = x

print(max_val)  # 9
```

**Python 简洁写法**：

```python
nums = [3, 7, 2, 9, 5]
print(max(nums))  # 9（内置函数）
```

---

### 七、例题：求和

**问题**：计算列表所有元素的和

```python
nums = [1, 2, 3, 4, 5]

sum_val = 0
for x in nums:
    sum_val += x

print(sum_val)  # 15
```

**Python 简洁写法**：

```python
nums = [1, 2, 3, 4, 5]
print(sum(nums))  # 15（内置函数）
```

---

### 八、列表切片

**取一部分**：

```python
nums = [0, 1, 2, 3, 4, 5]

print(nums[1:4])  # [1, 2, 3]（索引1到3，不包括4）
print(nums[:3])   # [0, 1, 2]（前3个）
print(nums[3:])   # [3, 4, 5]（从索引3开始）
```

**类比**：
```
切片就像"剪一段"
nums[1:4] = 从位置1剪到位置4（不包括4）
```

### 九、常用操作汇总

| 操作 | 代码 | 作用 |
|------|------|------|
| 创建 | `nums = [1,2,3]` | 创建列表 |
| 取值 | `nums[0]` | 取第1个 |
| 修改 | `nums[0] = 10` | 改第1个 |
| 添加 | `nums.append(4)` | 添加到末尾 |
| 删除 | `nums.pop(0)` | 删除第1个 |
| 长度 | `len(nums)` | 元素个数 |
| 最大 | `max(nums)` | 最大值 |
| 最小 | `min(nums)` | 最小值 |
| 求和 | `sum(nums)` | 总和 |
| 排序 | `nums.sort()` | 从小到大排序 |

---

## 总结

| 概念 | 要点 |
|------|------|
| 列表 | 一排编好号的"储物柜" |
| 索引 | 从 0 开始！不是从 1 |
| 遍历 | for x in nums 或 for i in range(len(nums)) |
| 常用函数 | max(), min(), sum(), len() |

**记住**：Python 里叫"列表"，索引从 0 开始！
