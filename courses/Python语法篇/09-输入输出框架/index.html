📋 题目格式示例

【题目描述】

计算两个整数的和。

【输入格式】

输入两个整数 a 和 b。

【输入样例】

3 5

【输出样例】

8

📝 一行输入两个数

输入：3 5（两个数在同一行，空格分隔）

a, b = map(int, input().split())
print(a + b)

拆解过程：

input()
→ "3 5"
.split()
→ ["3","5"]
map(int,...)
→ [3,5]
a,b = ...
→ a=3,b=5

📝 一行输入多个数

输入：1 2 3 4 5（变成列表）

nums = list(map(int, input().split()))
print(nums) # [1, 2, 3, 4, 5]
输入：
1 2 3 4 5
输出：
[1, 2, 3, 4, 5]

📝 多次输入（知道次数）

n = int(input())
for i in range(n):
    x = int(input())
    print(x * 2)
输入：
3
1
2
3
输出：
2
4
6

📝 持续输入（不知道次数）

while True:
    try:
        x = int(input())
        print(x * 2)
    except:
        break
解释： try 尝试读取，except 读不到时退出循环

📤 输出格式技巧

空格分隔

print(a, b, c)
# 输出：1 2 3

保留小数

x = 3.14159
print(f"{x:.2f}")
# 输出：3.14

不换行输出

for i in range(5):
    print(i, end=" ")
# 输出：0 1 2 3 4

格式化字符串

print(f"{a} + {b} = {a+b}")
# 输出：3 + 5 = 8

🧪 小测验

输入 "3 5" 后，`input().split()` 得到什么？

A ["3 5"]（一个字符串）
B ["3", "5"]（两个字符串）
C [3, 5]（两个整数）
D 3 5（两个整数）
