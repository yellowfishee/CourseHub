## 内容

### 一、竞赛题目的格式

**标准格式**：
```
【题目描述】
计算两个整数的和。

【输入格式】
输入两个整数 a 和 b。

【输出格式】
输出 a + b 的值。

【输入样例】
3 5

【输出样例】
8
```

**三步曲**：
```
1. 读取输入
2. 计算结果
3. 输出答案
```

---

### 二、单次输入

**一行输入多个值**：

```python
# 输入：3 5（两个数在同一行）
a, b = map(int, input().split())
print(a + b)
```

**拆解**：
```
input()       → "3 5"（字符串）
.split()      → ["3", "5"]（列表）
map(int, ...) → [3, 5]（整数）
a, b = ...    → a=3, b=5
```

**一行输入多个值（另一种方式）**：

```python
# 输入：1 2 3 4 5
nums = list(map(int, input().split()))
print(nums)  # [1, 2, 3, 4, 5]
```

---

### 三、多次输入

**情况1：知道输入次数**

```python
# 输入：第一行是 n，后面 n 行每行一个数
n = int(input())
for i in range(n):
    x = int(input())
    print(x * 2)
```

**输入示例**：
```
3
1
2
3
```

**输出**：
```
2
4
6
```

---

**情况2：不知道次数，读到结束**

```python
# 持续读取，直到输入结束
while True:
    try:
        x = int(input())
        print(x * 2)
    except:
        break
```

**解释**：
```
try: 尝试读取
except: 读不到时（输入结束），退出循环
```

---

**情况3：每行多个数，多行输入**

```python
# 第一行：n（行数）
# 后面 n 行：每行两个数
n = int(input())
for i in range(n):
    a, b = map(int, input().split())
    print(a + b)
```

**输入示例**：
```
3
1 2
3 4
5 6
```

**输出**：
```
3
7
11
```

### 四、输出格式

**格式1：输出单个结果**

```python
print(result)
```

**格式2：输出多个值（空格分隔）**

```python
print(a, b, c)  # 自动用空格分隔
# 或
print(f"{a} {b} {c}")
```

**格式3：保留小数位数**

```python
# 保留2位小数
x = 3.14159
print(f"{x:.2f}")  # 输出：3.14
```

**格式4：不换行输出**

```python
for i in range(5):
    print(i, end=" ")  # 输出：0 1 2 3 4（不换行）
print()  # 最后换行
```

---

### 五、常用模板

**模板1：一行两个数**

```python
a, b = map(int, input().split())
print(a + b)
```

**模板2：n 个数**

```python
n = int(input())
for i in range(n):
    x = int(input())
    # 处理 x
```

**模板3：列表输入**

```python
nums = list(map(int, input().split()))
# 处理列表
```

**模板4：持续输入**

```python
while True:
    try:
        line = input()
        # 处理 line
    except:
        break
```

### 六、例题：求和

**题目**：输入 n 个数，输出它们的和

**输入**：
```
5
1 2 3 4 5
```

**输出**：
```
15
```

**代码**：

```python
n = int(input())
nums = list(map(int, input().split()))
total = 0
for x in nums:
    total += x
print(total)
```

**更简洁的写法**：

```python
n = int(input())
nums = list(map(int, input().split()))
print(sum(nums))  # Python 内置的 sum 函数
```

> ⚠️ **Python最佳实践**：不要用 `sum` 做变量名，因为 `sum()` 是Python内置函数！用 `total` 代替。

### 七、常见错误

**错误1：忘记 split()**

```python
# 输入：3 5
a = int(input())  # 错误！只能读到 "3 5"，无法转成整数
```

**正确**：
```python
a, b = map(int, input().split())
```

---

**错误2：split() 后忘记 map**

```python
# 输入：3 5
a, b = input().split()  # 得到字符串 "3" 和 "5"
print(a + b)  # 输出 "35"，不是 8！
```

**正确**：
```python
a, b = map(int, input().split())
print(a + b)  # 输出 8
```

---

**错误3：输出格式不对**

```python
# 要求：输出两个数，空格分隔
print(a + b)  # 错误！只输出一个数
```

**正确**：
```python
print(a, b)  # 或 print(f"{a} {b}")
```

---

## 总结

| 格式 | 代码 |
|------|------|
| 一行两个数 | `a, b = map(int, input().split())` |
| 一行多个数 | `nums = list(map(int, input().split()))` |
| n 行输入 | `n = int(input())` + `for i in range(n)` |
| 持续输入 | `while True: try: ... except: break` |
| 保留小数 | `print(f"{x:.2f}")` |
| 不换行 | `print(x, end=" ")` |

**记住**：竞赛题的三步曲：输入 → 计算 → 输出
