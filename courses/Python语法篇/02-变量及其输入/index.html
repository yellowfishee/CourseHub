例题1-1　圆柱体的表面积

输入底面半径 r 和高 h，输出圆柱体的表面积

样例输入

3.5 9

样例输出

274.889

程序1-5　圆柱体的表面积

import math
r, h = map(float, input().split())
s1 = math.pi * r * r
s2 = 2 * math.pi * r * h
s = s1 * 2.0 + s2
print("Area = {:.3f}".format(s))
点击运行 ▶
✓ 执行结果（输入 3.5 9）
输出：Area = 274.889

赋值运算的理解

⚠️ = 不是"等于"！

= 是"赋值"：把右边的值放进左边的变量

a = 5 # a 盒子放 5
a = a + 1 # 算出来 6，再放进 a

变量是"喜新厌旧"的，新的值会覆盖旧的值！

变量交换：三变量法

如何交换 a=3 和 b=5，变成 a=5, b=3？

a = 3
b = 5
t = a # 把 a 的值先存到 t
a = b # 把 b 的值给 a
b = t # 把 t 里保存的值给 b
点击运行 ▶
✓ 执行结果
a = 5
b = 3
