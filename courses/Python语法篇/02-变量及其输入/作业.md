## 参考答案

**选择题：** 1-B, 2-C, 3-C, 4-B, 5-C

**填空题：**
6. 盒子/便利贴
7. 等待用户输入
8. 赋值（把右边的值赋给左边）
9. int, float
10. 临时存放/中转

**编程题（参考答案）：**

```python
# 题目1
a = int(input())
print(a * a)

# 题目2
a = int(input())
b = int(input())
print(a + b)
print(a - b)
print(a * b)

# 题目3
a = 10
b = 20
t = a
a = b
b = t
print(f"a = {a}")
print(f"b = {b}")

# 题目4
n = int(input())
bai = n // 100
shi = n // 10 % 10
ge = n % 10
print(f"百位：{bai}")
print(f"十位：{shi}")
print(f"个位：{ge}")

# 题目5
import math
r = float(input())
h = float(input())
area = 2 * math.pi * r * r + 2 * math.pi * r * h
print(f"Area = {area:.3f}")
```
