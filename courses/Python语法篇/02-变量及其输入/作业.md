## 编程题

### 简单题

**题目1：** 计算平方

**描述：** 让用户输入一个数字，然后输出这个数字的平方。

```
示例输入：5
示例输出：25
```

**提示：** 用 ** 2 或者 a * a

---

**题目2：** 计算矩形周长

**描述：** 输入矩形的宽和高，输出周长。周长 = 2 * (宽 + 高)

```
示例输入：5 8
示例输出：26
```

---

**题目3：** 三数求和

**描述：** 输入三个整数，输出它们的和。

```
示例输入：10 20 30
示例输出：60
```

---

### 中等题

**题目4：** 三位数分解

**描述：** 输入一个三位数，输出它的百位、十位、个位。

```
示例输入：127
示例输出：
百位：1
十位：2
个位：7
```

**提示：** 用 // 和 % 分离各位数字

---

## 参考答案

### 选择题答案
1-B, 2-C, 3-C, 4-C, 5-B

### 编程题参考答案

```python
# 题目1：计算平方
a = int(input())
print(a ** 2)
```

```python
# 题目2：计算矩形周长
width = float(input())
height = float(input())
perimeter = 2 * (width + height)
print(perimeter)
```

```python
# 题目3：三数求和
a = int(input())
b = int(input())
c = int(input())
print(a + b + c)
```

```python
# 题目4：三位数分解
n = int(input())
bai = n // 100
shi = n // 10 % 10
ge = n % 10
print(f"百位：{bai}")
print(f"十位：{shi}")
print(f"个位：{ge}")
```
