### 一、引入：上节课的问题

**提问**：上节课我们学了 print() 可以显示结果。但如果想让计算机"记住"你输入的数字，下次再用，能做到吗？

**回答**：需要用"变量"！

**类比**：
```
变量 = 便利贴
- 写上数字，贴在变量上（声明变量）
- 以后想用这个数字，直接看便利贴
```

---

### 二、变量是什么？

**通俗解释**：
变量就像一个**盒子**，盒子上贴了个名字。盒子里可以放东西（数字），以后想用的时候，找到盒子拿就行了。

**生活例子**：
```
你想记住一个数字，比如"5"
1. 找一个盒子
2. 给盒子贴个名字，比如叫 "a"
3. 把数字 5 放进盒子

→ 以后想知道盒子里是什么，直接看"a"就行了
```

**Python 代码**：

```python
a = 5      # 把数字 5 放进盒子 a
print(a)   # 显示 a 里面的数字
```

**解释**：
- `a = 5` 的意思是：把 5 放进叫 a 的盒子
- `print(a)` 的意思是：显示 a 里面的内容

---

### 三、如何让计算机读取输入？

**问题**：怎么让用户输入一个数字，计算机记住？

**Python 命令**：

```python
a = input()
```

**解释**：
- `input()` = 让计算机等待你输入，然后"记住"
- 输入的内容存在变量 `a` 里面

**完整例子**：

```python
a = input()          # 等待输入，存到 a
print("你输入的是:", a)
```

**类比**：
```
input() = 服务员等你点完菜才走
变量 a = 你的点菜单
```

---

### 四、input() 读取数字

**问题**：input() 读进来的数据是"字符串"，不是数字怎么办？

**解决方法**：用 `int()` 把字符串转成整数

```python
a = int(input())     # 读取输入，并转成整数
```

**或者**：用 `float()` 转成浮点数

```python
b = float(input())   # 读取输入，转成浮点数
```

**例子**：计算 a + b

```python
a = int(input())     # 读取第一个整数
b = int(input())     # 读取第二个整数
print(a + b)         # 输出它们的和
```

**运行示例**：
```
输入：3
输入：5
输出：8
```

### 五、赋值运算的理解

**重要理解**：`=` 不是"等于"，是"赋值"

**错误理解**：
```python
a = 5
a = a + 1   # 5 = 5 + 1 ？？？
```

**正确理解**：
```python
a = 5       # a 盒子里放 5
a = a + 1   # 把 a+1 的结果（6）再放到 a 里
            # 原来的 5 被覆盖了
```

**类比**：
```
a = 5   →  盒子里放 5
a = a+1 →  算出来 5+1=6，再把 6 放进盒子（原来的 5 丢了）
```

**特点**：变量是"喜新厌旧"的，新的值会覆盖旧的值。

### 六、整型和浮点型

**两种常见的数字类型**：

| 类型 | 名字 | 例子 | 用途 |
|------|------|------|------|
| int | 整数 | 1, 100, -5 | 人数、个数 |
| float | 浮点数 | 3.14, -0.5 | 金钱、重量 |

**转换函数**：

```python
int(3.14)    # → 3（取整数部分）
float(5)     # → 5.0
```

**为什么需要区分**？

```python
print(8 / 5)    # → 1.6（浮点数除法）
print(8 // 5)   # → 1（整数除法）
```

### 七、变量交换

**问题**：如何交换两个变量的值？比如 a=3, b=5，变成 a=5, b=3？

**方法1：三变量法（通用方法）**

```python
a = 3
b = 5
t = a    # 把 a 的值先存到 t
a = b    # 把 b 的值给 a
b = t    # 把 t 里保存的值给 b
print(a, b)   # 输出 5 3
```

**类比**：
```
借一个空杯子来交换酱油和醋：
1. 把酱油倒进空杯子（t = a）
2. 把醋倒进酱油瓶子（a = b）
3. 把酱油倒进醋瓶子（b = t）
```

**方法2：Python特有写法**

```python
a = 3
b = 5
a, b = b, a   # 一行搞定！
print(a, b)   # 输出 5 3
```

> ⚠️ **Python优势**：用 `a, b = b, a` 可以直接交换，不需要第三个变量！这是Python特有的"元组解包"功能。

### 八、综合例题

**例题**：计算圆柱体表面积

**公式**：表面积 = 2πr² + 2πrh

```python
import math

r = float(input())   # 输入半径
h = float(input())   # 输入高度

pi = math.pi         # π 的值
area = 2 * pi * r * r + 2 * pi * r * h

print("Area = %.3f" % area)
```

**拆解问题**：
1. 导入 math 模块
2. 读取 r 和 h
3. 用公式计算
4. 格式化输出
