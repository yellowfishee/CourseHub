## 内容

### 一、引入：什么是模块？

**问题**：Python有那么多功能，都在一个文件里吗？

**答案**：不是！Python把功能分成不同的模块，按需导入。

**类比**：
```
Python = 一个大工具箱
模块 = 工具箱里的小盒子
  - math盒子：数学工具
  - random盒子：随机数工具
  - datetime盒子：日期时间工具
用之前要先打开盒子（导入）
```

---

### 二、导入模块

**方式1：import 模块名**

```python
import math

print(math.sqrt(16))  # 4.0
print(math.pi)        # 3.141592653589793
```

**方式2：from 模块名 import 功能**

```python
from math import sqrt, pi

print(sqrt(16))  # 4.0（不用写math.）
print(pi)        # 3.141592653589793
```

**方式3：from 模块名 import *（不推荐）**

```python
from math import *  # 导入所有功能

print(sqrt(16))  # 4.0
# 问题：不知道sqrt来自哪里，容易冲突
```

**方式4：给模块起别名**

```python
import math as m

print(m.sqrt(16))  # 4.0
```

### 三、math模块 - 数学工具

**常用函数**：

| 函数 | 作用 | 例子 |
|------|------|------|
| sqrt(x) | 平方根 | sqrt(16) = 4.0 |
| pow(x, y) | x的y次方 | pow(2, 3) = 8.0 |
| ceil(x) | 向上取整 | ceil(3.1) = 4 |
| floor(x) | 向下取整 | floor(3.9) = 3 |
| round(x) | 四舍五入 | round(3.5) = 4 |
| abs(x) | 绝对值 | abs(-5) = 5 |

**常用常量**：

```python
import math

print(math.pi)  # 3.141592653589793（圆周率）
print(math.e)   # 2.718281828459045（自然常数）
```

**例子**：计算圆的面积

```python
import math

def circle_area(radius):
    return math.pi * radius ** 2

print(circle_area(5))  # 78.53981633974483
```

### 四、random模块 - 随机数工具

**常用函数**：

| 函数 | 作用 | 例子 |
|------|------|------|
| random() | 0~1随机小数 | random() |
| randint(a, b) | a~b随机整数 | randint(1, 100) |
| choice(seq) | 随机选一个 | choice([1,2,3]) |
| shuffle(seq) | 随机打乱 | shuffle(my_list) |

**例子**：

```python
import random

# 随机整数
print(random.randint(1, 100))  # 1~100之间的随机整数

# 随机选择
fruits = ["apple", "banana", "orange"]
print(random.choice(fruits))  # 随机选一个水果

# 随机打乱
cards = [1, 2, 3, 4, 5]
random.shuffle(cards)
print(cards)  # [3, 1, 5, 2, 4] 顺序随机
```

**例题：猜数字游戏**

```python
import random

secret = random.randint(1, 100)
guess = 0

while guess != secret:
    guess = int(input("猜一个1-100的数字: "))
    if guess < secret:
        print("太小了")
    elif guess > secret:
        print("太大了")
    else:
        print("猜对了！")
```

### 五、datetime模块 - 日期时间工具

**获取当前时间**：

```python
from datetime import datetime

now = datetime.now()
print(now)  # 2024-01-15 14:30:25.123456

print(now.year)   # 2024
print(now.month)  # 1
print(now.day)    # 15
print(now.hour)   # 14
print(now.minute) # 30
print(now.second) # 25
```

**格式化输出**：

```python
from datetime import datetime

now = datetime.now()
print(now.strftime("%Y-%m-%d"))       # 2024-01-15
print(now.strftime("%Y年%m月%d日"))    # 2024年01月15日
print(now.strftime("%H:%M:%S"))       # 14:30:25
```

**日期计算**：

```python
from datetime import datetime, timedelta

now = datetime.now()
tomorrow = now + timedelta(days=1)
next_week = now + timedelta(weeks=1)

print(tomorrow)
print(next_week)
```

### 六、第三方库

**什么是第三方库？**

别人写好的代码，可以安装使用。

**安装方法**：

```bash
pip install 库名
```

**常用第三方库**：

| 库名 | 用途 |
|------|------|
| requests | 网络请求 |
| numpy | 科学计算 |
| pandas | 数据分析 |
| matplotlib | 绑图 |
| pygame | 游戏开发 |

**例子**：安装并使用requests

```bash
pip install requests
```

```python
import requests

response = requests.get("https://api.github.com")
print(response.status_code)  # 200
```

### 七、自定义模块

**创建自己的模块**：

```python
# mymodule.py
def greet(name):
    return f"Hello, {name}!"

pi = 3.14159
```

**使用自己的模块**：

```python
import mymodule

print(mymodule.greet("World"))  # Hello, World!
print(mymodule.pi)              # 3.14159
```

### 八、常见错误

**错误1：模块不存在**

```python
import not_exist_module  # ModuleNotFoundError!
```

**错误2：拼写错误**

```python
import Math  # 错误！应该是小写 math
```

**错误3：忘记导入**

```python
print(sqrt(16))  # NameError! 忘记 import math
```

**正确**：

```python
import math
print(math.sqrt(16))  # 4.0
```

## 总结

| 概念 | 要点 |
|------|------|
| 模块 | Python的工具箱 |
| import | 导入整个模块 |
| from...import | 导入特定功能 |
| math | 数学函数 |
| random | 随机数 |
| datetime | 日期时间 |

**记住**：
- 用之前先导入
- 推荐用 `import 模块名` 或 `from 模块名 import 功能`
- 第三方库用 `pip install` 安装
