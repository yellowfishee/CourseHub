## 参考答案

**选择题：** 1-B, 2-C, 3-B, 4-B, 5-B

**填空题：**
6. 条件
7. 英文冒号 `:`
8. else
9. if、else
10. 与（或）、或（与）

**编程题（参考答案）：**

```python
# 题目1
n = int(input())
if n > 0:
    print("正数")
elif n < 0:
    print("负数")
else:
    print("非负数")

# 题目2
score = int(input())
if score >= 60:
    print("及格")
else:
    print("不及格")

# 题目3
n = int(input())
if n % 2 == 0:
    print("偶数")
else:
    print("奇数")

# 题目4
a = int(input())
b = int(input())
c = int(input())
if a >= b and a >= c:
    print(a)
elif b >= a and b >= c:
    print(b)
else:
    print(c)

# 例题1-4 鸡兔同笼
n = int(input())
m = int(input())
a = (4 * n - m) // 2
b = n - a
if m % 2 == 1 or a < 0 or b < 0:
    print("No answer")
else:
    print(f"{a} {b}")

# 例题1-5 三整数排序
a = int(input())
b = int(input())
c = int(input())
if a > b:
    a, b = b, a
if a > c:
    a, c = c, a
if b > c:
    b, c = c, b
print(a, b, c)

# 习题1-1 平均数
a = float(input())
b = float(input())
c = float(input())
print(f"{(a + b + c) / 3:.3f}")

# 习题1-6 三角形
a = int(input())
b = int(input())
c = int(input())
# 判断能否构成三角形
if a + b <= c or a + c <= b or b + c <= a:
    print("not a triangle")
else:
    # 判断是否为直角三角形
    sides = [a, b, c]
    sides.sort()
    if sides[2]**2 == sides[0]**2 + sides[1]**2:
        print("yes")
    else:
        print("no")

# 习题1-7 年份
year = int(input())
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print("yes")
else:
    print("no")
```
