## 编程题

### 简单题

**题目1：** 判断正负

**描述：** 输入一个整数，判断是正数还是负数（0算作非负数）。

```
示例输入1：5
示例输出1：正数

示例输入2：-3
示例输出2：负数

示例输入3：0
示例输出3：非负数
```

---

**题目2：** 判断奇偶

**描述：** 输入一个整数，判断是奇数还是偶数。

```
示例输入1：7
示例输出1：奇数

示例输入2：10
示例输出2：偶数
```

---

**题目3：** 判断及格

**描述：** 输入一个成绩（0-100），判断是否及格（60分及以上）。

```
示例输入1：75
示例输出1：及格

示例输入2：45
示例输出2：不及格
```

---

### 中等题

**题目4：** 判断闰年

**描述：** 输入年份，判断是否为闰年。
闰年条件：能被4整除但不能被100整除，或者能被400整除。

```
示例输入1：2024
示例输出1：yes

示例输入2：2023
示例输出2：no

示例输入3：2000
示例输出3：yes

示例输入4：1900
示例输出4：no
```

**提示：** 闰年的逻辑表达式

## 参考答案

### 选择题答案
1-B, 2-C, 3-B, 4-B, 5-B

### 编程题参考答案

```python
# 题目1：判断正负
n = int(input())
if n > 0:
    print("正数")
elif n < 0:
    print("负数")
else:
    print("非负数")
```

```python
# 题目2：判断奇偶
n = int(input())
if n % 2 == 0:
    print("偶数")
else:
    print("奇数")
```

```python
# 题目3：判断及格
score = int(input())
if score >= 60:
    print("及格")
else:
    print("不及格")
```

```python
# 题目4：判断闰年
year = int(input())
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print("yes")
else:
    print("no")
```
