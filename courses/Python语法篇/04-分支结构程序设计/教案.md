## 学习目标

- 理解**分支结构**：遇到岔路口做选择
- 学会用 **if** 语句：如果...那么...
- 学会用 **if...else**：两条路选择
- 学会用 **if...elif...else**：多条路选择
- 了解**逻辑运算**：与（and）、或（or）、非（not）
- 能用分支结构解决实际问题

---

## 内容

### 一、引入：生活中的选择

**提问**：如果你今天饿了，会怎么做？

**回答**：
- 饿了就吃饭
- 不饿就继续工作

**类比**：程序也会"选择"——根据条件决定下一步做什么。

---

### 二、if 语句：如果...那么...

**语法**：

```python
if 条件:
    执行的代码
```

**例子**：如果成绩及格，就打印"及格了"

```python
score = int(input())

if score >= 60:
    print("及格了！")
```

**解释**：
- `if` 后面的条件成立（True）时，才会执行缩进的代码
- 条件不成立（False）时，什么都不做

---

### 三、if...else：如果...否则...

**语法**：

```python
if 条件:
    执行的代码1
else:
    执行的代码2
```

**例子**：

```python
score = int(input())

if score >= 60:
    print("及格了！")
else:
    print("没及格，再接再厉！")
```

**流程图**：
```
score >= 60 ?
    ↓
 是 → 打印"及格了！"
 否 → 打印"没及格！"
```

---

### 四、if...elif...else：多条路选择

**语法**：

```python
if 条件1:
    代码1
elif 条件2:
    代码2
elif 条件3:
    代码3
else:
    代码4
```

**例子**：成绩评级

```python
score = int(input())

if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好")
elif score >= 60:
    print("及格")
else:
    print("不及格")
```

---

### 五、比较运算符

**常用的条件判断**：

| 运算符 | 含义 | 例子 |
|--------|------|------|
| == | 等于 | a == b |
| != | 不等于 | a != b |
| > | 大于 | a > b |
| < | 小于 | a < b |
| >= | 大于等于 | a >= b |
| <= | 小于等于 | a <= b |

**注意**：`=` 是赋值，`==` 是比较是否相等！

---

### 六、逻辑运算：与、或、非

**与（and）**：两个条件都成立才成立

```python
if score >= 60 and score < 90:
    print("及格了但不是满分")
```

**或（or）**：至少一个条件成立就成立

```python
if score == 100 or score < 60:
    print("满分或不及格")
```

**非（not）**：取反

```python
if not score < 60:
    print("及格了")
```

---

### 七、综合例题：判断奇偶数

**题目**：输入一个整数，判断是奇数还是偶数

```python
n = int(input())

if n % 2 == 0:
    print("偶数")
else:
    print("奇数")
```

**拆解**：
- 能被2整除 → 偶数（n % 2 == 0）
- 不能被2整除 → 奇数

---

### 八、综合例题：成绩评级

```python
score = int(input())

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(f"等级：{grade}")
```

## 总结

今天我们学了：

| 概念 | Python 写法 | 作用 |
|------|------------|------|
| 如果 | if 条件: | 条件满足时执行 |
| 否则 | else: | 条件不满足时执行 |
| 否则如果 | elif 条件: | 多个条件判断 |
| 与 | and | 两个条件都满足 |
| 或 | or | 满足其中一个 |
| 非 | not | 取反 |

**记住**：分支结构就是"岔路口"，根据条件选择走哪条路！
