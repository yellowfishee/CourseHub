例题1-4　鸡兔同笼

已知鸡和兔的总数量为n，总腿数为m

输入n和m，输出鸡和兔的数目，无解输出No answer

样例输入1：14 32

样例输出1：12 2

样例输入2：10 16

样例输出2：No answer

程序1-11　鸡兔同笼

n, m = map(int, input().split())
a = (4*n - m) // 2
b = n - a
if m % 2 == 1 or a < 0 or b < 0:
    print("No answer")
else:
    print(a, b)
点击运行 ▶
✓ 执行结果
输入 14 32 → 输出：12 2
输入 10 16 → 输出：No answer

例题1-5　三整数排序

输入3个整数，从小到大排序后输出

样例输入：20 7 33　　样例输出：7 20 33

⚠️ 注意：输入"111"时，程序1-12会输出6次"111"！

程序1-13　三整数排序（elif版）

a, b, c = map(int, input().split())
if a <= b <= c: print(a, b, c)
elif a <= c <= b: print(a, c, b)
elif b <= a <= c: print(b, a, c)
elif b <= c <= a: print(b, c, a)
elif c <= a <= b: print(c, a, b)
else: print(c, b, a)
点击运行 ▶
✓ 执行结果（输入 20 7 33）
输出：7 20 33

程序1-14　三整数排序（两两交换）

a, b, c = map(int, input().split())
if a > b: a, b = b, a # 执行完后 a≤b
if a > c: a, c = c, a # 执行完后 a≤c，且a≤b依然成立
if b > c: b, c = c, b
print(a, b, c)

✓ a已经是3个数中的最小值，只需检查b和c的顺序

if...elif...else

score = int(input())

if score >= 90:
