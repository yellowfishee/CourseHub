🏃 循环要花多少时间？

⚡

代码A：循环100次

for i in range(100):
    print(i)
✓ 快！
🐢

代码B：循环1000000次

for i in range(1000000):
    print(i)
✗ 慢！
循环次数越多 = 花的时间越长 ⏰

        print(i, j)

10

📊 时间复杂度是什么？

描述代码执行次数的增长趋势

循环次数 表示法 名称 类比
常数次 O(1) 常数 跑1圈
N 次 O(N) 线性 跑N圈
N×N 次 O(N²) 平方 跑N×N圈
N×N×N 次 O(N³) 立方 跑N³圈

📈 不同 N 值的执行次数

N 的值 O(1) O(N) O(N²) O(N³)
10 1 10 100 1,000
100 1 100 10,000 1,000,000
1000 1 1,000 1,000,000 1,000,000,000
⚠️ 警告：N² 和 N³ 增长得非常快！
1000³ = 1,000,000,000（十亿次）→ 可能超时！

🔍 实例分析

O(N) 代码

n = 1000
for i in range(n):
    print(i)
执行次数： 1000

O(N²) 代码

n = 1000
for i in range(n):
    for j in range(n):
        print(i, j)
执行次数： 1,000,000

⏰ 竞赛中的时间限制

时间限制 能承受的复杂度 N 的范围
1秒 O(N) N ≤ 10⁷（一千万）
1秒 O(N²) N ≤ 10³~10⁴（几千）
1秒 O(N³) N ≤ 10²（几百）
💡 判断方法
N=10000 → O(N²) 会执行 1亿次 → 可能超时！

🚀 如何优化循环？

思路1：减少循环层数

❌ 双层循环 O(N²)
✓ 单层循环 O(N)

思路2：利用数学公式

❌ 循环求和 O(N)
total = 0
for i in range(1, n+1):
    total += i
✓ 公式求和 O(1)
total = n * (n+1) // 2

📝 总结

概念 含义 危险程度
O(1) 常数次 🟢 安全
O(N) 单层循环 🟢 安全
O(N²) 双层嵌套 🟡 注意
O(N³) 三层嵌套 🔴 危险
循环层数越多，代价越大！
学会用公式代替循环，能让代码更快 ⚡
