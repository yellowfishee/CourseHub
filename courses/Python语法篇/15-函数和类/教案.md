### 一、引入：为什么需要函数？

**问题**：如果要多次做同样的事？

```python
print("欢迎小明！")
print("今天天气不错")
print("祝你学习愉快")

print("欢迎小红！")
print("今天天气不错")
print("祝你学习愉快")

print("欢迎小刚！")
print("今天天气不错")
print("祝你学习愉快")
```

**问题**：重复写了3遍同样的代码，太麻烦！

**解决方法**：用函数把代码打包起来

### 二、什么是函数？

**生活类比**：
```
函数 = 微波炉

放进食物（输入）→ 设置时间和模式（参数）→ 加热（执行）→ 拿出食物（输出）

你不需要知道微波炉内部怎么工作，只需要会用就行
```

**Python 函数**：
```python
def 欢迎问候(名字):
    print(f"欢迎{名字}！")
    print("今天天气不错")
    print("祝你学习愉快")

# 调用函数
欢迎问候("小明")
欢迎问候("小红")
欢迎问候("小刚")
```

### 三、定义函数

**基本结构**：

```python
def 函数名(参数):
    函数体（要执行的代码）
    return 返回值
```

**例1：没有参数的函数**

```python
def say_hello():
    print("Hello!")

say_hello()  # 调用函数
say_hello()  # 再调用一次
```

**例2：有参数的函数**

```python
def greet(name):
    print(f"你好，{name}！")

greet("小明")  # 你好，小明！
greet("小红")  # 你好，小红！
```

**例3：有返回值的函数**

```python
def add(a, b):
    return a + b

result = add(3, 5)
print(result)  # 8
```

### 四、参数和返回值

**参数**：函数的"输入"

```python
def power(base, exp):  # 两个参数
    return base ** exp

print(power(2, 3))  # 8（2的3次方）
print(power(5, 2))  # 25（5的2次方）
```

**返回值**：函数的"输出"

```python
def is_even(n):
    if n % 2 == 0:
        return True
    else:
        return False

if is_even(6):
    print("偶数")
```

**类比**：
```
参数 = 放进微波炉的食物
返回值 = 拿出来的食物
```

### 五、例题：计算阶乘

**问题**：用函数计算 n 的阶乘

```python
def factorial(n):
    result = 1
    for i in range(1, n + 1):
        result *= i
    return result

print(factorial(5))  # 120（5! = 5×4×3×2×1）
print(factorial(10)) # 3628800
```

### 六、类

**什么是类？**

把多个相关的数据和方法打包在一起。

**生活类比**：
```
学生档案：
- 姓名
- 年龄
- 班级
- 自我介绍（方法）

把这些信息打包在一起，就是一个"类"
```

> ⚠️ **Python术语**：Python 用"类"（class）来打包数据，C/C++ 用"结构体"（struct）。概念相似，但 Python 的类更强大，还可以包含方法。

**Python 的类**：

```python
class Student:
    def __init__(self, name, age, grade):
        self.name = name
        self.age = age
        self.grade = grade

# 创建学生对象
stu1 = Student("小明", 12, "六年级")
stu2 = Student("小红", 13, "初一")

# 使用
print(stu1.name)   # 小明
print(stu2.age)    # 13
```

**解释**：
```
class：定义一个"类型"
__init__：初始化函数（创建对象时自动调用）
self：代表"这个对象自己"
```

---

### 七、类中的方法

**方法**：类里面的函数

```python
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def introduce(self):
        print(f"我叫{self.name}，今年{self.age}岁")

    def grow_up(self):
        self.age += 1
        print(f"长大了一岁，现在{self.age}岁")

stu = Student("小明", 12)
stu.introduce()  # 我叫小明，今年12岁
stu.grow_up()    # 长大了一岁，现在13岁
```

---

### 八、函数 vs 方法

| 特点 | 函数 | 方法 |
|------|------|------|
| 定义位置 | 独立定义 | 在类里面定义 |
| 调用方式 | `函数名(参数)` | `对象.方法名(参数)` |
| 第一个参数 | 直接写参数 | 自动传入 self |

### 九、常见错误

**错误1：忘记调用函数**

```python
def say_hello():
    print("Hello!")

say_hello  # 错误！只是引用，没有调用
say_hello()  # 正确！加了括号才是调用
```

**错误2：忘记 return**

```python
def add(a, b):
    print(a + b)  # 只是打印，没有返回

result = add(1, 2)  # result 是 None！
```

**正确**：
```python
def add(a, b):
    return a + b  # 返回结果

result = add(1, 2)  # result 是 3
```

**错误3：忘记 self**

```python
class Student:
    def introduce():  # 错误！缺少 self
        print("hello")
```

**正确**：
```python
class Student:
    def introduce(self):  # 正确！有 self
        print("hello")
```

## 总结

| 概念 | 要点 |
|------|------|
| 函数 | 打包代码，可以重复使用 |
| 参数 | 函数的输入 |
| 返回值 | 函数的输出（return） |
| 类 | 打包数据和方法 |
| self | 代表"这个对象自己" |

**记住**：
- 函数用 `def` 定义，用 `函数名()` 调用
- 类用 `class` 定义，用 `类名()` 创建对象
