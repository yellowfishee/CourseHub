❓ 为什么需要函数？

如果要多次做同样的事...

print("欢迎小明！")
print("今天天气不错")
print("祝你学习愉快")

print("欢迎小红！")
print("今天天气不错")
print("祝你学习愉快")

# 重复写了很多遍... 😫
问题： 代码重复，难以维护

🔥 函数类比：微波炉

🔥
微波炉
1. 放进食物 → 输入（参数）
2. 设置时间 → 参数设置
3. 加热 → 执行代码
4. 拿出食物 → 输出（返回值）

💡 用函数解决

def 欢迎问候(名字):
    print(f"欢迎{名字}！")
    print("今天天气不错")

欢迎问候("小明")
欢迎问候("小红")

✓ 只写一遍，可以多次使用！

📝 定义函数的基本结构

def 函数名(参数):

    return 返回值

📊 有参数有返回值的函数

def add(a, b):
    return a + b

result = add(3, 5)
print(result) # 8

输入

a=3, b=5

输出

return 8

📊 例题：计算阶乘

用函数计算 n 的阶乘

def factorial(n):
    result = 1
    for i in range(1, n+1):
        result *= i
    return result

print(factorial(5)) # 120
print(factorial(10)) # 3628800
5!
= 120
10!
= 3628800

📋 类：打包数据和方法

📁 学生档案

姓名：小明

年龄：12

班级：六年级

把这些信息打包在一起

💻 Python 类

class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

stu = Student("小明", 12)
print(stu.name) # 小明

🔧 类中的方法

方法 = 类里面的函数

class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def introduce(self):
        print(f"我叫{self.name}，{self.age}岁")

stu = Student("小明", 12)
stu.introduce() # 我叫小明，12岁

🧪 小测验

调用函数需要什么？

A 只写函数名：say_hello
B 函数名加括号：say_hello()
C 写 def say_hello()
D 写 return say_hello

📝 总结

概念 要点
函数 打包代码，可以重复使用
参数 函数的输入
返回值 函数的输出（return）
类 打包数据和方法
self 代表"这个对象自己"
函数用 def 定义，类用 class 定义
