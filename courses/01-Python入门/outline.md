# Python 入门课程

这是一门面向零基础学员的Python入门课程。通过本课程，你将掌握Python的基本语法和编程思想，最终能够独立完成洛谷 NOI 入门级别的题目。

本课程共分为 7 个章节，20+ 课时。

## 题目说明

本课程精选洛谷 NOI 入门级题目，特点：

- **循序渐进**：从简单计算到复杂逻辑
- **概念对应**：每道题对应一个或多个核心概念
- **答案可验证**：洛谷在线评测，即时反馈对错
- **趣味性强**：包含图形输出、数字谜题等趣味题
