### 二、什么是计算机

#### 计算机的本质

计算机（Computer）= 会计算的机器

**核心流程**：接收输入 → 处理数据 → 输出结果

| 输入 | 处理 | 输出 |
|------|------|------|
| 键盘敲字 | CPU计算 | 屏幕显示 |
| 扫码支付 | 程序运算 | 支付成功 |
| 语音指令 | AI识别 | 语音回复 |

#### CPU是大脑

- CPU = Central Processing Unit（中央处理器）
- 负责思考和计算
- 速度极快，一秒能处理几十亿次运算

#### 内存是工作台

- 正在处理的数据会放在内存里
- 速度快，但断电后数据丢失
- 像我们的工作台一样

#### 硬盘是仓库

- 长期存储数据的地方
- 断电后数据保留
- 像仓库一样

### 三、什么是编程

**编程 = 告诉计算机要做什么**

人和计算机对话的语言，叫做"编程语言"。

#### 为什么需要编程语言？

- 计算机只懂0和1（二进制）
- 二进制太难记忆和书写
- 编程语言是人类能理解的形式

#### 编程就是写代码

```
代码示例：
print("Hello, World!")
```

这段代码的作用是：在屏幕上显示"Hello, World!"

### 五、为什么学编程

#### 1. 未来必备技能

- 数字化时代，编程和读书写字一样重要
- 几乎每个行业都需要编程人才

#### 2. 高薪就业

- 程序员工资普遍较高
- 人才缺口大，找工作容易

#### 3. 培养思维能力

- 逻辑思维
- 解决问题的能力
- 耐心和细心

### 六、为什么选择Python

#### Python的优势

| 特点 | 说明 |
|------|------|
| 简单易学 | 英语一样的语法，看代码像看文章 |
| 功能强大 | Web开发、数据分析、AI都能做 |
| 应用广泛 | Google、Facebook、Netflix都在用 |
| 社区活跃 | 遇到问题很容易找到答案 |

#### Python能做什么

- **网站开发**：Instagram、YouTube后台
- **数据分析**：处理海量数据，发现规律
- **人工智能**：机器学习、深度学习
- **自动化办公**：解放双手，提高效率
- **游戏开发**：简单2D游戏
- **爬虫**：获取网络数据
