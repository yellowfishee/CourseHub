### 二、什么是变量？（让计算机记住数据）

#### 生活类比：便利贴

```
你在便利贴上写"小明"，贴在电脑上
→ 这就是"变量"
→ 便利贴上的"小明"是"变量值"
→ 便利贴本身是"变量名"
```

#### 通俗解释

**变量** = 让计算机**记住**某个数据的方法

- **变量名** = 便利贴上的标签（用来找到它）
- **变量值** = 便利贴上写的内容（记住的数据）

#### 代码示例

```python
name = "小明"
```

**解释**：
- `name` = 变量名（便利贴的标签）
- `"小明"` = 变量值（便利贴上写的内容）
- `=` = 把右边的东西记在左边的名字下面

#### 记忆公式

```
变量名 = 变量值
  ↓
记住 = 记住什么
```

---

### 三、使用变量 = 使用变量里的值

#### 重点强调

**使用变量名，得到的是变量值！**

```python
name = "小明"
print(name)      # 显示：小明（不是显示"name"）
```

| 代码 | 显示什么 | 解释 |
|------|---------|------|
| `print(name)` | 小明 | 使用变量，得到的是变量里的值 |
| `print("name")` | name | 这是一句话，显示字面内容 |

#### 区别总结

```
print(名字)   → 名字是变量名，显示变量的值
print("名字") → "名字"是文字，显示字面内容
```

### 四、input() = 提问（让用户输入）

#### 生活类比

```
input() = 问一个问题，等对方回答
就像主持人问"你叫什么名字？" → 等你回答
```

#### 代码示例

```python
name = input("你叫什么名字？")
```

**解释**：
1. 计算机问："你叫什么名字？"
2. 你输入回答（比如"小红"）
3. 计算机把"小红"**记住**在 `name` 这个变量里

#### print() 和 input() 的区别

| 命令 | 做什么 | 类比 |
|------|--------|------|
| `print("你好")` | 显示"你好" | 回答问题 |
| `input("你叫什么？")` | 提问，等你输入 | 问问题 |

### 五、字符串 vs 整数（文字 vs 数字）

#### 生活类比

```
"123" = 写在纸上的数字（是文字）
 123  = 真正的数字（可以计算）

"5" + "3" = "53"（文字拼接）
 5 + 3   = 8（数字相加）
```

#### 本质区别

| 类型 | 例子 | 能计算吗 | 显示什么 |
|------|------|---------|---------|
| 字符串（文字） | `"5"`、`"hello"` | 不能 | 显示字面内容 |
| 整数（数字） | `5`、`100` | 能 | 真正的数字 |

#### input() 返回的是什么？

**input() 返回的是字符串！**

```python
age = input("你几岁？")
# 如果你输入 18
# age 的值是 "18"（字符串），不是 18（数字）
```

### 六、int() = 把文字变成数字

#### 为什么需要 int()？

```
你输入：5 3
→ input() 返回："5" 和 "3"（字符串）
→ "5" + "3" = "53"（文字拼接，错了！）
→ 需要先变成真正的数字
```

#### int() 的作用

```python
a = input("每人几个苹果？")   # 输入 5
b = input("几个同学？")       # 输入 3
# 此时 a = "5", b = "3"（都是字符串！）

a = int(a)  # 把 "5" 变成 5
b = int(b)  # 把 "3" 变成 3

print(a * b)  # 5 * 3 = 15，正确！
```

#### 简化写法

```python
a = int(input("每人几个苹果？"))  # 先输入，转成数字
b = int(input("几个同学？"))
print(a * b)
```

### 七、算法流程：输入 → 处理 → 输出

#### 生活类比：餐厅点餐

```
输入：服务员问你要几个苹果
处理：收银台计算总价
输出：告诉你总价多少钱
```

#### 苹果采购问题

**问题**：每人 5 个苹果，3 个同学，一共要买几个？

**拆解**：
```
大问题：计算总数
  ↓ 拆解
子问题1：每人几个？→ 5（输入）
子问题2：几个人？→ 3（输入）
子问题3：怎么算总数？→ 5 × 3（处理）
子问题4：显示结果→ print（输出）
```

**代码**：
```python
a = int(input())
b = int(input())
print(a * b)
```

### 九、总结

#### 今天学到的命令

| 命令 | 作用 | 类比 |
|------|------|------|
| `=` | 记住数据 | 写便利贴 |
| `变量名` | 使用变量里的值 | 读便利贴 |
| `input()` | 提问等输入 | 问问题 |
| `print()` | 显示内容 | 回答 |
| `int()` | 文字变数字 | 把文字变成能算的数字 |

#### 记住这个区别

```
print(名字)   → 名字是变量，显示变量的值
print("名字") → "名字"是文字，显示字面内容
```

#### 算法三步曲

```
输入 → 处理 → 输出
提问   计算   显示
```
