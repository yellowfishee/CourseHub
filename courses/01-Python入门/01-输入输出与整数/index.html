变量

什么是变量？

📝
便利贴
你在便利贴上写"小明"，贴在电脑上
变量 = 让计算机记住数据的方法
变量名
便利贴的标签（用来找到它）
变量值
便利贴上写的内容（记住的数据）
变量

变量的代码

name = "小明"
name
变量名（记住：便利贴标签）
"小明"
变量值（记住：便利贴内容）
=
记住（把右边的存在左边）

记忆公式：变量名 = 变量值（记住 = 记住什么）

重点！

使用变量 = 使用变量里的值

使用变量名，得到的是变量值！

name = "小明"
print(name) # 显示：小明
print("name") # 显示：name
print(name)
显示变量的值 → 小明
≠
print("name")
显示字面内容 → name
input() 和 print()

提问 vs 回答

❓
input() = 提问
等用户输入答案
name = input("你叫什么？")
💬
print() = 回答
直接显示内容
print("你好")

就像主持人问问题（input）vs 回答问题（print）

数据类型

字符串 vs 整数

文字 vs 数字
📝
字符串（文字）
"5"、"123"、"hello"
不能计算，只能显示
"5" + "3" = "53"
文字拼接，不是相加！
🔢
整数（数字）
5、123、100
能计算，能相加
5 + 3 = 8
真正的数字相加！
重要！

input() 返回的是字符串！

不管你输入什么，都是文字字符串

age = input("你几岁？")
# 输入 18
# age = "18"（字符串！不是数字！）
你输入
18
→
input()
"18"
→
变量 age
字符串"18"
类型转换

int() = 把文字变成数字

🔄
int() = 翻译器
把"123"翻译成 123（字符串→整数）
a = int(input())
# 输入 5 → a = 5（真正的数字！）
