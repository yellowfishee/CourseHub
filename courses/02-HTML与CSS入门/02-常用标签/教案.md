# 常用标签

## 学习目标

- 掌握标题标签 h1 ~ h6
- 掌握段落和换行
- 掌握链接和图片标签

## 常用标签一览

### 标题

```html
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
```

### 文本

```html
<p>这是一个段落</p>
<br>  <!-- 换行 -->
```

### 链接与图片

```html
<a href="https://example.com">访问链接</a>
<img src="logo.png" alt="图片描述">
```

## 标签分类

- **块级标签**：独占一行，如 `<h1>`, `<p>`, `<div>`
- **行内标签**：不独占一行，如 `<a>`, `<span>`
