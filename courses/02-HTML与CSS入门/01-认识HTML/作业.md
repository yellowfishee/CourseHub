# 作业：认识 HTML

## 基础练习

### 练习 1：创建你的第一个网页

请创建一个 HTML 文件，名为 `index.html`，包含以下内容：

1. 正确的 DOCTYPE 声明
2. `<html>`, `<head>`, `<body>` 结构
3. 在 `<head>` 中设置字符编码为 UTF-8
4. 在 `<body>` 中添加一个 `<h1>` 标题，内容为"我的第一个网页"

### 练习 2：完善网页内容

在上一题的基础上，添加：

- 一个 `<h2>` 副标题："这是我的个人主页"
- 一个 `<p>` 段落：介绍你自己（至少 2 句话）
- 一个 `<h3>` 标题："我的爱好"
- 至少 3 个列表项（`<ul>` 和 `<li>`）

## 提交要求

1. 代码必须包含正确的 HTML 结构
2. 所有标签必须正确闭合
3. 文件保存为 `index.html`，用浏览器打开验证

---

## 参考答案

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>我的网页</title>
</head>
<body>
    <h1>我的第一个网页</h1>
    <h2>这是我的个人主页</h2>
    <p>我叫小明，正在学习网页开发。这很有趣！</p>

    <h3>我的爱好</h3>
    <ul>
        <li>编程</li>
        <li>阅读</li>
        <li>运动</li>
    </ul>
</body>
</html>
```
