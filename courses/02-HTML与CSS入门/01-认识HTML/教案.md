# 认识HTML

## 学习目标

- 理解什么是 HTML
- 掌握 HTML 文档的基本结构
- 学会创建第一个网页

## 什么是 HTML

HTML（HyperText Markup Language）是超文本标记语言，用于描述网页的结构。

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>我的网页</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>
```

## HTML 文档结构

- `<!DOCTYPE html>` - 声明文档类型
- `<html>` - 根元素
- `<head>` - 头部信息
- `<body>` - 主体内容
