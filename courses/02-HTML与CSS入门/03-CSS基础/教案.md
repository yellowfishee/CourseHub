# CSS基础

## 学习目标

- 理解什么是 CSS
- 掌握 CSS 基本语法
- 学会三种引入方式

## 什么是 CSS

CSS（Cascading Style Sheets）层叠样式表，用于控制网页的外观。

```css
h1 {
    color: blue;
    font-size: 24px;
}
```

## CSS 引入方式

### 1. 内联样式

```html
<h1 style="color: red;">红色标题</h1>
```

### 2. 内部样式

```html
<style>
    h1 { color: blue; }
</style>
```

### 3. 外部样式（推荐）

```html
<link rel="stylesheet" href="style.css">
```

## 常用属性

| 属性 | 作用 |
|------|------|
| color | 文字颜色 |
| font-size | 字体大小 |
| background | 背景 |
| margin | 外边距 |
| padding | 内边距 |
