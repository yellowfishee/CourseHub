欧几里得算法

又叫辗转相除法

一种更快速的求最大公因数的方法！

📖 欧几里得是谁？

👴 古希腊数学家

欧几里得（Euclid）生活在公元前300年左右

他是几何学之父，写了一本叫《几何原本》的书

🎯 他的贡献

在《几何原本》中记载了一种求最大公因数的方法

这个方法用了两千多年，到现在还在用！

所以叫欧几里得算法

💡 核心原理

辗转相除法

如果 a ÷ b = c ... r（余数是r）

GCD(a, b) = GCD(b, r)

也就是说：大数除以小数，余数再和除数求最大公因数

🌰 举个例子

12 ÷ 8 = 1 ... 4

那么 GCD(12, 8) = GCD(8, 4)

而 GCD(8, 4) = 4（4能整除8）

✋ 手动模拟一遍

求 GCD(12, 8)

1

12 ÷ 8 = 1 ... 4，余数是4

2

现在算 GCD(8, 4)

3

8 ÷ 4 = 2 ... 0，余数是0！

✅

余数为0时，除数就是答案！GCD(12, 8) = 4

✋ 再来一遍

求 GCD(100, 24)

1

100 ÷ 24 = 4 ... 4，余数是4

2

现在算 GCD(24, 4)

3

24 ÷ 4 = 6 ... 0，余数是0！

✅

GCD(100, 24) = 4

⚖️ 两种方法对比

上次课的方法（暴力枚举）欧几里得算法（辗转相除）
从最小的数开始往下找用大数除以小数，取余数
需要减很多次用除法，很快！
慢快！

🌰 对比：求 GCD(1000000, 1)

暴力枚举：从1找到... 要减1000000次！

欧几里得：1000000 ÷ 1 = 1000000 ... 0，一步搞定！

💻 代码实现

纯循环，不用函数！

a = int(input())  # 大数
b = int(input())  # 小数

while b != 0:     # 余数不为0就继续
    r = a % b    # 求余数
    a = b       # b变成被除数
    b = r       # r变成除数

print(a)        # a就是最大公因数

🔍 代码逐行解释

1

r = a % b — 用大数除以小数，得到余数

2

a = b — 小数变成新的"大数"

3

b = r — 余数变成新的"小数"

4

重复，直到 b == 0

5

当 b == 0 时，a 就是答案！

🔍 追踪变量

求 GCD(12, 8)

a = 12, b = 8
┌─────────────────────────────┐
│ r = 12 % 8 = 4   │ a = 8, b = 4 │
│ r = 8 % 4 = 0    │ a = 4, b = 0 │
└─────────────────────────────┘
b == 0，输出 a = 4

📋 完整代码

a = int(input()) b = int(input()) while b != 0: r = a % b a = b b = r print(a) # 测试： # 输入：12 8 → 输出 4 # 输入：100 24 → 输出 4 # 输入：18 12 → 输出 6 # 输入：7 9 → 输出 1

📌 今天学了什么

  • 欧几里得算法：又叫辗转相除法
  • 核心公式：GCD(a, b) = GCD(b, a % b)
  • 终止条件：余数为0时，除数就是答案
  • 优点：比暴力枚举快很多！

while b != 0:

    r = a % b

    a, b = b, r
