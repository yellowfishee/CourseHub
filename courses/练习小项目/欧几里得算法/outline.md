## 学习目标

- 理解欧几里得算法（辗转相除法）的原理
- 使用 while 循环实现辗转相除法
- 掌握用除法求余数的方式找最大公因数
- 不使用函数，纯循环实现

## 核心概念

- **辗转相除法**：用大数除以小数，取余数，再用除数除以余数，重复直到余数为0
- **原理**：a ÷ b = c ... r，则 GCD(a, b) = GCD(b, r)

## 算法

```python
a = int(input())
b = int(input())

while b != 0:
    r = a % b
    a = b
    b = r

print(a)  # a就是最大公因数
```
