# 欧几里得算法 - 教案

## 基本信息

- 课时：欧几里得算法（辗转相除法）
- 目标学生：小学生/初中生（学过 while 循环）
- 预计时长：45分钟

## 学习目标

1. 理解欧几里得算法的原理
2. 掌握使用 while 循环实现辗转相除法
3. 理解为什么比暴力枚举更快

## 教学过程

### 1. 导入（5分钟）

**提问**：上节课我们用的是什么方法求最大公因数？

**引入**：暴力枚举法虽好，但数字大了就很慢。有没有更快的方法？

### 2. 故事讲解（5分钟）

- 欧几里得：古希腊数学家，几何学之父
- 《几何原本》两千多年前就记载了这个算法
- 至今还在用，说明非常有效！

### 3. 原理讲解（10分钟）

**核心公式**：
```
a ÷ b = c ... r
则 GCD(a, b) = GCD(b, r)
```

**生活类比**：就像传接力棒，大数把"棒"交给小数，余数继续传

### 4. 手动模拟（10分钟）

用 12 和 8 演示一遍：
1. 12 ÷ 8 = 1 ... 4，余数4
2. GCD(12, 8) = GCD(8, 4)
3. 8 ÷ 4 = 2 ... 0，余数0
4. b == 0，答案是4

### 5. 代码实现（10分钟）

```python
a = int(input())
b = int(input())

while b != 0:
    r = a % b
    a = b
    b = r

print(a)
```

### 6. 总结（5分钟）

- 核心：GCD(a, b) = GCD(b, a % b)
- 终止条件：b == 0
- 优点：快！

## 常见错误

1. **条件写错**：要用 `while b != 0`，不是 `while a != 0`
2. **变量赋值顺序**：先保存余数，再交换

2. 比较暴力枚举法和欧几里得算法的效率
