# Python 入门课程

这是一门面向零基础学员的Python入门课程。通过本课程，你将掌握Python的基本语法和编程思想，最终能够独立完成洛谷 NOI 入门级别的题目。

## 课程大纲

本课程共分为 7 个章节，每章 4 课时 + 4 道洛谷题目，共 28 课时。

---

**学习内容：** 编程是什么、计算机三大步、变量、输入输出、算法三步曲

| 课时 | 主题 | 核心概念 |
|------|------|---------|
| 01-00 | Python导学 | 编程思维、计算机三大步、print() |
| 01-01 | 变量 | 变量名=变量值、便利贴类比 |
| 01-02 | 输入输出 | input()提问、print()回答、int()转换 |
| 01-03 | 算法三步曲 | 输入→处理→输出 |
