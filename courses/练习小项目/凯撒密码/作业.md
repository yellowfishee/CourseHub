# 凯撒密码作业

## 作业要求

请根据课堂所学知识，完成以下凯撒密码加密程序。

### 基础练习

**题目：数字密码加密**

编写一个程序，实现以下功能：
1. 输入一个多位数字密码（如：123456）
2. 将每一位数字往后挪 3 位
3. 使用取余运算确保结果仍是 0-9 的数字
4. 输出加密后的密码

**示例：**
```
输入：123
输出：456

输入：789
输出：012
```

**提示：**
- 使用 `input()` 获取输入
- 使用 `for` 循环遍历每一位数字
- 使用 `% 10` 取余

---

### 进阶练习

**题目：字母密码加密**

编写一个程序，实现以下功能：
1. 输入一个字母字符串（如："abc"）
2. 将每个字母往后挪 3 位（a→d, b→e, c→f...）
3. 输出加密后的字符串

**示例：**
```
输入：abc
输出：def

输入：xyz
输出：abc
```

**提示：**
- 使用 `ord()` 获取字母的 ASCII 编号
- 编号往后挪 3 位
- 使用 `chr()` 将编号转回字母
- 注意：x, y, z 往后挪要循环回 a, b, c

---

### 挑战练习（选做）

**题目：数字解密程序**

编写一个解密程序，将加密后的数字密码还原成原来的密码。

**示例：**
```
输入：456
输出：123
```
