取模运算 %

让它绕回来！

(新位置) % (总数) = 实际位置
26个字母，位置 0-25，超过25就从头开始
pos = (25 + 3) % 26 # Z的位置 + 3
pos = 28 % 26
pos = 2 # 回到位置2，就是 C

Z → C（绕回开头）

04
动手实现凯撒密码
算法步骤
  • 遍历字符串的每个字符
  • 判断是否为字母
  • 用 ord() 获取数字编码
  • 位移 + 取模处理环绕
  • 用 chr() 转回字符
  • 拼接成新的字符串
代码实现（加密）
def caesar_encrypt(text, key):
result = ""
for char in text:
if char.isupper():
# 大写字母: A=65, 相对位置0-25
pos = (ord(char) - 65 + key) % 26
result += chr(pos + 65)
elif char.islower():
# 小写字母: a=97
pos = (ord(char) - 97 + key) % 26
result += chr(pos + 97)
else:
# 非字母保持不变
result += char
return result
关键代码解析
pos = (ord(char) - 65 + key) % 26
  • ord(char) - 65 → 把字母变成 0-25
  • + key → 向后移动 key 位
  • % 26 → 超过25就绕回开头
  • chr(pos + 65) → 变回字母
运行示例
message = "HELLO WORLD"
key = 3
encrypted = caesar_encrypt(message, key)
print(encrypted)
# 输出: KHOOR ZRUOG

解密怎么做？

解密 = 反向移动
