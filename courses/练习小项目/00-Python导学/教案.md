## 学习目标

- 理解编程是"**告诉计算机做什么**"
- 理解计算机的**工作流程**（接收 → 处理 → 展示）
- 建立**分解问题**的思维方式
- 了解Python语言为什么适合初学者

---

## 内容

### 一、引入：从一个问题开始

**提问**：同学们每天都在用微信、抖音、王者荣耀——有没有想过，这些 App 是怎么做出来的？

**追问**：如果我们想让计算机帮我们做一件事，应该怎么做？

---

### 二、编程思维训练：如何让计算机做事？

#### 问题分解的例子

**问题**：让计算机显示 "Hello"

**分解过程**：

```
大问题：让计算机显示 "Hello"
  ↓ 分解
小问题1：谁来显示？→ 屏幕
小问题2：怎么让屏幕显示？→ 用一个"命令"告诉它
小问题3：什么命令？→ print() —— 就是"打印/显示"的意思

→ print("Hello")
```

#### 编程小公式

```
遇到问题 → 分解成小问题 → 每个小问题都能用代码解决
```

---

### 三、计算机是怎么工作的？

#### 计算机的三件"事"

| 阶段 | 做什么 | 例子 |
|------|--------|------|
| **接收** | 接收你给的信息 | 键盘打字、鼠标点击 |
| **处理** | 按照你的命令计算 | CPU 开始运算 |
| **展示** | 把结果显示给你看 | 屏幕显示、播放声音 |

#### 生活类比

```
计算机 = 像一个餐厅服务员
接收 = 记录你点的菜
处理 = 厨房做菜
展示 = 把菜端到你面前
```

---

### 四、print() 到底是什么？

#### print() 的三句话解释

| 问题 | 答案 |
|------|------|
| print() **是什么**？ | 一个"命令"，告诉计算机"显示出来" |
| print() **实际上是做什么**？ | 控制屏幕把文字显示出来 |
| **类比** | 就像打印机打印东西，print 就是"打印到屏幕" |

#### print() 的工作流程

```
你在代码里写：print("Hello")
         ↓
计算机收到这个命令
         ↓
屏幕显示：Hello
```

### 五、编程能做什么

#### 用"接收→处理→展示"来分析

| 应用 | 接收 | 处理 | 展示 |
|------|------|------|------|
| 微信 | 收到消息 | 保存到服务器 | 显示消息给你看 |
| 抖音 | 录制视频 | 推荐算法 | 推送你喜欢的内容 |
| 美团 | 选餐下单 | 计算配送路线 | 显示骑手位置 |

### 七、我们的学习路径

```
阶段一：Python基础
    ↓ 学习什么
    先学会"输出"（print）→ 再学"输入"（input）→ 然后学"计算"
阶段二：数据结构
    ↓ 学习什么
    学会存储数据（列表、字典）
阶段三：Web开发
    ↓ 学习什么
    学会做网站
阶段四：数据分析/AI
    ↓ 学习什么
    学会处理数据、训练AI
```
