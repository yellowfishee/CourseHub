最大公因数与最小公倍数

今天我们学两个重要的数学概念

还有它们的 Python 实现！

🎮 生活中的公因数和公倍数

🌰 举例：12和8

公因数：1, 2, 4（都能被它们整除）

最大公因数：4

公倍数：24, 48, 72...（同时是它们的倍数）

最小公倍数：24

🏫 类比：排队

有12个男生和8个女生分组，要求每组人数相同且尽量大

最多能分成 4 组，每组3男2女

❓

核心问题

怎么用 while 循环 和 break 来找最大公因数？

💡 解题思路

辗转相除法（减法版）

📝 规则：

1. 比较两个数，谁大就减掉小的那个

2. 重复这个过程，直到两个数相等

3. 相等的那个数，就是最大公因数！

12 和 8：
12 > 8，12 - 8 = 4（现在 4 和 8）
8 > 4，8 - 4 = 4（现在 4 和 4）
4 == 4，结束！最大公因数是 4

💻 代码实现

求最大公因数 GCD

def gcd(a, b):
    while True:            # 无限循环
        if a > b:            # a 太大
            a = a - b      # a 减小
        else:               # b 太大
            b = b - a      # b 减小
        if a == b:        # 相等了
            break          # 跳出循环！
    return a            # 返回最大公因数

🎯 break 的作用

找到答案就停下！

🚗 类比：开车

while True 就像一直往前开

break 就像看到目的地就停车

不然会一直开下去（死循环）！

# 没有 break = 死循环（永远停不下来） while True: print("停不下来！") # 有 break = 找到答案就停 while True: if 找到了答案: break   # 停车！

📝 最小公倍数 LCM

其实很简单！

📐 公式：

LCM(a, b) = a × b ÷ GCD(a, b)

def lcm(a, b):
    g = gcd(a, b)    # 先求最大公因数
    return a * b // g  # 公式

为什么这样？记住就行啦！

📋 完整代码

def gcd(a, b):
    while True:
        if a > b:
            a = a - b
        else:
            b = b - a
        if a == b:
            break
    return a

def lcm(a, b):
    return a * b // gcd(a, b)

# 测试
print(gcd(12, 8))  # 输出 4
print(lcm(12, 8))  # 输出 24

✏️ 课堂练习

用 while + break 求 18 和 12 的最大公因数

📝 步骤：

1. 18 > 12，18 - 12 = 6（现在 6 和 12）

2. 12 > 6，12 - 6 = 6（现在 6 和 6）

3. 6 == 6，结束！

答案：6

它们的 最小公倍数 是多少？

📌 今天学了什么

  • 最大公因数 GCD：两个数的最大公共除数
  • 最小公倍数 LCM：两个数的最小公共倍数
  • while True + break：循环直到找到答案
  • 辗转相除法：大数减小数，直到相等
