最大公因数与最小公倍数

今天我们学两个重要的数学概念

还有它们的 Python 实现！

📖 什么是公因数

用举例来理解

🌰 例子：12 和 8

12 的因数：1, 2, 3, 4, 6, 12

8 的因数：1, 2, 4, 8

公因数：1, 2, 4（同时出现在两个列表里的数）

🔑 最大公因数

12 和 8 的最大公因数是 4

就是所有公因数里最大的那个！

📖 什么是公倍数

继续用 12 和 8 举例

🌰 例子：12 和 8

12 的倍数：12, 24, 36, 48...

8 的倍数：8, 16, 24, 36, 48...

公倍数：24, 36, 48...（同时是两个数的倍数）

🔑 最小公倍数

12 和 8 的最小公倍数是 24

就是所有公倍数里最小的那个！

❓

怎么用代码找最大公因数？

思路：从最小的数开始找

谁小从谁开始，找到就停（break）！

💡 解题思路

暴力枚举法

1

先找出两个数中最小的那个

2

从这个最小的数开始，逐个往下检查

3

检查这个数能不能同时整除 a 和 b

4

如果可以 → 找到啦！打印出来，然后 break

5

如果不可以 → 继续往下找（minn = minn - 1）

✋ 手动模拟一遍

求 12 和 8 的最大公因数

1

minn = 8（8比较小）

2

8 能整除 12 吗？不能 → 继续

3

minn = 7，7 能整除 12 和 8 吗？不能 → 继续

4

minn = 6，6 能整除 12 和 8 吗？不能 → 继续

5

minn = 5，5 能整除 12 和 8 吗？不能 → 继续

6

minn = 4，4 能整除 12 吗？能！能整除 8 吗？能！

✅

找到啦！答案是 4！

💻 代码实现

求最大公因数

a = int(input())
b = int(input())

# 1. 找出最小的数
minn = 0
if a < b:
    minn = a
else:
    minn = b

# 2. 从小到大逐个检查，找到就停
while 1:        # while True 也可以
    if b % minn == 0 and a % minn == 0:
        print(minn)
        break     # 找到啦！停！
    minn = minn - 1  # 找不到就继续往下找

🎯 break 的作用

找到答案就停下！

🚗 类比：开车

while 1 就像一直往前开

break 就像看到目的地就停车

不然会一直开下去（死循环）！

# 没有 break = 死循环（永远停不下来） while 1: print("停不下来！") # 有 break = 找到答案就停 while 1: if 找到了答案: break   # 停车！

📝 最小公倍数

记住这个公式就行！

LCM(a, b) = a × b ÷ GCD(a, b)

🌰 验证：12 和 8

GCD(12, 8) = 4

LCM(12, 8) = 12 × 8 ÷ 4 = 96 ÷ 4 = 24

之前算的最小公倍数是24，对上了！

📋 完整代码

# 最大公因数 a = int(input()) b = int(input())
minn = 0
if a < b:
    minn = a
else:
    minn = b

while 1:
    if b % minn == 0 and a % minn == 0:
        print(minn)
        break
    minn = minn - 1

# 最小公倍数 = a * b // 最大公因数

📌 今天学了什么

  • 最大公因数 GCD：两个数的最大公共除数
  • 最小公倍数 LCM：两个数的最小公共倍数
  • while 1 + break：循环直到找到答案
  • 暴力枚举法：从最小的数开始逐个往下找

📚 公式要记住：

LCM(a, b) = a × b ÷ GCD(a, b)
