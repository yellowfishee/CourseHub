# 最大公因数与最小公倍数 - 教案

## 基本信息

- 课时：最大公因数与最小公倍数
- 目标学生：小学生/初中生（学过 while 循环）
- 预计时长：45分钟

## 学习目标

1. 理解最大公因数（GCD）和最小公倍数（LCM）的概念
2. 掌握使用 while + break 找最大公因数
3. 学会最小公倍数的公式

## 教学过程

### 1. 导入（5分钟）

**提问**：12和8同时能被哪些数整除？

**引入**：生活中经常遇到"分组"问题，需要找最大公因数

### 2. 概念讲解（10分钟）

- **最大公因数**：两个数的最大公共除数（12和8的GCD是4）
- **最小公倍数**：两个数的最小公共倍数（12和8的LCM是24）

### 3. 算法原理（10分钟）

**辗转相除法（减法版）**：
1. 大的数减小数
2. 重复直到相等
3. 相等的数就是答案

**生活类比**：就像两个人互相给对方钱，直到钱一样多

### 4. 代码实现（15分钟）

```python
def gcd(a, b):
    while True:
        if a > b:
            a = a - b
        else:
            b = b - a
        if a == b:
            break
    return a

def lcm(a, b):
    return a * b // gcd(a, b)
```

### 5. 课堂练习（10分钟）

用辗转相除法求 18 和 12 的最大公因数

答案：6

### 6. 总结（5分钟）

- while True + break：找到答案就停
- LCM(a, b) = a × b ÷ GCD(a, b)

## 常见错误

1. **忘记 break**：死循环
2. **return 位置错误**：要在循环外面 return
