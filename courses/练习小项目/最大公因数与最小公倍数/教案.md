# 最大公因数与最小公倍数 - 教案

## 基本信息

- 课时：最大公因数与最小公倍数
- 目标学生：小学生/初中生（学过 while 循环）
- 预计时长：45分钟

## 学习目标

1. 理解最大公因数（GCD）和最小公倍数（LCM）的概念
2. 掌握使用 while + break 从小到大找最大公因数的方法
3. 记住最小公倍数的公式

## 教学过程

### 1. 导入（5分钟）

**提问**：12和8同时能被哪些数整除？

**引入**：生活中经常遇到"分组"问题，需要找最大公因数

### 2. 概念讲解（10分钟）

- **因数**：能整除这个数的数
- **最大公因数**：两个数的最大公共除数
  - 12的因数：1,2,3,4,6,12
  - 8的因数：1,2,4,8
  - 公因数：1,2,4
  - 最大公因数：4
- **最小公倍数**：两个数的最小公共倍数
  - 12的倍数：12,24,36...
  - 8的倍数：8,16,24...
  - 最小公倍数：24

### 3. 算法原理（10分钟）

**暴力枚举法**：
1. 先找出两个数中较小的那个
2. 从这个最小的数开始，逐个往下检查
3. 检查这个数能不能同时整除 a 和 b
4. 如果可以 → 找到啦！break

**手动模拟**：用 12 和 8 演示一遍过程

### 4. 代码实现（15分钟）

```python
a = int(input())
b = int(input())

minn = 0
if a < b:
    minn = a
else:
    minn = b

while 1:
    if b % minn == 0 and a % minn == 0:
        print(minn)
        break
    minn = minn - 1
```

### 5. 最小公倍数公式（5分钟）

**记住就行：**

```
最小公倍数 = a × b ÷ 最大公因数
LCM(a, b) = a * b // GCD(a, b)
```

验证：12 × 8 ÷ 4 = 24 ✓

### 6. 总结（5分钟）

- while 1 + break：找到答案就停
- 从最小开始，逐个往下找
- LCM = a × b ÷ GCD

## 常见错误

1. **忘记 break**：死循环
2. **判断条件写反**：要用 `a % minn == 0`（能整除）
3. **minn 忘记减1**：找不到要继续往下
