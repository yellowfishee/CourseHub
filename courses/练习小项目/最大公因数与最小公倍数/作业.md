### 研究题参考答案

**1. 欧几里得算法是什么？**

欧几里得算法（也叫辗转相除法）是一种更高效的求最大公因数的方法。它的原理是：用较大的数除以较小的数，得到余数；然后用较小的数除以余数，重复这个过程，直到余数为0，最后的除数就是最大公因数。

**2. 两种方法的区别：**

| 今天的方法 | 欧几里得算法 |
|-----------|-------------|
| 从最小的数开始往下找（逐个减1） | 用除法求余数 |
| 慢（最坏要减很多次） | 快（除法次数少） |

**3. 欧几里得算法例子：**

求 12 和 8 的最大公因数：

```
12 ÷ 8 = 1 ... 4  （余数是4）
8 ÷ 4 = 2 ... 0   （余数是0，所以答案是4）
```

用代码实现欧几里得算法：
```python
def gcd(a, b):
    while b != 0:
        a, b = b, a % b
    return a
```
