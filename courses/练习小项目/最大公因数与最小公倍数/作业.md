## 参考答案

**选择题：** 1.B, 2.B, 3.B, 4.A

**填空题：** 5. 5, 10, 5　　6. GCD(a, b)

**编程题参考答案：**

```python
# 题目1：gcd 函数
def gcd(a, b):
    while True:
        if a > b:
            a = a - b
        else:
            b = b - a
        if a == b:
            break
    return a

# 题目2：lcm 函数
def lcm(a, b):
    return a * b // gcd(a, b)

# 题目3：三个数的 GCD 和 LCM
def gcd3(a, b, c):
    return gcd(gcd(a, b), c)

def lcm3(a, b, c):
    return lcm(lcm(a, b), c)

a, b, c = map(int, input().split())
print(f"GCD: {gcd3(a, b, c)}")
print(f"LCM: {lcm3(a, b, c)}")
```

---

**研究题答案：**

1. 欧几里得算法由古希腊数学家**欧几里得**发明
2. 欧几里得算法就是**辗转相除法**（两种叫法）
3. 12345 和 54321 的最大公因数是 **3**

（可以用 `gcd(12345, 54321)` 代码验证）
