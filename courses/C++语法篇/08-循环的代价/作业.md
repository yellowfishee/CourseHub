## 编程题

### 简单题

**题目1：累加求和**

**描述：** 输入一个整数 n，计算 1+2+3+...+n 的和。

```
示例输入：
100

示例输出：
5050
```

---

**题目2：求阶乘**

**描述：** 输入一个整数 n（0 ≤ n ≤ 20），输出 n! 的值。

```
示例输入：
5

示例输出：
120
```

**提示：** 用 long long 存放结果

---

**题目3：阶乘累加**

**描述：** 输入一个整数 n（1 ≤ n ≤ 10），计算 S = 1! + 2! + 3! + ... + n!。

```
示例输入：
4

示例输出：
33
```

解释：1! + 2! + 3! + 4! = 1 + 2 + 6 + 24 = 33

---

### 中等题

**题目4：阶乘之和（优化版）**

**题目来源：** 洛谷 P1000（风格题）

**描述：** 输入一个整数 n（1 ≤ n ≤ 20），计算 S = 1! + 2! + 3! + ... + n! 的末6位。由于结果可能较大，请输出对 1000000 取模后的值。

```
示例输入：
10

示例输出：
37913
```

**提示：** 每步取模可以避免溢出；如果 n > 25，答案不变（因为25!末尾有6个0）

## 参考答案

### 选择题答案
1-B, 2-C, 3-B, 4-B, 5-C

### 编程题参考答案

```cpp
// 题目1：累加求和
#include <stdio.h>
int main() {
    int n;
    long long sum = 0;
    scanf("%d", &n);
    for (int i = 1; i <= n; i++) {
        sum += i;
    }
    printf("%lld\n", sum);
    return 0;
}
```

```cpp
// 题目2：求阶乘
#include <stdio.h>
int main() {
    int n;
    long long fact = 1;
    scanf("%d", &n);
    for (int i = 1; i <= n; i++) {
        fact *= i;
    }
    printf("%lld\n", fact);
    return 0;
}
```

```cpp
// 题目3：阶乘累加
#include <stdio.h>
int main() {
    int n;
    long long sum = 0, fact = 1;
    scanf("%d", &n);
    for (int i = 1; i <= n; i++) {
        fact *= i;
        sum += fact;
    }
    printf("%lld\n", sum);
    return 0;
}
```

```cpp
// 题目4：阶乘之和（优化版）
#include <stdio.h>
int main() {
    const int MOD = 1000000;
    int n;
    long long sum = 0, fact = 1;
    scanf("%d", &n);
    if (n > 25) n = 25;
    for (int i = 1; i <= n; i++) {
        fact = (fact * i) % MOD;
        sum = (sum + fact) % MOD;
    }
    printf("%lld\n", sum);
    return 0;
}
```
