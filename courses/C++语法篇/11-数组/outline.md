## 课程概述

本课介绍数组的基本概念和使用方法。数组是存储大量同类数据的数据结构，是算法的基础。

## 学习目标

- 掌握一维数组的声明和使用
- 掌握二维数组的声明和使用
- 学会用数组解决实际问题
- 理解数组下标的范围

## 核心概念

1. **数组声明**: int a[100]
2. **数组下标**: a[0], a[1], ..., a[99]
3. **二维数组**: int a[10][10]
4. **数组初始化**: memset清零
5. **memcpy复制**: 数组整体复制
