# 教案：11-数组

## 教学目标

1. **理解数组概念**: 数组是存储大量同类数据的数据结构
2. **掌握数组声明**: 一维数组和二维数组
3. **学会数组操作**: 初始化、访问、复制
4. **解决实际问题**: 用数组解决逆序输出、开灯问题等

## 引入阶段 (5分钟)

### 生活类比：储物柜

想象一个储物柜：
- 每个柜子有编号（0, 1, 2, ...）
- 每个柜子可以放一个东西
- 数组就像一排编好号的储物柜

## 讲解阶段 (15分钟)

### 核心概念

```c
int a[100];  // 声明100个整数的数组
a[0] = 10;   // 给第一个格子赋值
a[1] = 20;   // 给第二个格子赋值
```

### 提示提取

- **提示3-1**: 数组下标从0开始，最大是maxn-1
- **提示3-2**: 数组大小要稍大一些
- **提示3-3**: n++和++n的区别
- **提示3-4**: 大数组声明在main函数外
- **提示3-5**: 二维数组的使用
- **提示3-6**: C语言简洁语法的使用

### 重要函数

1. **memset**: 数组清零
2. **memcpy**: 数组复制

## 交互练习 (15分钟)

### 练习1: 逆序输出
把数组中的数逆序输出

### 练习2: 找最大值
在数组中找最大元素

### 练习3: 开灯问题
模拟开灯关灯的过程

## 总结 (5分钟)

### 本课要点

1. 数组下标从0开始
2. 大数组要声明在main函数外
3. 使用memset初始化数组
4. 数组是算法的基础数据结构
