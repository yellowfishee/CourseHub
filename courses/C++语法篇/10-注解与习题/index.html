🔄 三种循环对比

for循环

for(初始化;条件;调整)
  循环体;

适合：已知循环次数

while循环

while(条件)
  循环体;

适合：条件驱动

do-while

do {
  循环体;
} while(条件);

特点：至少执行一次

💡 提示2-8 + 提示2-14

🔁 嵌套循环

for(int i = 1; i <= n; i++)  // 外层循环n次
  for(int j = 1; j <= n; j++)  // 内层循环n次
    // 循环体执行 n×n 次

n = 10

执行 100 次

n = 100

执行 10,000 次

🐛 调试技巧

当程序结果不正确时，可以用输出中间结果的方法来查找问题。

int n, count = 0;
scanf("%d", &n);
while(n > 1) {
  if(n % 2 == 1) n = n * 3 + 1;
  else n /= 2;
  count++;
  printf("n = %lld, count = %d\n", n, count);  // 输出中间结果
}

💡 提示2-11

在观察无法找出错误时，可以用"输出中间结果"的方法查错。

🔧 取模防溢出

当计算结果可能超过int范围时，可以在每步取模：

// 错误写法：可能溢出
factorial = factorial * j;

// 正确写法：每步取模
factorial = (factorial * j) % MOD;

公式

(a × b) % mod = ((a % mod) × (b % mod)) % mod

💡 提示2-16

在每步计算之后对n取余，结果不变。

⏱️ 计时函数

#include <time.h>

int main() {
  // ... 程序代码 ...

  printf("Time: %.2f\n",
    (double)clock() / CLOCKS_PER_SEC);
  return 0;
}

💡 提示2-17

clock()返回程序运行时间（处理器时间），要除以CLOCKS_PER_SEC转换为秒。
