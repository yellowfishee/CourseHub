💡 关键

循环就是"重复做一件事"，但要记得给个停止条件！

📝 三种循环对比

for循环：知道要循环多少次

for(int i = 0; i < 10; i++) {
  printf("%d ", i);
}

while循环：先判断后执行

int i = 0;
while(i < 10) {
  printf("%d ", i);
  i++;
}

do-while循环：先执行后判断

int i = 0;
do {
  printf("%d ", i);
  i++;
} while(i < 10);

🐛 常见Bug：死循环

忘记让循环变量变化，会导致程序卡死！

❌ 错误代码

int i = 0;
while(i < 10) {
  printf("%d", i);
  // i++ 忘了！
}

✓ 正确代码

int i = 0;
while(i < 10) {
  printf("%d", i);
  i++;
}

💡 提示

检查循环时，先问自己：循环变量改变了吗？条件会变成假吗？

📬 生活类比：邮递员

📥

输入 (scanf)

就像邮递员收信：

  • • 打开信箱 (scanf)
  • • 取出信件 (读取数据)
  • • 确认收到 (返回值)
📤

输出 (printf)

就像寄信：

  • • 写好内容 (要输出的数据)
  • • 放进信封 (格式化)
  • • 寄出去 (显示)

💡 关键

scanf和printf是程序与外界交流的窗口！

🔍 scanf的返回值

int result = scanf("%d %d", &a, &b);

result = 2  // 成功读取了2个变量
result = 1  // 只成功读取了1个
result = 0  // 第一个就没读成功
result = EOF // 文件结束了 (-1)

返回2

a和b都成功

返回1

只有a成功

返回EOF

输入结束

✅ 循环读入模式

读取多组数据，直到输入结束：

int x;
while(scanf("%d", &x) == 1) {
  // 处理 x
  printf("%d ", x * 2);
}

💡 解读

while(scanf == 1) 的意思是："只要成功读取了一个数，就继续循环"

📁 文件读写复习

方法1：freopen 重定向

freopen("data.in", "r", stdin);
freopen("data.out", "w", stdout);

✓ 简单，代码不用改

方法2：fopen + fprintf/fscanf

FILE *f = fopen("data.in", "r");
fscanf(f, "%d", &x);

✓ 灵活，明确指定文件

⚠️ 变量初始化

未初始化的变量就像盲盒，里面是什么不知道！

❌ 危险代码

int min, max;

📋 多组数据处理模式

格式：n组数据，n=0结束，输出Case编号

int n, x, kase = 0;
while(scanf("%d", &n) == 1 && n != 0) {
  int sum = 0;
  for(int i = 0; i < n; i++) {
    scanf("%d", &x);
    sum += x;
  }
  if(kase) printf("\n");
  printf("Case %d: %d\n", ++kase, sum);
}
