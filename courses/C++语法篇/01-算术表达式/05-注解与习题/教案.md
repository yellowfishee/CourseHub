### 常见错误分类

#### 1. 语法错误（Syntax Error）

**错误原因：** 代码写错了格式

```c
printf("%d\n", 1 + 2);    // 正确
printf("%d\n", 1 + 2)；   // 错误：分号是中文分号
```

#### 2. 头文件错误

**错误原因：** 用了没包含的头文件

```c
#include <stdio.h>    // 正确：包含标准输入输出
// 如果用 printf 但没包含头文件，会报错
```

#### 3. 格式字符串错误

```c
int a = 5;
printf("%d\n", a);    // 正确：输出整数用 %d
printf("%f\n", a);    // 错误：int 应该用 %d，不是 %f
double b = 3.14;
printf("%lf\n", b);   // 正确：double 用 %lf
```

#### 4. 变量类型错误

```c
// 整数除法和浮点数除法
printf("%d\n", 8 / 5);      // 错误：结果是1，不是1.6
printf("%.1f\n", 8.0 / 5.0); // 正确：结果是1.6
```

### 调试技巧

#### 看报错信息的三个步骤

1. **找到错误在哪一行** - 报错会告诉你是第几行
2. **看清是什么类型的错误** - Syntax Error? Warning?
3. **检查那一行的代码** - 分号、引号、格式符是否正确

```c
// 示例报错
error: expected ';' after 'return' statement
    return 0
            ^
            ;
```

这个报错告诉我们：
- 第5行有错误
- 错误类型：缺少分号
- 问题：return 0 后面没有分号

### 竞赛题目要点

1. **不要加提示语** - 直接 scanf()，不要 printf("请输入")
2. **输出格式要严格** - 题目说保留3位小数，就用 %.3f
3. **整数除法要小心** - 用 8.0/5.0 而不是 8/5

## 习题讲解

### 题目1：平均数（average）

**题目描述：** 输入3个整数，输出它们的平均值，保留3位小数。

**思路分析：**
1. 读取3个数字
2. 计算它们的和
3. 除以3得到平均值
4. 保留3位小数输出

**参考答案：**
```c
#include <stdio.h>
int main() {
    double a, b, c;
    scanf("%lf %lf %lf", &a, &b, &c);
    printf("%.3f\n", (a + b + c) / 3);
    return 0;
}
```

### 题目2：温度（temperature）

**题目描述：** 输入华氏温度 f，输出对应的摄氏温度 c，保留3位小数。
**公式：** c = 5(f-32)/9

**思路分析：**
1. 读取华氏温度
2. 用公式计算摄氏温度
3. 保留3位小数输出

**参考答案：**
```c
#include <stdio.h>
int main() {
    double f, c;
    scanf("%lf", &f);
    c = 5.0 * (f - 32.0) / 9.0;
    printf("%.3f\n", c);
    return 0;
}
```

### 题目3：连续和（sum）

**题目描述：** 输入正整数 n，输出 1+2+...+n 的值。

**思路分析：**
1. 读取 n
2. 使用循环累加，或者用求和公式 n*(n+1)/2

**参考答案：**
```c
#include <stdio.h>
int main() {
    int n;
    scanf("%d", &n);
    printf("%d\n", n * (n + 1) / 2);
    return 0;
}
```

### 题目4：打折（discount）

**题目描述：** 一件衣服95元，若消费满300元，可打八五折。输入购买衣服件数，输出需要支付的金额（单位：元），保留两位小数。

**思路分析：**
1. 计算总价 = 件数 × 95
2. 如果总价 >= 300，打八五折
3. 保留两位小数输出

**参考答案：**
```c
#include <stdio.h>
int main() {
    int n;
    double total;
    scanf("%d", &n);
    total = n * 95.0;
    if (total >= 300.0) {
        total *= 0.85;
    }
    printf("%.2f\n", total);
    return 0;
}
```
