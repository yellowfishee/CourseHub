## 内容

### 一、引入：计算机最擅长做计算

**提问**：计算机最擅长什么？

**回答**：计算！速度超级快。

**生活例子**：
```
你去餐厅吃饭：
服务员记下你的菜名（输入）→ 厨房做菜（计算）→ 服务员端菜给你（输出）
计算机也一样：
接收数据（输入）→ 计算处理 → 显示结果（输出）
```

---

### 二、第一个 C++ 命令：printf()

**问题**：怎么让计算机显示一个数字？

**C++ 代码**：

```cpp
#include <stdio.h>

int main() {
    printf("%d\n", 1 + 2);
    return 0;
}
```

**解释**：
- `#include <stdio.h>` = 引入"标准输入输出"工具
- `printf()` = 让计算机显示内容
- `%d` = 告诉计算机会显示一个整数
- `\n` = 换行

**类比**：
```
printf() = 屏幕上的"打印机"
把内容"打印"到屏幕上
```

---

### 三、加减乘除：四个基本运算

**在 C++ 中**：

| 数学符号 | C++ 写法 | 例子 | 结果 |
|---------|----------|------|------|
| + | + | 1 + 2 | 3 |
| - | - | 10 - 3 | 7 |
| × | * | 3 * 4 | 12 |
| ÷ | / | 15 / 3 | 5 |

**重点**：乘号是 `*`（星号），除号是 `/`（斜杠）！

**动手试试**：

```cpp
printf("%d\n", 1 + 2);      // 显示 3
printf("%d\n", 100 - 50);   // 显示 50
printf("%d\n", 3 * 4);      // 显示 12
printf("%d\n", 10 / 2);     // 显示 5
```

### 四、重要发现：整数除法和浮点数除法

**实验1**：运行下面代码

```cpp
printf("%d\n", 8 / 5);
```

**结果**：显示 `1`？不是 1.6！

**再实验**：

```cpp
printf("%.1f\n", 8.0 / 5.0);  // 结果是 1.6
printf("%d\n", 8 / 5);         // 结果是 1
```

**规则**：

| 运算 | C++ 写法 | 结果 | 类型 |
|------|----------|------|------|
| 浮点数除法 | 8.0 / 5.0 | 1.6 | 浮点数 |
| 整数除法 | 8 / 5 | 1 | 整数（取整） |
| 取余数（取余） | 8 % 5 | 3 | 整数 |

**提示**：`%d` 输出整数，`%f` 输出浮点数

### 五、数学函数：sqrt() 求平方根

**问题**：怎么计算 √3（3的平方根）？

**C++ 代码**：

```cpp
#include <stdio.h>
#include <math.h>

int main() {
    printf("%.8f\n", sqrt(3));
    return 0;
}
```

**解释**：
- `#include <math.h>` = 引入数学函数库
- `sqrt(x)` = 计算 x 的平方根
- `%.8f` = 显示8位小数的浮点数

**类比**：
```
sqrt() = 计算器上的 √ 按钮
sqrt(9) = 3，因为 3 × 3 = 9
```

### 六、复杂表达式：混合运算

**问题**：计算 `1 + 2 × √3 ÷ (5 - 0.1)`

**C++ 代码**：

```cpp
#include <stdio.h>
#include <math.h>

int main() {
    printf("%.8f\n", 1 + 2 * sqrt(3) / (5 - 0.1));
    return 0;
}
```

**运算符优先级**（先算什么）：
```
1. 括号 ()
2. 乘除 * / %
3. 加减 + -
4. 从左到右
```

### 七、常见错误和调试

**错误1**：分号丢了

```cpp
printf("%d", 1 + 2)   // 错误！丢了分号
printf("%d", 1 + 2);  // 正确！
```

**错误2**：中文括号

```cpp
printf("%d", (1 + 2)）；  // 错误！用了中文括号
printf("%d", (1 + 2));    // 正确！
```

**错误3**：除号用错了

```cpp
printf("%d", 8 \ 5);   // 错误！反斜杠不是除号
printf("%d", 8 / 5);   // 正确！用斜杠
```
