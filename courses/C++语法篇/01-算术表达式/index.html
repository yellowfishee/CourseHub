程序1-2　计算并输出8/5的值，保留小数点后1位

#include <stdio.h>
int main() {
    printf("%.1f\n", 8.0 / 5.0);
    return 0;
}

输出：1.6

重要发现：除法的两种结果

✓ 浮点数除法
printf("%.1f", 8.0 / 5.0);

1.6

用小数点表示浮点数

✓ 整数除法
printf("%d", 8 / 5);

1

取商的整数部分

取余数 %：8 % 5 = 3

提示1-1：规范

整数值用 %d 输出，实数用 %f 输出

"整数值"指的是1+2、8/5这样"整数之间的运算"

只要运算符两边都是整数，运算结果也是整数

printf("%d", 8 / 5); // 输出 1
printf("%.1f", 8.0 / 5.0); // 输出 1.6

程序1-3　复杂的表达式计算

#include <stdio.h>
#include <math.h>
int main() {
    printf("%.8f\n", 1 + 2 * sqrt(3) / (5 - 0.1));
    return 0;
}

计算：1 + 2×√3 ÷ (5-0.1)

数学函数：sqrt() 求平方根

#include <stdio.h>
#include <math.h>
int main() {
    printf("%.8f\n", sqrt(9.0));
    return 0;
}

输出：3.00000000

⚠️ 使用数学函数需要 #include <math.h>
