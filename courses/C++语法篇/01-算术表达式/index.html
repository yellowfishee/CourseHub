🏪 超市找零钱的启示

你去超市买了 7块钱 的东西，付了 10块钱。

收银员会怎么找零？

10 - 7 = 3 块！

💡 计算机做计算，就像收银员找零一样！

我们告诉它数字和运算，它就帮我们算出结果

第一个计算程序

让计算机计算 1 + 2 等于几

#include <stdio.h>
int main() {
    printf("%d", 1 + 2);
    return 0;
}

计算机显示：3

➕➖ 加法和减法

和小学数学一模一样！

数学C++例子结果

✖️ 乘法 - 注意符号变了！

📝 重要区别：

数学里写 3 × 5

C++里要写 3 * 5

乘号变成了一颗星 *

printf("%d", 3 * 5);  // 显示 15

➗ 除法 - 也有小陷阱！

📝 重要区别：

数学里写 15 ÷ 3

C++里要写 15 / 3

除号变成了斜杠 /

printf("%d", 15 / 3);  // 显示 5

⚠️ 整数除法的陷阱

整数除法（丢小数点）
8 / 5

显示：1

8除以5等于1余3，只显示整数部分！

小数除法（保留小数）
8.0 / 5.0

显示：1.6

带小数点的数相除，才有小数！

💡 记住：整数和整数除，结果还是整数！

🔢 新知识：取余数 %

🍔 想象你买鸡腿：

13块钱一个，你有 45块钱

能买 3 个，还剩 6 块

45 / 5 = 3 （能买几个）

45 % 5 = 0 （余数）

% 符号就是取余数！

🎮 小测验：17 % 5 = ?

17除以5等于几？余几？

3
2
7

提示：17 ÷ 5 = 3 余 2

所以 17 % 5 = 2 ✓

❌ 常见错误

✗ 错误写法

✓ 正确写法

printf("%d", 8 / 5);

用斜杠 /

✗ 错误写法

printf("%d", 8 × 5);

用了数学符号 ×

✓ 正确写法

printf("%d", 8 * 5);

用星号 *
