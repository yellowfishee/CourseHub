## 编程题

### 简单题

**题目1：** 计算两数之和

**描述：** 输入两个整数，输出它们的和。

```
示例输入：
3 5
示例输出：
8
```

---

**题目2：** 计算圆柱体积

**描述：** 输入圆柱的半径 r 和高 h，输出圆柱体积（保留1位小数）。
公式：体积 = π × r² × h，用 `acos(-1.0)` 表示 π。

```
示例输入：
2 5
示例输出：
62.8
```

**提示：** 先算 r 的平方，再乘以 π 和 h。

---

**题目3：** 计算余数

**描述：** 输入两个整数 a 和 b（a > b），输出 a 除以 b 的余数。

```
示例输入：
17 5
示例输出：
2
```

---

### 中等题

**题目4：** 计算Digitsum

**描述：** 输入一个三位整数，计算它的各位数字之和。
例如：123 → 1 + 2 + 3 = 6

```
示例输入：
456
示例输出：
15
```

**提示：** 百位 = n / 100，十位 = n / 10 % 10，个位 = n % 10

## 参考答案

### 选择题答案

1. B, 2. B, 3. B, 4. C, 5. B

### 编程题参考答案

```cpp
// 题目1：计算两数之和
#include <stdio.h>
int main() {
    int a, b;
    scanf("%d%d", &a, &b);
    printf("%d\n", a + b);
    return 0;
}

// 题目2：计算圆柱体积
#include <stdio.h>
int main() {
    double r, h;
    scanf("%lf%lf", &r, &h);
    double volume = acos(-1.0) * r * r * h;
    printf("%.1f\n", volume);
    return 0;
}

// 题目3：计算余数
#include <stdio.h>
int main() {
    int a, b;
    scanf("%d%d", &a, &b);
    printf("%d\n", a % b);
    return 0;
}

// 题目4：计算Digitsum
#include <stdio.h>
int main() {
    int n;
    scanf("%d", &n);
    int bai = n / 100;
    int shi = n / 10 % 10;
    int ge = n % 10;
    printf("%d\n", bai + shi + ge);
    return 0;
}
```
