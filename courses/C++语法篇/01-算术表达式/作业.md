## 参考答案

**选择题：** 1-B, 2-B, 3-B, 4-C, 5-B

**填空题：**
6. 整数、浮点数
7. /、/
8. 余数
9. 3.0
10. 分号 `;`

**编程题（参考答案）：**

```cpp
// 题目1
#include <stdio.h>
int main() {
    printf("%d\n", 1 + 2 + 3 + 4 + 5);
    return 0;
}

// 题目2
#include <stdio.h>
int main() {
    printf("%.1f\n", 10.0 / 4.0);
    return 0;
}

// 题目3
#include <stdio.h>
#include <math.h>
int main() {
    printf("%f\n", sqrt(16) + sqrt(9));
    return 0;
}

// 题目4
#include <stdio.h>
#include <math.h>
int main() {
    double r = 3.5, h = 9;
    double pi = acos(-1.0);
    double area = 2 * pi * r * r + 2 * pi * r * h;
    printf("%.3f\n", area);
    return 0;
}

// 题目5
#include <stdio.h>
int main() {
    printf("%d\n", 17 / 5);
    printf("%d\n", 17 % 5);
    return 0;
}
```
