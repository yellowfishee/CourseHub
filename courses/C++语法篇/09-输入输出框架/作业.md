## 编程题

### 简单题

**题目1：计算多组数据的和**

**描述：** 输入若干整数（每行一个，输入到文件末尾结束），输出它们的和。

```
示例输入：
1
2
3
4
5

示例输出：
15
```

**提示：** 使用 `while(scanf("%d", &x) == 1)` 读取所有数据

---

**题目2：找最大最小值**

**描述：** 输入若干整数（每行一个，输入到文件末尾结束），输出其中的最大值和最小值。

```
示例输入：
5
3
9
1
7

示例输出：
9 1
```

**提示：** 初始化最大值为很小的数，最小值为很大的数

---

**题目3：计算平均值**

**描述：** 输入若干整数（每行一个），计算它们的平均值（保留2位小数）。

```
示例输入：
10
20
30

示例输出：
20.00
```

---

### 中等题

**题目4：计算多组阶乘**

**描述：** 输入若干行，每行一个整数 n（0 ≤ n ≤ 20），对每行分别输出 n! 的值。每行输出一个结果，格式为 "Case X: Y"，其中 X 是从1开始的编号，Y 是计算结果。

```
示例输入：
5
3
0

示例输出：
Case 1: 120
Case 2: 6
Case 3: 1
```

**提示：** 用计数器给输出编号

## 参考答案

### 选择题答案
1-B, 2-C, 3-B, 4-B, 5-C

### 编程题参考答案

```cpp
// 题目1：计算多组数据的和
#include <stdio.h>
int main() {
    int x;
    long long sum = 0;
    while (scanf("%d", &x) == 1) {
        sum += x;
    }
    printf("%lld\n", sum);
    return 0;
}
```

```cpp
// 题目2：找最大最小值
#include <stdio.h>
#include <limits.h>
int main() {
    int x, max = INT_MIN, min = INT_MAX;
    while (scanf("%d", &x) == 1) {
        if (x > max) max = x;
        if (x < min) min = x;
    }
    printf("%d %d\n", max, min);
    return 0;
}
```

```cpp
// 题目3：计算平均值
#include <stdio.h>
int main() {
    int x, count = 0;
    long long sum = 0;
    while (scanf("%d", &x) == 1) {
        sum += x;
        count++;
    }
    if (count > 0) {
        printf("%.2f\n", (double)sum / count);
    }
    return 0;
}
```

```cpp
// 题目4：计算多组阶乘
#include <stdio.h>
int main() {
    int n, caseNum = 0;
    while (scanf("%d", &n) == 1) {
        caseNum++;
        long long fact = 1;
        for (int i = 1; i <= n; i++) {
            fact *= i;
        }
        printf("Case %d: %lld\n", caseNum, fact);
    }
    return 0;
}
```
