# 教案：09-输入输出框架

## 教学目标

1. **理解scanf返回值**: 知道如何用 while(scanf("%d", &x) == 1) 处理不确定数量的数据
2. **掌握文件读写**: freopen 和 fopen 两种方式
3. **了解比赛规则**: 什么情况下用标准I/O，什么情况下用文件
4. **学会条件编译**: 本地测试用文件，提交用标准I/O
5. **避免常见错误**: 变量未初始化、文件名错误等

## 引入阶段 (5分钟)

### 生活类比：快递站的工作流程

想象一个快递站：

## 讲解阶段 (15分钟)

### 核心问题：如何处理不确定数量的输入？

**错误方法**（不知道有多少数据）：
```c
// 不知道什么时候结束
scanf("%d", &x);  // 怎么判断结束？
```

**正确方法**（利用scanf返回值）：
```c
while(scanf("%d", &x) == 1) {
    // 处理x
}
```

**原理**: scanf返回成功读取的变量个数，读到文件末尾返回0

## 文件读写框架 (10分钟)

### 方法1：重定向方式（简单）
```c
#define LOCAL
#include <stdio.h>

#ifdef LOCAL
freopen("data.in", "r", stdin);
freopen("data.out", "w", stdout);
#endif

int main() {
    // 程序代码
    return 0;
}
```

### 方法2：fopen方式（灵活）
```c
FILE *fin = fopen("data.in", "rb");
FILE *fout = fopen("data.out", "wb");
fscanf(fin, "%d", &x);
fprintf(fout, "%d\n", ans);
fclose(fin);
fclose(fout);
```

## 总结 (5分钟)

### 本课要点

1. **scanf返回值判断结束**: while(scanf("%d", &x) == 1)
2. **两种文件读写方式**: freopen（简单）和fopen（灵活）
3. **条件编译切换**: #ifdef LOCAL ... #endif
4. **变量必须初始化**: 未赋值值不确定
5. **多组数据要重置变量**: 避免影响下一组
