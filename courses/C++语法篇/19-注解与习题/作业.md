## 编程题

### 简单题

**题目1：** 函数调用

**描述：** 编写一个函数 `int max2(int a, int b)`，返回两个整数中的较大值。然后在主函数中输入两个整数，调用这个函数，输出较大值。

```
示例输入：10 20
示例输出：20
```

**提示：** 用 if 语句判断，或者用三元运算符

---

**题目2：** 阶乘计算

**描述：** 编写一个递归函数 `int factorial(int n)`，计算 n 的阶乘。在主函数中输入一个非负整数 n，输出 n!。

```
示例输入：5
示例输出：120
```

**提示：** 递归出口是 n=0 或 n=1 时返回 1

---

**题目3：** 结构体入门

**描述：** 定义一个结构体 `Point`，包含两个整型成员 `x` 和 `y`，分别表示点的横坐标和纵坐标。输入两个点的坐标，输出它们的和（横坐标相加，纵坐标相加）。

```
示例输入：1 2
        3 4
示例输出：4 6
```

---

### 中等题

**题目4：** 递归求斐波那契

**描述：** 编写递归函数 `int fib(int n)`，返回斐波那契数列第 n 项。斐波那契数列：F(0)=0, F(1)=1, F(n)=F(n-1)+F(n-2)（n>=2）。输入 n，输出 F(n)。

```
示例输入：10
示例输出：55
```

**提示：** 注意 F(0)=0, F(1)=1

## 参考答案

### 编程题参考答案

```cpp
// 题目1：函数调用 - 求较大值
#include <iostream>
using namespace std;

int max2(int a, int b) {
    if (a > b) {
        return a;
    } else {
        return b;
    }
}

int main() {
    int a, b;
    cin >> a >> b;
    cout << max2(a, b) << endl;
    return 0;
}
```

```cpp
// 题目2：阶乘计算（递归）
#include <iostream>
using namespace std;

int factorial(int n) {
    if (n == 0 || n == 1) {
        return 1;
    }
    return n * factorial(n - 1);
}

int main() {
    int n;
    cin >> n;
    cout << factorial(n) << endl;
    return 0;
}
```

```cpp
// 题目4：递归求斐波那契
#include <iostream>
using namespace std;

int fib(int n) {
    if (n == 0) {
        return 0;
    }
    if (n == 1) {
        return 1;
    }
    return fib(n - 1) + fib(n - 2);
}

int main() {
    int n;
    cin >> n;
    cout << fib(n) << endl;
    return 0;
}
```
