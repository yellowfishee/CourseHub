📦 函数基础 - 什么是函数？

函数 = 程序的"自动售货机"！

投入数据 → 处理 → 返回结果

📝 函数的定义语法

返回类型 函数名(参数类型 参数名, ...) {
    // 函数体
    return 结果值;
}

示例：求最大值

int max(int a, int b) {
  if (a > b) return a;
  return b;
}

示例：打招呼

void greet(char name[]) {
  printf("你好，%s!\n", name);
}

📞 函数调用四步曲

1️⃣

暂停主程序

调用函数时，主函数暂停

2️⃣

传递参数

把实参复制给形参（值传递）

3️⃣

执行函数体

按顺序执行函数内的语句

4️⃣

返回结果

把返回值带回调用者，恢复执行

📦 值传递：复印文件

💡 值传递就像复印文件：原文件不变，函数拿到副本

void tryChange(int x) {
    x = x + 100; // 修改的是副本！
}

int main() {
    int a = 10;
    tryChange(a); // 传递a的值（副本）
    printf("a = %d\n", a);
}

函数内

x = 110

函数外

a = 10（不变！）

📍 指针传递：给钥匙

💡 指针传递就像给钥匙：可以通过钥匙直接操作原文件

void change(int *p) {
    *p = *p + 100; // 通过地址直接修改原变量！
}

int main() {
    int a = 10;
    change(&a); // 传递a的地址
    printf("a = %d\n", a);
}

函数内

*p = 110

函数外

a = 110（变了！）

🔄 交换两个数 - 值传递 vs 指针传递

❌ 值传递（错误）

void swap(int a, int b) {
  int temp = a;
  a = b; b = temp;
}

// 调用后 x=10, y=20（没变！）

✅ 指针传递（正确）

void swap(int *a, int *b) {
  int temp = *a;
  *a = *b; *b = temp;
}

// 调用后 x=20, y=10（交换成功！）

📊 数组作为参数

💡 数组名本身就是地址，传递给函数时不需要 & 符号！

void printArray(int arr[], int n) {
    for(int i = 0; i < n; i++)
        printf("%d ", arr[i]);
}

int main() {
    int a[5] = {1,2,3,4,5};
    printArray(a, 5); // a 就是地址！
}

数组是地址传递！

在函数内部修改数组会影响外部的原始数组

🏠 结构体：数据打包

💡 结构体就像一个"数据容器"，把相关数据打包在一起

📋

学生信息

姓名："小明"

年龄：15

分数：95.5

struct Student {
    char name[50]; // 姓名
    int age; // 年龄
    double score; // 分数
};

🖊️ 结构体的使用

// 声明结构体变量
struct Student s1, s2;

// 给成员赋值（使用点号 . 访问）
strcpy(s1.name, "小明");
s1.age = 15;
s1.score = 95.5;

// 访问成员并打印
printf("姓名：%s\n", s1.name);
printf("年龄：%d\n", s1.age);
printf("分数：%.1f\n", s1.score);

输出：

姓名：小明

年龄：15

分数：95.5

🔄 递归基础 - 俄罗斯套娃

💡 递归就像俄罗斯套娃：打开一个大娃娃，里面有一个中娃娃，再打开...

大娃娃
→
中娃娃
→
小娃娃
→
小小娃娃

关键：最小的娃娃是空的！（终止条件）

如果一直打开不停止，就会永远继续！

🎯 递归的两个必要条件

✅ 终止条件

到最小问题时停止

"最小的娃娃是空的"

没有这个 = 无限循环！

✅ 递归公式

大问题分解为小问题

"打开娃娃，里面还有娃娃"

每次都向终止条件靠近

📝 阶乘的递归实现

递归公式：n! = n × (n-1)!

终止条件：1! = 1

int factorial(int n) {
    if (n == 1) return 1; // 终止条件
    return n * factorial(n - 1);
}

factorial(5) = 5 × 4 × 3 × 2 × 1 = 120

📝 斐波那契数列

1, 1, 2, 3, 5, 8, 13, 21, ...

递归公式：Fib(n) = Fib(n-1) + Fib(n-2)

终止条件：Fib(1) = 1, Fib(2) = 1

int fib(int n) {
    if (n <= 2) return 1;
    return fib(n - 1) + fib(n - 2);
}

📚 调用栈：后进先出

调用栈就像一叠盘子：最后放上去的，最先被拿下来

fib(1) - 返回 1
↑
fib(2) - 返回 2
↑
fib(3) - 返回 3
← 后进先出 (LIFO)

每一层递归调用都要等里面那一层算完才能返回

📝 数字各位求和

digitSum(12345) = 1+2+3+4+5 = 15

递归公式：digitSum(n) = n%10 + digitSum(n/10)

终止条件：digitSum(n) = n（当 n < 10 时）

int digitSum(int n) {
    if (n < 10) return n;
    return n % 10 + digitSum(n / 10);
}

🏛️ 汉诺塔问题

将n个盘子从A柱借助B柱移到C柱

三步曲：

1. 把上面的 n-1 个从 A 借助 C 移到 B

2. 把最大的从 A 直接移到 C

3. 把 n-1 个从 B 借助 A 移到 C

void hanoi(int n, char from, char to, char temp) {
    if (n == 0) return;
    hanoi(n-1, from, temp, to);
    printf("%c→%c ", from, to);
    hanoi(n-1, temp, to, from);
}

🔄 递归 vs 循环 怎么选？

🔁 循环

🪆 递归

适合分治问题
