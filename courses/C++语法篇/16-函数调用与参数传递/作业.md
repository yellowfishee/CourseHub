## 编程题

### 简单题

**题目1：** 指针交换变量

**描述：** 编写一个函数 `swap`，接收两个整型指针，在函数内部交换它们指向的值。在 main 函数中读入两个整数，调用 swap 后输出交换后的两个数。

**示例输入：**
```
3 5
```

**示例输出：**
```
5 3
```

---

**题目2：** 求数组最大值（用函数）

**描述：** 编写一个函数 `max_of_array`，接收一个整数数组和数组长度，返回数组中的最大值。在 main 函数中读入数组长度n和n个整数，调用该函数并输出最大值。

**示例输入：**
```
5
3 9 1 7 5
```

**示例输出：**
```
9
```

---

**题目3：** 指针判断奇偶

**描述：** 编写一个函数 `is_even`，接收一个整型指针，如果该指针指向的数是偶数返回1，否则返回0。在 main 函数中读入一个整数，调用该函数并输出 "EVEN" 或 "ODD"。

**示例输入1：**
```
10
```

**示例输出1：**
```
EVEN
```

**示例输入2：**
```
7
```

**示例输出2：**
```
ODD
```

### 中等题

**题目4：** 数组值反转

**题目来源：** 自出

**描述：** 编写一个函数 `reverse_array`，接收一个整数数组和数组长度，将数组中的元素反转（不返回新数组，直接在原数组上修改）。在 main 函数中读入数组长度n和n个整数，调用 reverse_array 后输出反转后的数组。

**示例输入：**
```
5
1 2 3 4 5
```

**示例输出：**
```
5 4 3 2 1
```

**提示：** 使用指针或下标都可以，只需遍历数组的前 n/2 个元素，将它与对应的后半部分元素交换。

### 编程题参考答案

```cpp
// 题目1：指针交换变量
#include <stdio.h>
void swap(int *a, int *b) {
    int t = *a;
    *a = *b;
    *b = t;
}
int main() {
    int x, y;
    scanf("%d %d", &x, &y);
    swap(&x, &y);
    printf("%d %d\n", x, y);
    return 0;
}
```

```cpp
// 题目2：求数组最大值（用函数）
#include <stdio.h>
int max_of_array(int a[], int n) {
    int max = a[0];
    for (int i = 1; i < n; i++) {
        if (a[i] > max) {
            max = a[i];
        }
    }
    return max;
}
int main() {
    int n;
    scanf("%d", &n);
    int a[105];
    for (int i = 0; i < n; i++) {
        scanf("%d", &a[i]);
    }
    printf("%d\n", max_of_array(a, n));
    return 0;
}
```

```cpp
// 题目3：指针判断奇偶
#include <stdio.h>
int is_even(int *p) {
    return (*p) % 2 == 0;
}
int main() {
    int x;
    scanf("%d", &x);
    if (is_even(&x)) {
        printf("EVEN\n");
    } else {
        printf("ODD\n");
    }
    return 0;
}
```

```cpp
// 题目4：数组值反转
#include <stdio.h>
void reverse_array(int a[], int n) {
    for (int i = 0; i < n / 2; i++) {
        int t = a[i];
        a[i] = a[n - 1 - i];
        a[n - 1 - i] = t;
    }
}
int main() {
    int n;
    scanf("%d", &n);
    int a[105];
    for (int i = 0; i < n; i++) {
        scanf("%d", &a[i]);
    }
    reverse_array(a, n);
    for (int i = 0; i < n; i++) {
        printf("%d", a[i]);
        if (i < n - 1) printf(" ");
    }
    printf("\n");
    return 0;
}
```
