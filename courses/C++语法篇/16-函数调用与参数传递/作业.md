## 参考答案

**选择题：** 1-B, 2-B, 3-B, 4-B, 5-B

**填空题：**
6. &
7. *
8. 地址/指针
9. 地址
10. 地址

**编程题答案：**

```c
// 题目1
void swap(int *p, int *q) {
    int temp = *p;
    *p = *q;
    *q = temp;
}

// 题目2
void increment(int *p) {
    (*p)++;
}

// 题目3
int* findMax(int *a, int n) {
    int *max = a;
    for(int i = 1; i < n; i++)
        if(a[i] > *max) max = &a[i];
    return max;
}
```
