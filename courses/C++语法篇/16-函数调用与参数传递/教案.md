# 16-函数调用与参数传递 - 教案

---

## 教学目标

1. 知识目标：理解值传递和指针传递的区别
2. 能力目标：能够使用指针在函数中修改外部变量
3. 情感目标：培养严谨的编程思维

## 教学重点

- 值传递的原理
- 指针的概念和使用
- 指针传递的应用

## 教学难点

- 理解指针和地址的概念
- 正确使用解引用运算符

## 教学过程

### 导入（5分钟）

提问：为什么swap函数交换不了两个变量的值？

### 新课讲解（30分钟）

**值传递**
- 传递数据的副本
- 函数内部修改不影响外部变量

**指针基础**
- 指针是存储地址的变量
- &运算符：取地址
- *运算符：解引用

**指针传递**
- 传递地址而不是值
- 函数内部可以修改外部变量

### 实践操作（10分钟）

学生动手编写指针交换程序。

---

## 提示与答案

### 提示

1. &x 表示变量x的地址
2. *p 表示访问p指向的变量
3. 数组名本身就是地址

### 参考答案

**编程题答案：**

```c
// swap函数（指针版本）
void swap(int *p, int *q) {
    int temp = *p;
    *p = *q;
    *q = temp;
}
```
