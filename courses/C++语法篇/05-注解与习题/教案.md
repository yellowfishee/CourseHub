# 教案：05-注解与习题

## 教学目标

1. **了解C语言标准**: 知道C89/C99/C11的区别和历史
2. **掌握数据类型**: int, double, char等的基本特性
3. **熟悉输入格式**: %d, %f, %lf, %c等的正确使用
4. **避免常见错误**: 格式符不匹配、变量未初始化等

## 引入阶段 (5分钟)

### 快速回顾

回顾第1章学习的四大内容：
1. **算术表达式**: 计算功能
2. **变量及其输入**: 存储和读取数据
3. **顺序结构**: 一步步执行
4. **分支结构**: 根据条件选择

## 讲解阶段 (15分钟)

### C语言标准

- **C89/C90**: 第一个ANSI标准
- **C99**: 加入了//注释、bool类型等
- **C11**: 最新的C标准

### 提示提取

- **提示1-1**: C语言使用/*...*/作为注释，不能嵌套
- **提示1-2**: printf和scanf的返回值含义不同
- **提示1-3**: 赋值表达式有返回值
- **提示1-4**: 整数除法和浮点数除法的区别
- **提示1-5**: 整数溢出问题
- **提示1-6**: 强制类型转换不会改变原变量

### 数据类型与输入格式

- **int**: 整数，约-21亿到21亿
- **double**: 浮点数，双精度
- **char**: 字符
- **%d**: int类型
- **%f**: float类型
- **%lf**: double类型
- **%c**: char类型

## 交互练习 (15分钟)

判断各个说法是否正确

### 练习2: 选择正确的格式符
给定变量类型，选择正确的scanf/printf格式符

### 练习3: 找出bug
分析代码中的错误

## 总结 (5分钟)

### 本课要点

1. C语言有多个标准版本
2. 不同数据类型占用空间不同
3. 格式符必须与变量类型匹配
4. 避免常见错误
