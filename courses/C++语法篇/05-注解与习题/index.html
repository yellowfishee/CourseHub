🔢 数据类型

类型 含义 格式符 示例
int 整数 %d 123, -456
double 浮点数 %lf 3.14, -0.5
char 字符 %c 'a', 'Z'

💡 int 范围

通常是 -2,147,483,648 ~ 2,147,483,647（约 -21亿 ~ +21亿）

📝 格式说明符

printf

%d - int

%f - float, double

%c - char

%.2f - 保留2位小数

scanf

%d - int

%lf - double (注意是lf)

%c - char

%s - 字符串

⚠️ 注意

scanf 读取 double 用 %lf，而 printf 用 %f！

🐛 常见错误

❌ 错误1：格式符不匹配

double x; scanf("%f", &x); // 错！

✓ scanf("%lf", &x);

❌ 错误2：忘记&

scanf("%d", x); // 错！

✓ scanf("%d", &x);

❌ 错误3：整数除法

printf("%f", 5/2); // 输出0.000000

✓ printf("%f", 5.0/2);

📋 赋值表达式

赋值表达式有返回值！

int a, b;
a = b = 3;  // 从右到左执行
// a = 3, b = 3

int x = (5 + (3 = 2)); // 错误！

💡 提示1-3

赋值表达式本身也有值，等于被赋值变量的值。但赋值号左边必须是变量，不能是表达式。

⚠️ 整数溢出

int 能表示的范围有限：

int a = 2147483647;
a = a + 1;  // 溢出！变成 -2147483648

int b = -2147483648;
b = b - 1;  // 溢出！变成 2147483647

💡 提示1-5

整数溢出是一种未定义行为，应该尽量避免。

🔄 强制类型转换

double pi = 3.14159;
int n = (int)pi;  // n = 3，小数部分被截断

int a = 5, b = 2;
printf("%f", (double)a / b); // 输出 2.500000

💡 提示1-6

强制类型转换不会改变原变量，只是生成了一个新的转换后的值。
