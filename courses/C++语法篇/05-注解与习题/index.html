逻辑命令：而且、或者 🔗

🤝

而且(AND) = 两个都要满足

就像进游乐园：要买票而且身高够高才能玩 

if (age >= 12 && height >= 150) {
    cout << "可以玩过山车";
}
🍕🍔

或者(OR) = 满足一个就行

就像点餐：要米饭或者要面条（有一样就行） 

if (food == "米饭" || food == "面条") {
    cout << "选择正确";
}

小测验：逻辑命令 🧠

age = 20, height = 140
这个条件能通过吗？

if (age >= 12 && height >= 150) {
    cout << "可以玩过山车";
}
A 能通过（两个条件都满足）
B 不能通过（身高不够140<150）

提示：而且(&&)需要两个条件都满足才行！

多层分支：else if 🏢

🎢

else if 就像多层筛选

就像买电影票：
成人票 > 学生票 > 儿童票 > 老人票 

if (age >= 18) {
    cout << "成人票";
} else if (age >= 6) {
    cout << "学生票";
} else {
    cout << "儿童票";
}

小测验：else if 🧠

如果 age = 6，会显示什么？

if (age > 6) {
    cout << "学生票";
} else if (age >= 6) {
    cout << "儿童票";
} else {
    cout << "婴儿票";
}
A "学生票"
B "儿童票"
C "婴儿票"

提示：6 > 6 是假的，但 6 >= 6 是真的！

常见错误总结 ⚠️

❌ 错误：用一个等号比较

if (a = 5)  // 这是赋值！不是比较！

✓ 正确：用两个等号比较

if (a == 5)  // 这是比较！

❌ 错误：忘记大括号

if (a > b)
    cout << a;  // 只有这一行受if控制！
    cout << b;  // 这行不受if控制！

逻辑命令总结 🧮

&&

而且 (AND)

两边都要对才是对

true && true = true

true && false = false

||

或者 (OR)

有一边对就是对

true || false = true

false || false = false

🎯 小技巧：而且要两个都对，或者有一个对就行
