## 编程题

### 简单题

**题目1：** 交换两个变量

**描述：** 输入两个整数 a 和 b，交换它们的值后输出。

```
示例输入：
3 5
示例输出：
5 3
```

---

**题目2：** 计算各位数字之和

**描述：** 输入一个三位数，输出各位数字之和。
例如：123 → 1 + 2 + 3 = 6

```
示例输入：
456
示例输出：
15
```

---

**题目3：** ASCII码转换

**描述：** 输入一个字符，输出它的ASCII码值。

```
示例输入：
a
示例输出：
97
```

---

### 中等题

**题目4：** 判断三位数

**描述：** 输入一个三位数，判断它是不是"水仙花数"。
水仙花数：一个三位数，它的各位数字的立方和等于它本身。
例如：153 = 1³ + 5³ + 3³，所以 153 是水仙花数。

```
示例输入1：
153
示例输出1：
Yes

示例输入2：
124
示例输出2：
No
```

**提示：** 先分离出百位、十位、个位，然后计算立方和进行比较。

### 编程题参考答案

```cpp
// 题目1：交换两个变量
#include <stdio.h>
int main() {
    int a, b, t;
    scanf("%d%d", &a, &b);
    t = a;
    a = b;
    b = t;
    printf("%d %d\n", a, b);
    return 0;
}

// 题目2：计算各位数字之和
#include <stdio.h>
int main() {
    int n;
    scanf("%d", &n);
    int bai = n / 100;
    int shi = n / 10 % 10;
    int ge = n % 10;
    printf("%d\n", bai + shi + ge);
    return 0;
}

// 题目3：ASCII码转换
#include <stdio.h>
int main() {
    char c;
    scanf("%c", &c);
    printf("%d\n", (int)c);
    return 0;
}

// 题目4：判断水仙花数
#include <stdio.h>
int main() {
    int n;
    scanf("%d", &n);
    int bai = n / 100;
    int shi = n / 10 % 10;
    int ge = n % 10;
    if (bai*bai*bai + shi*shi*shi + ge*ge*ge == n) {
        printf("Yes\n");
    } else {
        printf("No\n");
    }
    return 0;
}
```
