🔄 C语言 vs C++

C++在C语言的基础上增加了许多新特性，让编程更方便、更安全

C语言特点

  • • 面向过程的编程
  • • 需要手动管理内存
  • • 使用malloc/free
  • • 没有命名空间概念
  • • printf/scanf输入输出

  • • 引用（Reference）
  • • 命名空间（Namespace）

📍 引用是什么？

引用就是变量的"别名"，就像一个人的外号一样

👤

原名：张三

变量名：a

=
📛

外号：小张

引用名：r

=
👤📛

都是同一个人！

修改r就是修改a

💡 关键点

引用不是复制，而是同一个变量的另一个名字！

📝 引用的定义和使用

// 引用的定义语法：类型 & 引用名 = 原变量名;

int a = 10;  // 定义整型变量 a
int &r = a;  // 定义引用 r，它是变量 a 的别名

printf("a = %d, r = %d\n", a, r);  // 输出：a = 10, r = 10
r = 20;  // 修改 r，就是修改 a
printf("a = %d, r = %d\n", a, r);  // 输出：a = 20, r = 20

✓ 正确写法

int a = 10;
int &r = a;

✗ 错误写法

int &r;  // 错误：引用必须初始化！
int &r = 100;  // 错误：不能直接绑定数字！

⚖️ 引用 vs 指针

引用（Reference）

  • • 定义时必须初始化
  • • 初始化后不能改变（不能重新绑定）
  • • 使用时不需要 * 解引用
  • • 语法更简洁、更直观
  • • 更安全（不可能是空值）
int &r = a;
r = 20;  // 直接使用

指针（Pointer）

  • • 定义时不需要初始化
  • • 可以随时改变指向
  • • 使用时需要 * 解引用
  • • 可以是NULL（空指针）
  • • 更灵活但也更危险
int *p = &a;
*p = 20;  // 需要解引用

💡 建议

能用引用解决的问题优先用引用，指针更灵活但也更危险。C++中引用是首选！

📞 引用作为函数参数

使用引用作为函数参数，可以实现类似指针的效果，但语法更简洁！

使用指针（可以，但语法复杂）

void swap(int *p, int *q) {
  int temp = *p;
  *p = *q;
  *q = temp;
}

// 调用
swap(&x, &y);

使用引用（推荐！）

void swap(int &a, int &b) {
  int temp = a;
  a = b;
  b = temp;
}

// 调用
swap(x, y);

✓ 引用版本的 swap 调用更直观，不需要取地址！

💡 生活类比

就像一个学校有两个叫"张三"的学生，学校用"一年级张三"和"二年级张三"来区分他们。命名空间就是给名字加上了"年级"前缀！

📝 命名空间的定义和使用

// 定义命名空间
namespace Math {
  int add(int a, int b) { return a + b; }
  int sub(int a, int b) { return a - b; }
}

// 使用命名空间中的函数
int result = Math::add(10, 20);  // 使用 :: 作用域运算符

方法1：直接使用

Math::add(1, 2);

方法2：using声明

using Math::add;
add(1, 2);

方法3：using编译指令

using namespace Math;
add(1, 2);

📦 常用的std命名空间

C++标准库的所有内容都在 std 命名空间中

C++标准输入输出

#include <iostream>

// 需要使用 std:: 或 using
std::cout << "Hello" << std::endl;
std::cin >> x;

使用using简化

#include <iostream>
using namespace std;

// 直接使用
cout << "Hello" << endl;
cin >> x;

💡 建议

初学时可以使用 using namespace std; 简化代码，但在大型项目中建议使用 std:: 前缀明确指出命名空间。

🗄️ new和delete：动态内存管理

new和delete是C++特有的动态内存分配方式，比malloc/free更安全

C语言方式（malloc/free）

int *p = (int *)malloc(sizeof(int));
free(p);

C++方式（new/delete）

int *p = new int;
delete p;

// 分配数组
int *arr = new int[100];
delete[] arr;

⚠️ 重要规则

malloc/free 和 new/delete 不能混用！用 new 分配的内存必须用 delete 释放，用 new[] 分配的必须用 delete[] 释放！

📥 C++输入：cin

cin是C++的标准输入流对象，比scanf更简单直观

#include <iostream>
using namespace std;

int a, b;
cin >> a >> b;  // 输入两个整数

double x;
cin >> x;  // 输入浮点数

char s[100];
cin >> s;  // 输入字符串（不含空格）

✓ cin 的优点

  • • 自动识别变量类型
  • • 不需要格式说明符
  • • 语法更简单

✗ cin 的缺点

  • • 输入速度比scanf慢
  • • 对空格处理不如scanf灵活

📤 C++输出：cout

cout是C++的标准输出流对象，比printf更简单直观

#include <iostream>
using namespace std;

int a = 10;
double x = 3.14;

cout << "Hello World" << endl;
cout << "a = " << a << endl;
cout << "x = " << x << endl;
cout << a << " + " << x << " = " << a + x << endl;

endl 是什么？

endl 表示换行（end of line），相当于 printf 中的 \n

📝 cin/cout 完整示例

// C++风格的 Hello World
#include <iostream>
using namespace std;

int main() {
  // 输入两个数
  int a, b;
  cout << "请输入两个整数：";
  cin >> a >> b;

  // 输出运算结果
  cout << a << " + " << b << " = " << a + b << endl;
  cout << a << " - " << b << " = " << a - b << endl;
  cout << a << " * " << b << " = " << a * b << endl;
  cout << a << " / " << b << " = " << a / b << endl;

  return 0;
}

运行效果：

请输入两个整数：10 20

10 + 20 = 30

10 - 20 = -10

10 * 20 = 200

10 / 20 = 0

📋 本章总结

引用

  • • 变量的别名

  • • 不能重新绑定
  • • 更安全、更简洁

  • • 避免名字冲突
