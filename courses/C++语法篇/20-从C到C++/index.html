📍 引用是什么？

💡 生活类比：外号

👤

原名：张三

变量名：a

=
📛

外号：小张

引用名：r

=
👤📛

都是同一个人！

修改r就是修改a

引用 = 变量的"别名"，同一个东西的两个名字！

📝 引用的定义和使用

// 定义语法：类型 & 引用名 = 原变量;

int a = 10; // 定义变量 a
int &r = a; // 定义引用 r，它是 a 的别名

printf("a=%d, r=%d\n", a, r); // a=10, r=10
r = 20; // 修改 r，就是修改 a！
printf("a=%d, r=%d\n", a, r); // a=20, r=20

📋 引用的规则

✅ 正确写法
int a = 10;
int &r = a; // 正确：初始化为已有变量
❌ 错误写法
int &r; // 错误！引用必须初始化
int &r = 100; // 错误！不能绑定数字

⚠️ 引用必须在定义时初始化，且之后不能改变！

⚖️ 引用 vs 指针

📍 引用

定义时必须初始化

初始化后不能改变

使用不需要 * 解引用

更安全！

�🔍 指针

不需要初始化

可以随时改变指向

使用需要 * 解引用

更灵活但更危险！

📞 引用作为函数参数 - swap例子

❌ 用指针（麻烦）
void swap(int *a, int *b) {
  int temp = *a;
  *a = *b; *b = temp;
}

// 调用要取地址
swap(&x, &y);
✅ 用引用（简单！）
void swap(int &a, int &b) {
  int temp = a;
  a = b; b = temp;
}

// 调用直接传变量
swap(x, y);

命名空间 = 给名字加"前缀"，避免冲突！

📝 命名空间的定义和使用

// 定义命名空间
namespace Math {
    int add(int a, int b) { return a + b; }
    int sub(int a, int b) { return a - b; }
}

// 使用：命名空间名::成员
int result = Math::add(10, 20);

:: 叫作用域运算符，表示"属于哪个空间"

📋 三种使用命名空间的方式

方式1：直接用::

Math::add(1, 2);

方式2：using声明

using Math::add;
add(1, 2);

方式3：using namespace

using namespace Math;
add(1, 2);

📦 std 命名空间

C++标准库的所有内容都在 std 命名空间中！

cin, cout, endl, vector, string... 都属于 std

写全称

std::cout << "你好" << std::endl;
std::cin >> x;

用 using（推荐入门）

using namespace std;
cout << "你好" << endl;
cin >> x;

🗄️ new和delete是什么？

💡 生活类比：租房

🏠

租房子

new（申请内存）

→
🏃‍♂️

住进去

使用内存

→
🏠❌

退房子

delete（释放内存）

⚠️ 用完必须还回去！不然就是"内存泄漏"！

📝 new和delete语法

❌ C语言方式

int *p = (int*)malloc(sizeof(int));
...
free(p);

✅ C++方式

int *p = new int;
...
delete p;

// 数组用 new[]
int *arr = new int[100];
delete[] arr;

📋 new/delete 重要规则

⚠️ 配对使用！

new → delete

new[] → delete[]

❌ 不能混用！

用 new 分配的不能用 free 释放

用 malloc 分配的不能用 delete 释放

✅ 推荐

在C++中尽量用 new/delete

更安全、更现代

📥📤 cin和cout是什么？

📥 cin

C++的标准输入

相当于C的 scanf

cin >> a >> b;
📤 cout

C++的标准输出

相当于C的 printf

cout << "Hello" << endl;

📥 cin 输入详解

#include <iostream>
using namespace std;

int a, b;
cin >> a >> b; // 输入两个整数

double x;
cin >> x; // 输入浮点数

char s[100];
cin >> s; // 输入字符串（不含空格）

💡 cin 会自动识别变量类型，不需要格式说明符！

📤 cout 输出详解

#include <iostream>
using namespace std;

int a = 10;
double x = 3.14;

cout << "Hello World" << endl; // 输出字符串并换行
cout << "a = " << a << endl; // a = 10
cout << "x = " << x << endl; // x = 3.14
cout << a << " + " << x << " = " << a + x << endl;

💡 endl 表示"换行"，相当于 printf 的 \n

📝 cin/cout 完整示例

// C++风格的计算器
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cout << "输入两个数：";
    cin >> a >> b;

    cout << a << " + " << b << " = " << a+b << endl;
    cout << a << " - " << b << " = " << a-b << endl;
    return 0;
}

📥 scanf

缺点
