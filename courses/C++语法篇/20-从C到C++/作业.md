## 编程题

### 简单题

**题目1：** 引用入门

**描述：** 编写一个函数 `void swap(int &a, int &b)`，使用引用作为参数，交换两个整数的值。在主函数中输入两个整数，调用函数交换后，输出交换后的两个数。

```
示例输入：10 20
示例输出：20 10
```

**提示：** 引用就是变量的别名，在函数里直接修改引用的值就是修改原变量

---

**题目2：** 命名空间

**描述：** 下面的代码中，有两个同名变量 `x`，一个在 `namespace A` 中，值为 100；一个在 `namespace B` 中，值为 200。请使用命名空间来访问这两个变量，输出它们的和。

```cpp
#include <iostream>
using namespace std;

namespace A {
    int x = 100;
}

namespace B {
    int x = 200;
}

int main() {
    // 你的代码：输出 A::x + B::x 的值
    return 0;
}
```

```
示例输出：300
```

**提示：** 使用 `命名空间::变量名` 的方式访问

**题目3：** new和delete

**描述：** 使用 new 动态分配一个整数，初始化为 42，然后输出它的值，最后用 delete 释放内存。

```
示例输出：42
```

**提示：** new 会返回一个指针，用 *指针 来访问值

### 中等题

**题目4：** 动态数组

**描述：** 使用 new 动态创建一个长度为 n 的整数数组，输入 n 和 n 个整数，输出这 n 个整数的和。

```
示例输入：5
        1 2 3 4 5
示例输出：15
```

**提示：** `new int[n]` 创建一个长度为 n 的数组，返回指针

## 参考答案

### 编程题参考答案

```cpp
// 题目1：引用入门 - 交换两数
#include <iostream>
using namespace std;

void swap(int &a, int &b) {
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    int a, b;
    cin >> a >> b;
    swap(a, b);
    cout << a << " " << b << endl;
    return 0;
}
```

```cpp

using namespace std;

namespace A {
    int x = 100;
}

namespace B {
    int x = 200;
}
```

```cpp
// 题目3：new和delete - 动态内存
#include <iostream>
using namespace std;

int main() {
    int *p = new int;
    *p = 42;
    cout << *p << endl;
    delete p;
    return 0;
}
```

```cpp
// 题目4：动态数组 - 求和
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int *arr = new int[n];
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }
    int sum = 0;
    for (int i = 0; i < n; i++) {
        sum += arr[i];
    }
    cout << sum << endl;
    delete[] arr;
    return 0;
}
```
