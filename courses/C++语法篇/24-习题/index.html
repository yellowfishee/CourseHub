💻 完整程序示例

输入5个数，求最大值：

#include <iostream>
#include <vector>
using namespace std;

int main() {
    int n;
    cin >> n;
    vector<int> v(n);
    for (int i = 0; i < n; i++) {
        cin >> v[i];
    }
    
    int maxVal = v[0];
    for (int i = 1; i < n; i++) {
        if (v[i] > maxVal) maxVal = v[i];
    }
    cout << maxVal << endl;
    return 0;
}

⚠️ 易错点总结

数组下标从0开始

a[5]的数组，最大下标是4

for循环注意结束条件

i < n 不是 i <= n

string用+拼接

不是"Hi" + "World"而是"Hi" + " " + "World"

递归必须有终止条件

否则会无限递归导致崩溃
