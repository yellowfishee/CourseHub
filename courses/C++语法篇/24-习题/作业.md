## 编程题

### 简单题

**题目1：** 顺序结构 - 三位数反转

**描述：** 输入一个三位数，将它的各位数字反序输出（百位变个位，个位变百位）。

```
示例输入：127
示例输出：721
```

**提示：** 用 / 和 % 分离各位数字

---

**题目2：** 分支结构 - 比较大小

**描述：** 输入三个整数，输出最大的那个。

```
示例输入：3 8 1
示例输出：8
```

**提示：** 可以用 if-else，也可以用 max 函数

---

**题目3：** 循环结构 - 求和

**描述：** 输入正整数 n，计算 1+2+...+n 的和。

```
示例输入：100
示例输出：5050
```

**提示：** 用 for 循环累加，或者用公式 n*(n+1)/2

---

### 中等题

**题目4：** 综合 - 质数判断

**描述：** 输入一个正整数 n（n<=100000），判断它是否是质数。质数是只能被 1 和自己整除的大于 1 的整数。输出 "yes" 表示是质数，"no" 表示不是。

```
示例输入1：17
示例输出1：yes

示例输入2：18
示例输出2：no
```

**提示：** 只需要检查到 sqrt(n)，如果有任何数能整除 n，则不是质数

## 参考答案

### 编程题参考答案

```cpp
// 题目1：三位数反转
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int bai = n / 100;
    int shi = n / 10 % 10;
    int ge = n % 10;
    cout << ge * 100 + shi * 10 + bai << endl;
    return 0;
}
```

```cpp
// 题目2：比较大小 - 求最大值
#include <iostream>
using namespace std;

int main() {
    int a, b, c;
    cin >> a >> b >> c;
    int max_val = a;
    if (b > max_val) max_val = b;
    if (c > max_val) max_val = c;
    cout << max_val << endl;
    return 0;
}
```

```cpp
// 题目3：求和
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    long long sum = 0;
    for (int i = 1; i <= n; i++) {
        sum += i;
    }
    cout << sum << endl;
    return 0;
}
```

```cpp
// 题目4：质数判断
#include <iostream>
#include <cmath>
using namespace std;

bool isPrime(int n) {
    if (n <= 1) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false;
    for (int i = 3; i * i <= n; i += 2) {
        if (n % i == 0) return false;
    }
    return true;
}

int main() {
    int n;
    cin >> n;
    if (isPrime(n)) {
        cout << "yes" << endl;
    } else {
        cout << "no" << endl;
    }
    return 0;
}
```
