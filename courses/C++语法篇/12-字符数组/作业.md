## 编程题

### 简单题

**题目1：统计字符个数**

**描述：** 输入一个字符串（不超过100个字符），统计其中字母 'a' 出现的次数。

```
示例输入：
abracadabra

示例输出：
5
```

---

**题目2：字符串长度**

**描述：** 输入一个字符串（不超过100个字符），输出它的长度。不要使用 strlen 函数，自己遍历计算。

```
示例输入：
hello

示例输出：
5
```

---

**题目3：判断回文串**

**描述：** 输入一个字符串（不超过100个字符），判断它是否是回文串（正读和反读都一样）。是输出 "YES"，否输出 "NO"。

```
示例输入：
level

示例输出：
YES
```

**提示：** 比较第 i 个和倒数第 i 个字符

---

### 中等题

**题目4：字符串压缩**

**描述：** 输入一个字符串（只包含小写字母，长度不超过100），统计每个字母出现的次数，输出压缩后的格式。例如 "aaabbbcc" 压缩后输出 "a3b3c2"。

```
示例输入：
aaabbbcc

示例输出：
a3b3c2
```

**提示：** 用数组统计每个字母出现次数，然后遍历输出

## 参考答案

### 选择题答案
1-B, 2-C, 3-B, 4-C, 5-B, 6-B

### 编程题参考答案

```cpp
// 题目1：统计字符个数
#include <stdio.h>
int main() {
    char s[101];
    scanf("%s", s);
    int count = 0;
    for (int i = 0; s[i] != '\0'; i++) {
        if (s[i] == 'a') {
            count++;
        }
    }
    printf("%d\n", count);
    return 0;
}
```

```cpp
// 题目2：字符串长度
#include <stdio.h>
int main() {
    char s[101];
    scanf("%s", s);
    int len = 0;
    while (s[len] != '\0') {
        len++;
    }
    printf("%d\n", len);
    return 0;
}
```

```cpp
// 题目3：判断回文串
#include <stdio.h>
int main() {
    char s[101];
    scanf("%s", s);
    int len = 0;
    while (s[len] != '\0') len++;
    
    int isPalindrome = 1;
    for (int i = 0, j = len - 1; i < j; i++, j--) {
        if (s[i] != s[j]) {
            isPalindrome = 0;
            break;
        }
    }
    printf("%s\n", isPalindrome ? "YES" : "NO");
    return 0;
}
```

```cpp
// 题目4：字符串压缩
#include <stdio.h>
int main() {
    char s[101];
    scanf("%s", s);
    int count[26] = {0};
    
    for (int i = 0; s[i] != '\0'; i++) {
        count[s[i] - 'a']++;
    }
    
    for (int i = 0; i < 26; i++) {
        if (count[i] > 0) {
            printf("%c%d", 'a' + i, count[i]);
        }
    }
    printf("\n");
    return 0;
}
```
