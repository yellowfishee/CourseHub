💡 重要

⚠️ gets 的危险

❌ gets（危险！）

char s[10];
gets(s);  // 不检查边界！

如果输入超过9个字符，会溢出！

✅ fgets（安全）

char s[100];
fgets(s, sizeof(s), stdin);

指定最大长度，安全！

💡 提示3-8/3-9

gets 存在"缓冲区溢出"漏洞，永远不要使用gets，应该用 fgets 代替。

💥 缓冲区溢出

当输入超过数组大小时，多余的字符会覆盖其他内存：

char name[10];
char password[20];

printf("请输入名字：");
gets(name);  // 输入: "abcdefghijklmnop"

// name装不下，溢出了！
// password的内容被破坏了！

⚠️ 缓冲区溢出是很多安全漏洞的根源！

✅ fgets 详解

char s[100];
fgets(s, sizeof(s), stdin);

参数1

数组名

s

参数2

最大字符数

sizeof(s)

参数3

输入来源

stdin

💡 注意

fgets 会保留输入中的换行符 '\n'，如果不想保留，可以用 s[strlen(s)-1] = '\0' 去掉

🔧 字符串函数

这些函数都在 头文件中：

函数 作用 示例
strlen 求长度 strlen("Hi") = 2

📏 strlen 求长度

char s[] = "Hello";
int len = strlen(s);  // len = 5

// 注意：strlen不计算'\0'
printf("%d", sizeof(s));  // 6（包含'\0'）

strlen

= 5 个字符

sizeof

= 6 字节（包含\0）

💡 区别

strlen = 字符数（不算'\0'）
sizeof = 数组大小（算'\0'）

🔍 strcmp 比较字符串

int strcmp(const char *s1, const char *s2);

// 返回值：
// = 0 两个字符串相等
// < 0 s1 < s2（字典序）
// > 0 s1 > s2（字典序）

比较示例：

strcmp("apple", "banana")  // < 0（a < b）
strcmp("hello", "hello")  // = 0（相等）
strcmp("xyz", "abc")  // > 0（x > a）

📋 strcpy 和 strcat

strcpy - 复制

char a[20];
strcpy(a, "Hello");
// a 变成了 "Hello"

strcat - 连接

char a[20] = "Hello";
strcat(a, "World");
// a 变成了 "HelloWorld"

⚠️ 注意

strcpy 和 strcat 不会检查目标数组是否够大，使用时要确保数组足够大！

🎯 填空练习

在空格中填入正确的数字：

char s[] = "Hi";
printf("%d", strlen(s));  // 输出
printf("%d", sizeof(s));  // 输出

提示：strlen不计算\0，sizeof计算\0

🐛 常见Bug

❌ Bug 1: 忘记加 &

char s[100];
scanf("%s", s);  // 正确！s已经是数组名

❌ Bug 2: 越界

char s[5];
strcpy(s, "Hello");  // 危险！需要6个字节

❌ Bug 3: 用 == 比较字符串

char a[10], b[10];
if(a == b)  // 错误！比较的是地址，不是内容
if(strcmp(a, b) == 0)  // 正确！

📝 例题：回文串

判断一个字符串是否是回文（正读反读都一样）：

char s[100];
fgets(s, sizeof(s), stdin);
int len = strlen(s) - 1;  // 去掉\n
int is_pal = 1;
for(int i = 0; i < len/2; i++) {
  if(s[i] != s[len-1-i]) {
    is_pal = 0;
    break;
  }
}
