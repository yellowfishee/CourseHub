# 教案：12-字符数组

## 教学目标

1. **理解字符串概念**: 字符串是以'\0'结尾的字符数组
2. **掌握字符串输入**: gets和fgets的区别
3. **学会字符串函数**: strlen, strcmp, strcpy等
4. **了解安全问题**: 缓冲区溢出的危害

## 引入阶段 (5分钟)

### 字符串是什么？

字符串就是"一串字符"，比如 "Hello" 包含5个字符。

在C语言中，字符串用字符数组存储，最后以 '\0'（空字符）结尾。

## 讲解阶段 (15分钟)

### 字符串声明

```c
char s[100];  // 声明一个能存99个字符+'\0'的数组
char s[] = "Hello";  // 自动计算长度
```

### 提示提取

### 重要函数

1. **strlen**: 求字符串长度
2. **strcmp**: 比较两个字符串
3. **strcpy**: 复制字符串
4. **strcat**: 连接字符串

### gets vs fgets

```c
// 危险！gets不检查边界
gets(s);

// 安全！fgets指定最大长度
fgets(s, sizeof(s), stdin);
```
