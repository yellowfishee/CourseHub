🚦 十字路口的比喻

想象你走在路上：

如果红灯亮 → 停！

如果绿灯亮 → 走！

💡 分支结构就是这样！

根据条件是否满足

决定执行哪一段代码

🔀 if 语句

"如果...那么..."

if (条件) {
    执行这里
}

如果条件成立，就执行大括号里的代码

条件不成立，就跳过这段代码

🔢 比较运算符

数学C++意思
>>大于
<<小于
≥>=大于等于
≤<=小于等于
===等于（两个等号！）
≠!=不等于

⚠️ 重要：等于要用两个等号 ==

一个等号 = 是赋值！两个等号 == 是比较！

📝 if 例子

如果年龄大于18，显示"成年"

int age = 20;

if (age >= 18) {
    printf("成年\n");
}

因为 age = 20 ≥ 18，所以显示：成年

↔️ if...else 语句

"如果...否则..."

if (条件) {
    条件成立时执行
} else {
    条件不成立时执行
}

二选一：必执行其中一个！

📝 if...else 例子

判断年龄是成年还是未成年

int age = 15;

if (age >= 18) {
    printf("成年\n");
} else {
    printf("未成年\n");
}

因为 age = 15 < 18，所以显示：未成年

📊 流程图

开始
↓
age >= 18 ?
↓

printf("成年")

❓ 小测验

如果 age = 18，会显示什么？

if (age > 18) {
    printf("大于18");
} else {
    printf("不大于18");
}
大于18
不大于18

提示：age = 18，18 > 18 是 false（相等不是大于）

🔀 else if 多重判断

多个条件，多条路

if (score >= 90) {
    printf("优秀\n");
} else if (score >= 60) {
    printf("及格\n");
} else {
    printf("不及格\n");
}

条件按顺序检查，第一个满足的执行

📝 else if 完整例子

成绩评判（score = 75）

int score = 75;

if (score >= 90) {
    printf("优秀\n");
} else if (score >= 60) {
    printf("及格\n");
} else {
    printf("不及格\n");
}

75 < 90 ✗ → 75 >= 60 ✓ → 显示：及格

🔗 逻辑运算符

数学C++意思
且&&两个都成立
或!取反

&& = AND = "并且"

|| = OR = "或者"

! = NOT = "不是"

📝 逻辑运算例子

// 判断是否在18-60岁之间
int age = 25;

if (age >= 18 && age <= 60) {
    printf("在工作年龄\n");
}

18 <= 25 ✓ 且 25 <= 60 ✓ → 两个都成立 → 在工作年龄

❌ 常见错误

✗ 一个等号

if (a = 5)

这是赋值，不是比较！

✓ 两个等号

if (a == 5)

这是比较！

✗ 少写分号

if (a > 5);
{ ... }

if 后面不能加分号！

✓ 正确写法

if (a > 5)
{ ... }

🎯 拖拽挑战

把正确的比较运算符拖到框里

int age = 18;
