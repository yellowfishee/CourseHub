## 选择题

**1.** 以下哪个是正确的 C++ if 语句？

A. `if (score >= 60) { printf("及格"); }`

B. `if score >= 60: printf("及格");`

C. `if score >= 60 then printf("及格");`

D. `if score >= 60 printf("及格");`

**2.** `=` 和 `==` 的区别是：

A. 没有区别

B. `=` 是比较，`==` 是赋值

C. `=` 是赋值，`==` 是比较

D. `=` 用于条件判断，`==` 用于赋值

**3.** `if (a > 0 && a < 10)` 表示：

A. a 大于 0 或 a 小于 10

B. a 大于 0 且 a 小于 10

C. a 不大于 0 且 a 不小于 10

D. a 等于 0 或 a 等于 10

**4.** `if (score >= 90) printf("A"); else if (score >= 80) printf("B");` 中，如果 score = 85，输出是：

A. A　　　B. B　　　C. A B　　　D. 什么都不输出

A. `year % 4 == 0`

B. `year % 4 == 0 && year % 100 != 0`

## 编程题

### 简单题

**题目1：** 判断正负零

**描述：** 输入一个整数，判断它是正数、负数还是零。

```
示例输入1：5
示例输出1：正数

示例输入2：-3
示例输出2：负数

示例输入3：0
示例输出3：零
```

---

**题目2：** 判断奇偶

**描述：** 输入一个整数，判断它是奇数还是偶数。

```
示例输入1：7
示例输出1：奇数

示例输入2：10
示例输出2：偶数
```

---

**题目3：** 判断是否及格

**描述：** 输入一个成绩（0-100），判断是否及格（60分及以上）。

```
示例输入1：75
示例输出1：及格

示例输入2：45
示例输出2：不及格
```

---

### 中等题

**题目4：** 比较三个数的大小

**描述：** 输入三个整数，输出其中最大的那个。

```
示例输入：
3 8 1
示例输出：
8
```

**提示：** 可以用两个 if 语句比较，或者假设第一个最大再用 if 判断。

## 参考答案

### 选择题答案

1. A, 2. C, 3. B, 4. B, 5. D

### 编程题参考答案

```cpp
// 题目1：判断正负零
#include <stdio.h>
int main() {
    int n;
    scanf("%d", &n);
    if (n > 0) {
        printf("正数\n");
    } else if (n < 0) {
        printf("负数\n");
    } else {
        printf("零\n");
    }
    return 0;
}

// 题目2：判断奇偶
#include <stdio.h>
int main() {
    int n;
    scanf("%d", &n);
    if (n % 2 == 0) {
        printf("偶数\n");
    } else {
        printf("奇数\n");
    }
    return 0;
}

// 题目3：判断是否及格
#include <stdio.h>
int main() {
    int score;
    scanf("%d", &score);
    if (score >= 60) {
        printf("及格\n");
    } else {
        printf("不及格\n");
    }
    return 0;
}

// 题目4：比较三个数的大小
#include <stdio.h>
int main() {
    int a, b, c, max;
    scanf("%d%d%d", &a, &b, &c);
    max = a;
    if (b > max) {
        max = b;
    }
    if (c > max) {
        max = c;
    }
    printf("%d\n", max);
    return 0;
}
```
