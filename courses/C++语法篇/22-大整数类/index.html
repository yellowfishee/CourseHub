➕ 加法实现

BigInteger BigInteger::operator+(BigInteger b) {
    string a = this->digits;
    string result = "";
    
    int carry = 0;
    for (int i = 0; i < max(a.size(), b.digits.size()); i++) {
        int da = i < a.size() ? a[a.size()-1-i] - '0' : 0;
        int db = i < b.digits.size() ? b.digits[b.digits.size()-1-i] - '0' : 0;
        int sum = da + db + carry;
        result = char(sum % 10 + '0') + result;
        carry = sum / 10;
    }
    return BigInteger(result);
}

🔍 比较大小

大整数比较的规则：

1. 先比较长度

"123" vs "9999" → 9999 更大（4位 > 3位）

2. 长度相同，比第一位

"123" vs "456" → 456 更大（1 < 4）

3. 第一位相同，继续比下一位

"123" vs "129" → 129 更大（3 < 9）

🔍 比较实现

bool BigInteger::operator<(BigInteger b) {
    if (digits.size() != b.digits.size())
        return digits.size() < b.digits.size();
    return digits < b.digits;  // 字典序比较
}

C++的string可以直接用<比较！
