## 编程题

### 简单题

**题目1：** 字符串数字相加

**描述：** 输入两个非负整数（可能很大，超出 int 范围），它们都是不超过 100 位的正整数。输出它们的和。

```
示例输入：12345678901234567890
        98765432109876543210
示例输出：111111111011111111100
```

**提示：** 用字符串存储大整数，从低位到高位逐位相加，注意进位

---

**题目2：** 字符串数字比大小

**描述：** 输入两个非负整数（可能很大），比较它们的大小。输出较大的那个数。

```
示例输入：9999999999999999999999
        1111111111111111111111
示例输出：9999999999999999999999
```

**提示：** 先比较长度，长度相同再逐位比较

---

**题目3：** 大整数减法

**描述：** 输入两个非负整数（不超过 100 位），a 减去 b，假设 a >= b。输出结果。

```
示例输入：1000000000000000000000
        1
示例输出：999999999999999999999
```

**提示：** 从低位到高位逐位相减，注意借位

---

### 中等题

**题目4：** 大整数乘法

**描述：** 输入两个非负整数（不超过 50 位），输出它们的乘积。

```
示例输入：123456789
        987654321
示例输出：121932631112635269
```

**提示：** 用数组模拟乘法竖式，结果数组的大小约为两个数长度之和

## 参考答案

### 编程题参考答案

```cpp
// 题目1：大整数相加
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    string a, b;
    cin >> a >> b;
    reverse(a.begin(), a.end());
    reverse(b.begin(), b.end());
    string result;
    int carry = 0;
    int maxLen = max(a.length(), b.length());
    for (int i = 0; i < maxLen; i++) {
        int da = i < a.length() ? a[i] - '0' : 0;
        int db = i < b.length() ? b[i] - '0' : 0;
        int sum = da + db + carry;
        result.push_back(sum % 10 + '0');
        carry = sum / 10;
    }
    if (carry > 0) {
        result.push_back(carry + '0');
    }
    reverse(result.begin(), result.end());
    cout << result << endl;
    return 0;
}
```

```cpp
// 题目3：大整数减法
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    string a, b;
    cin >> a >> b;
    reverse(a.begin(), a.end());
    reverse(b.begin(), b.end());
    string result;
    int borrow = 0;
    for (int i = 0; i < a.length(); i++) {
        int da = a[i] - '0';
        int db = i < b.length() ? b[i] - '0' : 0;
        int diff = da - db - borrow;
        if (diff < 0) {
            diff += 10;
            borrow = 1;
        } else {
            borrow = 0;
        }
        result.push_back(diff + '0');
    }
    while (result.length() > 1 && result.back() == '0') {
        result.pop_back();
    }
    reverse(result.begin(), result.end());
    cout << result << endl;
    return 0;
}
```

```cpp
// 题目4：大整数乘法
#include <iostream>
#include <string>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    string a, b;
    cin >> a >> b;
    reverse(a.begin(), a.end());
    reverse(b.begin(), b.end());
    vector<int> result(a.length() + b.length(), 0);
    for (int i = 0; i < a.length(); i++) {
        for (int j = 0; j < b.length(); j++) {
            result[i + j] += (a[i] - '0') * (b[j] - '0');
        }
    }
    int carry = 0;
    for (int i = 0; i < result.size(); i++) {
        int sum = result[i] + carry;
        result[i] = sum % 10;
        carry = sum / 10;
    }
    while (result.size() > 1 && result.back() == 0) {
        result.pop_back();
    }
    for (int i = result.size() - 1; i >= 0; i--) {
        cout << result[i];
    }
    cout << endl;
    return 0;
}
```
