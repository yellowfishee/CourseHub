## 内容

### 一、while循环的基本结构

**C++ 代码**：

```cpp
while (条件) {
    循环体;
}
```

**for循环对比**：

```cpp
// for循环
for (初始化; 条件; 调整) {
    循环体;
}

// 等价的 while 循环
初始化;
while (条件) {
    循环体;
    调整;
}
```

**生活例子**：
```
while (还没吃饱) {
    继续吃;
}
// 吃饱了，退出循环
```

### 二、例题：3n + 1 问题

**猜想**：对于任意大于1的自然数 n：
- 若 n 为奇数，则 n = 3n + 1
- 若 n 为偶数，则 n = n / 2
最终一定会变成 1。

**例子**：3 → 10 → 5 → 16 → 8 → 4 → 2 → 1

**代码**：

```cpp
#include <stdio.h>

int main() {
    int n, count = 0;
    scanf("%d", &n);
    while (n > 1) {
        if (n % 2 == 1) {
            n = n * 3 + 1;
        } else {
            n = n / 2;
        }
        count++;
    }
    printf("%d\n", count);
    return 0;
}
```

> ⚠️ **提示**：当需要统计某种事物的个数时，可以用一个变量来充当计数器。

### 三、do-while 循环

**特点**：循环体**至少执行一次**，然后再判断条件。

```cpp
do {
    循环体;
} while (条件);
```

**while vs do-while**：

| 循环类型 | 可能执行次数 | 适用场景 |
|----------|-------------|----------|
| while | 0次或多次 | 循环次数不确定，但可能一次都不执行 |
| do-while | 至少1次 | 循环次数不确定，但至少执行一次 |

**生活例子**：
```
do {
    吃一口;
} while (还没吃饱);
// 至少吃了一口
```

### 四、例题：近似计算

**问题**：计算 π/4 = 1 - 1/3 + 1/5 - 1/7 + ...，直到最后一项小于 10⁻⁶

**代码**：

```cpp
#include <stdio.h>

int main() {
    double sum = 0;
    int i = 0;
    do {
        double term = 1.0 / (i * 2 + 1);
        if (i % 2 == 0) {
            sum += term;
        } else {
            sum -= term;
        }
        i++;
    } while (term >= 1e-6);
    printf("%.6f\n", sum * 4);
    return 0;
}
```

> ⚠️ **提示**：do-while 循环的格式为 `do { 循环体 } while(条件);`，循环体至少执行一次。

### 五、break 和 continue

**break**：跳出整个循环

```cpp
while (1) {
    scanf("%d", &n);
    if (n < 0) break;  // 遇到负数，退出
}
```

**continue**：跳过本次循环，继续下一次

```cpp
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) continue;  // 跳过偶数
    printf("%d\n", i);  // 只打印奇数
}
```

---

### 六、while 和 for 的选择

| 场景 | 推荐使用 |
|------|----------|
| 已知循环次数（如 1 到 n） | for 循环 |
| 循环次数不确定（如直到条件满足） | while 循环 |
| 至少需要执行一次 | do-while 循环 |

> ⚠️ **提示**：for 循环的三个部分都是可以省略的。`for(;;)` 就是一个死循环。

---

### 七、常见错误

**错误 1**：忘了更新循环变量

```cpp
int i = 1;
while (i <= 10) {  // 错！i 永远不会增加，变成死循环
    printf("%d\n", i);
}
```

**错误 2**：混淆 `=` 和 `==`

```cpp
while (n = 1) {  // 错！这是赋值，不是比较
    // ...
}
while (n == 1) {  // 对！这是比较
    // ...
}
```

## 总结

| 概念 | C++ 写法 | 作用 |
|------|----------|------|
| while循环 | while (条件) { } | 当条件满足时重复执行 |
| do-while循环 | do { } while (条件); | 至少执行一次后判断 |
| 计数器 | count++ | 统计次数 |
| break | break; | 跳出循环 |
| continue | continue; | 跳过本次循环 |

**记住**：while 循环适合"直到..."类型的循环，for 循环适合"数圈"类型的循环。
