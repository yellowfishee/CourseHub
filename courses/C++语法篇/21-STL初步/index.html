🚗 vector - 动态数组

🚗➡️🚗🚗➡️🚗🚗🚗

什么是动态数组？

📚

就像一个会变大变小的书架

书多了就变大，书少了就变小

🏠 生活类比：神奇的书包

🎒

普通书包

只能装固定数量

→
🎒✨

vector书包

想装多少装多少！

普通数组：int a[5]; 固定5个

vector：vector<int> v; 想要几个要几个

vector的基本用法

// 1. 包含头文件
#include <vector>

// 2. 声明一个装整数的盒子
vector<int> v;

// 3. 往盒子里放东西
v.push_back(5);   // 放5
v.push_back(3);   // 放3
v.push_back(8);   // 放8

push_back = "推入后面" = 从后面加一个元素

小测验 1

v.push_back(10) 这行代码是什么意思？

A 从前面删一个元素
B 从后面加一个10
C 把10放到最前面
D 清空所有元素

vector的大小

vector<int> v;
v.push_back(5);
v.push_back(3);
v.push_back(8);

int n = v.size();  // n = 3
cout << n;        // 输出 3
📊

size() = 数一数盒子里有多少东西

动手练习：拖拽填空

创建vector并添加3个数字，正确输出是什么？

vector<int> v;
v.push_back(?);
v.push_back(?);
v.push_back(?);
cout << v.size();

把正确的数字拖到"?"处：

1
2
3
4

访问vector的元素

vector<int> v;
v.push_back(5);   // v[0] = 5
v.push_back(3);   // v[1] = 3
v.push_back(8);   // v[2] = 8

cout << v[0];  // 输出 5
cout << v[2];  // 输出 8

就像普通数组一样，用下标[ ]访问

注意：从0开始编号！

遍历vector

vector<int> v;
v.push_back(5);
v.push_back(3);
v.push_back(8);

// 用循环打印所有元素
for (int i = 0; i < v.size(); i++) {
    cout << v[i] << " ";
}
// 输出: 5 3 8

用size()知道要循环多少次

📝 string - 字符串

🔤

什么是字符串？

一串字符连在一起

H e l l o

= "Hello"

🏠 生活类比：火车车厢

🚃🚃🚃🚃

string就像一列火车

🚃 每节车厢 = 一个字符

🚂 整列火车 = 整个字符串

🔢 车厢编号 = 字符位置

➕ 可以加车厢 = 字符串拼接

string的基本用法

// 1. 包含头文件
#include <string>

// 2. 声明字符串
string s = "Hello";

// 3. 打印
cout << s;       // 输出 Hello

// 4. 获取长度
cout << s.length();  // 输出 5

length() = 数一数有多少个字符

字符串拼接

string s1 = "Hello";
string s2 = "World";

// 用 + 号拼接
string s3 = s1 + " " + s2;
cout << s3;  // 输出 Hello World
🔗

+ 号可以把字符串连在一起

访问字符串的字符

string s = "Hello";

cout << s[0];  // 输出 H
cout << s[1];  // 输出 e
cout << s[4];  // 输出 o
H[0] e[1] l[2] l[3] o[4]

和vector一样，用[下标]访问

🔑 迭代器 - Iterator

🔑

什么是迭代器？

就像一个遥控器，可以一个一个地遍历容器

begin() = 指向第一个元素

end() = 指向最后一个之后的"虚拟"位置

🏠 生活类比：点名

👨‍🏫

老师点名，从第一个人开始

👤 学生1

begin()

👤 学生2

👤 学生3

end()

end()指向"最后一个人之后"，是一个虚拟位置

迭代器的基本用法

vector<int> v = {5, 3, 8};

// 用迭代器遍历
for (auto it = v.begin(); it != v.end(); it++) {
    cout << *it << " ";
}
// 输出: 5 3 8

*it = 取出战车里的东西

auto = 自动推断类型

📏 sort - 排序算法

#include <algorithm>

vector<int> v = {3, 1, 4, 1, 5};

sort(v.begin(), v.end());

// 现在 v = {1, 1, 3, 4, 5}
📊

sort(开始位置, 结束位置) = 从小到大排序

小测验 4

sort(v.begin(), v.end()) 之后

v = {1, 2, 3, 4, 5}

原来v可能是？

A {5, 3, 1, 2, 4}
B {1, 2, 3, 4, 5}
C {5, 4, 3, 2, 1}
D {1, 1, 2, 3, 5}

📏 find - 查找算法

#include <algorithm>

vector<int> v = {3, 1, 4, 1, 5};

auto it = find(v.begin(), v.end(), 4);

if (it != v.end()) {
    cout << "找到了！";
} else {
    cout << "没找到";
}

find(开始, 结束, 要找的东西)

返回找到的位置，没找到就返回end()

📏 max和min

// 求最大值和最小值
int a = 5;
int b = 3;

int m = max(a, b);  // m = 5
int n = min(a, b);  // n = 3

cout << m << " " << n;  // 输出 5 3
🔼 max
🔽 min

📏 swap - 交换

int a = 5;
int b = 3;

swap(a, b);

cout << a << " " << b;  // 输出 3 5
a=5
⇄
b=3

swap = 换位置！
