## 编程题

### 简单题

**题目1：** vector基本操作

**描述：** 创建一个 vector，依次插入 5 个整数 10, 20, 30, 40, 50，然后输出第 3 个元素的值（下标从0开始）。

```
示例输出：30
```

**提示：** 用 push_back 添加元素，用下标访问

---

**题目2：** string基本操作

**描述：** 输入一个字符串，输出它的长度和第一个字符。

```
示例输入：Hello
示例输出：5
H
```

**提示：** string 用 .length() 获取长度，用 [0] 或 .front() 获取第一个字符

---

**题目3：** vector遍历

**描述：** 输入 n（n<=100），然后输入 n 个整数，存入 vector，逆序输出所有元素。

```
示例输入：5
        1 2 3 4 5
示例输出：5 4 3 2 1
```

**提示：** 用 for 循环从 n-1 遍历到 0

---

### 中等题

**题目4：** 字符串处理

**描述：** 输入一个字符串，把其中的大写字母改成小写字母，其他字符不变，然后输出。

```
示例输入：Hello World
示例输出：hello world
```

**提示：** 遍历字符串的每个字符，用 isupper() 判断大写，用 tolower() 转换

## 参考答案

### 编程题参考答案

```cpp
// 题目1：vector基本操作
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> v;
    v.push_back(10);
    v.push_back(20);
    v.push_back(30);
    v.push_back(40);
    v.push_back(50);
    cout << v[2] << endl;
    return 0;
}
```

```cpp
// 题目2：string基本操作
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s;
    cin >> s;
    cout << s.length() << endl;
    cout << s[0] << endl;
    return 0;
}
```

```cpp
// 题目3：vector遍历 - 逆序输出
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int n;
    cin >> n;
    vector<int> v(n);
    for (int i = 0; i < n; i++) {
        cin >> v[i];
    }
    for (int i = n - 1; i >= 0; i--) {
        cout << v[i] << " ";
    }
    cout << endl;
    return 0;
}
```

```cpp
// 题目4：字符串处理 - 大写转小写
#include <iostream>
#include <string>
#include <cctype>
using namespace std;

int main() {
    string s;
    getline(cin, s);
    for (int i = 0; i < s.length(); i++) {
        if (isupper(s[i])) {
            s[i] = tolower(s[i]);
        }
    }
    cout << s << endl;
    return 0;
}
```
