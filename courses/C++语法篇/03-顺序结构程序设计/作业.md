## 选择题

**1.** C++ 程序的执行顺序是：

A. 从下往上　　　B. 从上往下　　　C. 随机执行　　　D. 只执行第一行

---

**2.** `printf("%.3f", 3.1415926)` 的显示结果是：

A. 3.14　　　B. 3.142　　　C. 3.141　　　D. 3.1416

---

**3.** `printf("%05d", 42)` 的显示结果是：

A. 42　　　B. 00042　　　C. 42000　　　D. 42___（下划线）

---

**4.** 分解三位数 456 的百位、十位、个位，正确的是：

A. 百位=456/100, 十位=456/10, 个位=456%10

B. 百位=456/100, 十位=456/10%10, 个位=456%10

C. 百位=456%100, 十位=456%10, 个位=456/10

D. 百位=456/100, 十位=456%10, 个位=456/10

---

**5.** 交换变量 a 和 b 的值，正确的是：

A. `a = b, b = a`　　　B. `t = a; a = b; b = t`　　　C. `a = b`　　　D. 直接交换不需要变量

## 编程题

### 简单题

**题目1：** 三位数分解

**描述：** 输入一个三位数，分别输出百位、十位、个位，每行一个。

```
示例输入：
127
示例输出：
百位：1
十位：2
个位：7
```

---

**题目2：** 三位数反转

**描述：** 输入一个三位数，反转后输出。
例如：123 → 321

```
示例输入：
127
示例输出：
721
```

---

**题目3：** 华氏转摄氏

**描述：** 输入华氏温度 f，输出摄氏温度 c（保留2位小数）。
公式：c = 5 × (f - 32) / 9

```
示例输入：
100
示例输出：
37.78
```

---

### 中等题

**题目4：** 三位数各位求和

**描述：** 输入一个三位数，计算并输出各位数字之和。
例如：123 → 1 + 2 + 3 = 6

```
示例输入：
456
示例输出：
15
```

**提示：** 先分离出百位、十位、个位，再相加。

### 编程题参考答案

```cpp
// 题目1：三位数分解
#include <stdio.h>
int main() {
    int n;
    scanf("%d", &n);
    int bai = n / 100;
    int shi = n / 10 % 10;
    int ge = n % 10;
    printf("百位：%d\n", bai);
    printf("十位：%d\n", shi);
    printf("个位：%d\n", ge);
    return 0;
}

// 题目2：三位数反转
#include <stdio.h>
int main() {
    int n;
    scanf("%d", &n);
    int bai = n / 100;
    int shi = n / 10 % 10;
    int ge = n % 10;
    printf("%d%d%d\n", ge, shi, bai);
    return 0;
}

// 题目3：华氏转摄氏
#include <stdio.h>
int main() {
    double f, c;
    scanf("%lf", &f);
    c = 5.0 * (f - 32.0) / 9.0;
    printf("%.2f\n", c);
    return 0;
}

// 题目4：三位数各位求和
#include <stdio.h>
int main() {
    int n;
    scanf("%d", &n);
    int bai = n / 100;
    int shi = n / 10 % 10;
    int ge = n % 10;
    printf("%d\n", bai + shi + ge);
    return 0;
}
```
