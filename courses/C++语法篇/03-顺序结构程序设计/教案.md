## 内容

### 一、顺序执行

**C++ 代码的执行顺序**：

```cpp
int a = 5;      // 第1步
int b = 3;      // 第2步
int c = a + b;  // 第3步
printf("%d\n", c);  // 第4步
```

**执行顺序**：
```
从上往下，一步一步执行
第1步 → 第2步 → 第3步 → 第4步
```

---

### 二、格式化输出

**printf() 格式化符号**：

| 符号 | 含义 | 例子 |
|------|------|------|
| %d | 整数 | 5 |
| %.nf | 浮点数，保留n位小数 | %.3f → 3.142 |
| %0nd | 宽度n，不足前面补0 | %03d → 005 |

**例子**：

```cpp
printf("%d\n", 5);           // 5
printf("%.3f\n", 3.14159);   // 3.142
printf("%03d\n", 5);          // 005
```

---

### 三、三位数反转

**题目**：输入一个三位数，反转后输出

```cpp
int main() {
    int n, bai, shi, ge;
    scanf("%d", &n);
    bai = n / 100;
    shi = n / 10 % 10;
    ge = n % 10;
    printf("%d%d%d\n", ge, shi, bai);
    return 0;
}
```

**拆解**：
```
127 / 100 = 1（百位）
127 / 10 % 10 = 2（十位）
127 % 10 = 7（个位）
反转：721
```

---

### 四、变量交换复习

**三变量法**：

```cpp
int a = 3, b = 5, t;
t = a;    // t = 3
a = b;    // a = 5
b = t;    // b = 3
```

---

### 五、竞赛规则提醒

1. **不要加提示语**
   - ❌ `printf("请输入a:"); scanf("%d", &a);`
   - ✅ `scanf("%d", &a);`

2. **输出格式要严格**
   - 题目说保留3位小数，就用 `%.3f`

3. **每行末尾要有换行** `\n`

## 总结

|------|----------|------|
| 顺序执行 | 一行一行 | 从上往下依次执行 |
| 整数格式化 | %d | 输出整数 |
