🍳 做菜的比喻

做番茄炒蛋的步骤：

1 番茄洗一洗，切一切

2 鸡蛋打到碗里，搅拌

3 锅里倒油，炒一炒

4 出锅！

💡 顺序执行就像做菜！

一步一步来，上一步做完才做下一步

计算机程序也是这样运行的

➡️ 什么是顺序执行？

程序里的命令，从上往下，一条一条执行

// 第一步
int a = 10;

// 第二步
int b = 5;

// 第三步
printf("%d", a + b);

显示结果：15

🔄 变量可以重新赋值

变量里的值可以被覆盖

int x = 10;
printf("%d", x);  // 显示 10

x = 20;
printf("%d", x);  // 显示 20

💡 就像把盒子里的东西换掉！

x = 10 是"记住10"

x = 20 是"记住20"，覆盖了前面的

🔄🔄 交换两个变量的值

怎么把 a 和 b 的值互换？

交换前：

a = 3

b = 5

交换后：

a = 5

b = 3

🤔 直接写 a = b; 行不行？

❌ 错误的交换方法

int a = 3, b = 5;
a = b;  // a 变成了 5
b = a;  // b 还是 5，没有变！

结果：a = 5, b = 5

❌ 交换失败！3丢了！

✅ 正确的交换方法

需要一个"临时工"来帮忙

int a = 3, b = 5;
int temp = a;  // temp = 3
a = b;        // a = 5
b = temp;   // b = 3

结果：a = 5, b = 3 ✓

🎯 排序练习：交换步骤

把正确的顺序拖到框里

int a = 3, b = 5;
①;
②;
③;
b = temp
temp = a
a = b

🔢 三位数反转问题

问题：输入一个三位数，输出它的反转

比如：输入 123

输出：321

💡 思考：怎么把百位、十位、个位拆开？

这要用到除法和取余数！

📐 分解三位数

以 123 为例：

百位

1

123 / 100 = 1

十位

2

123 % 100 / 10 = 2

个位

3

123 % 10 = 3

🔄 反转公式

百位a，十位b，个位c 变成 cba

// 123 → 321
result = a * 100 + b * 10 + c
        = 1 * 100 + 2 * 10 + 3
        = 100 + 20 + 3
        = 321

📝 完整代码

#include <stdio.h>
int main() {
    int n, a, b, c, result;
    scanf("%d", &n);
    a = n / 100;
    b = n % 100 / 10;
    c = n % 10;
    result = c * 100 + b * 10 + a;
    printf("%d\n", result);
    return 0;
}

❌ 常见错误

✗ 忘记声明变量

a = 3;
printf("%d", a);

a 未声明！

✓ 先声明

int a;
a = 3;

✗ 用 printf 输入

printf("%d", &a);

scanf 才是输入！

✓ 用 scanf 输入

scanf("%d", &a);
