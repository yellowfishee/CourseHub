例题1-2　三位数反转

输入一个三位数，分离出百位、十位、个位，反转后输出

#include <stdio.h>
int main() {
    int n;
    scanf("%d", &n);
    printf("%d%d%d\n", n%10, n/10%10, n/100);
    return 0;
}

输入：127 → 输出：721

程序1-6　三位数反转（1）

// 分离百位、十位、个位
bai = n / 100; // 127/100 = 1
shi = n / 10 % 10; // 127/10%10 = 2
ge = n % 10; // 127%10 = 7

⚠️ 如果个位是0，如输入520，输出025还是25？

程序1-7　三位数反转（2）

把结果存到变量 m 中，再输出

#include <stdio.h>
int main() {
    int n, m;
    scanf("%d", &n);
    m = (n%10)*100 + (n/10%10)*10 + (n/100);
    printf("%03d\n", m);
    return 0;
}

输入：127 → 输出：721　　输入：520 → 输出：025

例题1-3　交换变量

输入两个整数 a 和 b，交换二者的值，然后输出

样例输入：824 16　　样例输出：16 824

程序1-8　变量交换（1）三变量法

int a, b, t;
scanf("%d%d", &a, &b);
t = a; // t = 3，把a的值备份到t
a = b; // a = 5，b的值赋给a
b = t; // b = 3，把t中备份的值赋给b
printf("%d %d\n", a, b);

类比：借一个空杯子交换酱油和醋

程序1-9　变量交换（2）加减法

a = a + b; // a = 824 + 16 = 840
b = a - b; // b = 840 - 16 = 824
a = a - b; // a = 840 - 824 = 16

⚠️ 这个方法适用范围窄，不推荐使用

程序1-10　变量交换（3）直接输出

int a, b;
scanf("%d%d", &a, &b);
printf("%d %d\n", b, a); // 直接调换顺序输出！
