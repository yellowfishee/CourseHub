## 内容

### 一、变量是什么？

**通俗解释**：
变量就像一个**盒子**，盒子上贴了个名字。盒子里可以放数据（数字），以后想用的时候，找到盒子拿就行了。

**生活例子**：
```
你想记住一个数字，比如"5"
1. 找一个盒子（声明变量）
2. 给盒子贴个名字，比如叫 "a"（变量名）
3. 把数字 5 放进盒子（赋值）
```

**C++ 代码**：

```cpp
int a;
a = 5;
printf("%d\n", a);
```

---

### 二、声明变量

**C++ 中声明变量的方式**：

```cpp
int a;        // 声明一个整数变量 a
double b;     // 声明一个浮点数变量 b
```

**常见数据类型**：

| 类型 | 含义 | 例子 |
|------|------|------|
| int | 整数 | 1, 100, -5 |
| double | 浮点数 | 3.14, -0.5 |

**注意**：变量必须先声明才能使用！

---

### 三、scanf() 读取输入

**问题**：怎么让用户输入一个数字，计算机记住？

**C++ 代码**：

```cpp
int a;
scanf("%d", &a);
```

**解释**：
- `scanf()` = 让计算机读取输入
- `%d` = 告诉计算机会读一个整数
- `&a` = 把读到的数存进变量 a
- **& 非常重要！不能漏掉！**

**完整例子**：

```cpp
#include <stdio.h>

int main() {
    int a, b;
    scanf("%d%d", &a, &b);
    printf("%d\n", a + b);
    return 0;
}
```

### 四、赋值运算的理解

**重要理解**：`=` 不是"等于"，是"赋值"

**错误理解**：
```cpp
a = 5;
a = a + 1;  // 5 = 5 + 1 ？？？
```

**正确理解**：
```cpp
a = 5;      // a 盒子里放 5
a = a + 1;  // 把 a+1 的结果（6）再放到 a 里
            // 原来的 5 被覆盖了
```

**特点**：变量是"喜新厌旧"的，新的值会覆盖旧的值。

---

### 五、变量交换

**问题**：如何交换两个变量的值？比如 a=3, b=5，变成 a=5, b=3？

**方法：三变量法**

```cpp
int a = 3, b = 5, t;
t = a;   // t = 3
a = b;   // a = 5
b = t;   // b = 3
```

**类比**：
```
借一个空杯子来交换酱油和醋：
1. 把酱油倒进空杯子（t = a）
2. 把醋倒进酱油瓶子（a = b）
3. 把酱油倒进醋瓶子（b = t）
```

### 六、综合例题

**例题**：计算圆柱体表面积

**公式**：表面积 = 2πr² + 2πrh

```cpp
#include <stdio.h>
#include <math.h>

int main() {
    double r, h, s;
    scanf("%lf%lf", &r, &h);
    double pi = acos(-1.0);
    s = 2 * pi * r * r + 2 * pi * r * h;
    printf("Area = %.3f\n", s);
    return 0;
}
```

## 总结

今天我们学了：

| 概念 | C++ 写法 | 作用 |
|------|----------|------|
| 声明整数 | int a; | 创建一个整数变量 |
| 声明浮点数 | double b; | 创建一个浮点数变量 |
| 输入 | scanf("%d", &a); | 读取用户输入 |
| 赋值 | a = 5; | 把值存进变量 |
| 变量交换 | t = a; a = b; b = t; | 交换两个变量 |

**记住**：变量就像便利贴，用完还可以贴新的东西上去！
