## 选择题

**1.** 在 C++ 中，声明整数变量正确的是：

A. `int a;`　　　B. `int a`　　　C. `integer a;`　　　D. `a int;`

---

**2.** `scanf("%d", &a)` 中 `&` 的作用是：

A. 表示 and　　　B. 取地址　　　C. 引用　　　D. 可以省略

---

A. 3　　　B. 2　　　C. 5　　　D. 0

---

**4.** `double` 类型用于存放：

A. 整数　　　B. 浮点数（小数）　　　C. 字符　　　D. 文字

---

**5.** 交换变量 a 和 b 的值，需要借助几个额外变量？

A. 0　　　B. 1　　　C. 2　　　D. 不需要

## 编程题

### 简单题

**题目1：** 输出平方

**描述：** 输入一个整数，输出它的平方。

```
示例输入：
5
示例输出：
25
```

---

**题目2：** 计算平均值

**描述：** 输入两个整数，输出它们的平均值（保留1位小数）。

```
示例输入：
3 7
示例输出：
5.0
```

---

**题目3：** 交换两个变量

**描述：** 输入两个整数 a 和 b，交换它们的值后输出。
提示：需要用一个临时变量来存放。

```
示例输入：
10 20
示例输出：
20 10
```

---

### 中等题

**题目4：** 三位数分解

**描述：** 输入一个三位数，分别输出百位、十位、个位。

```
示例输入：
127
示例输出：
百位=1, 十位=2, 个位=7
```

**提示：** 百位 = n / 100，十位 = n / 10 % 10，个位 = n % 10

---

## 参考答案

### 选择题答案

1. A, 2. B, 3. C, 4. B, 5. B

### 编程题参考答案

```cpp
// 题目1：输出平方
#include <stdio.h>
int main() {
    int a;
    scanf("%d", &a);
    printf("%d\n", a * a);
    return 0;
}

// 题目2：计算平均值
#include <stdio.h>
int main() {
    int a, b;
    scanf("%d%d", &a, &b);
    printf("%.1f\n", (a + b) / 2.0);
    return 0;
}

// 题目3：交换两个变量
#include <stdio.h>
int main() {
    int a, b, t;
    scanf("%d%d", &a, &b);
    t = a;
    a = b;
    b = t;
    printf("%d %d\n", a, b);
    return 0;
}

// 题目4：三位数分解
#include <stdio.h>
int main() {
    int n;
    scanf("%d", &n);
    int bai = n / 100;
    int shi = n / 10 % 10;
    int ge = n % 10;
    printf("百位=%d, 十位=%d, 个位=%d\n", bai, shi, ge);
    return 0;
}
```
