## 参考答案

**选择题：** 1-A, 2-B, 3-C, 4-B, 5-B

**填空题：**
6. 盒子/便利贴
7. 读取输入、输出
8. 赋值（把右边的值赋给左边）
9. 整数、浮点数
10. 临时存放/中转

**编程题（参考答案）：**

```cpp
// 题目1
#include <stdio.h>
int main() {
    int a;
    scanf("%d", &a);
    printf("%d\n", a * a);
    return 0;
}

// 题目2
#include <stdio.h>
int main() {
    int a, b;
    scanf("%d%d", &a, &b);
    printf("%d\n", a + b);
    return 0;
}

// 题目3
#include <stdio.h>
int main() {
    int a = 10, b = 20, t;
    t = a; a = b; b = t;
    printf("a = %d, b = %d\n", a, b);
    return 0;
}

// 题目4
#include <stdio.h>
int main() {
    int n, bai, shi, ge;
    scanf("%d", &n);
    bai = n / 100;
    shi = n / 10 % 10;
    ge = n % 10;
    printf("百位=%d, 十位=%d, 个位=%d\n", bai, shi, ge);
    return 0;
}

// 题目5
#include <stdio.h>
#include <math.h>
int main() {
    double r, h, area;
    scanf("%lf%lf", &r, &h);
    double pi = acos(-1.0);
    area = 2 * pi * r * r + 2 * pi * r * h;
    printf("Area = %.3f\n", area);
    return 0;
}
```
