⚠️ = 不是"等于"！

= 是"赋值"：把右边的值放进左边的变量

a = 5; // a 盒子放 5
a = a + 1; // 算出来 6，再放进 a

变量是"喜新厌旧"的，新的值会覆盖旧的值！

变量交换：三变量法

如何交换 a=3 和 b=5 的值？

int a = 3, b = 5, t;

t = a; // t = 3
a = b; // a = 5
b = t; // b = 3

就像用空杯子交换酱油和醋
