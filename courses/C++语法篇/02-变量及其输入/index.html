赋值运算的理解

⚠️ = 不是"等于"！

= 是"赋值"：把右边的值放进左边的变量

a = 5; // a 盒子放 5
a = a + 1; // 算出来 6，再放进 a

变量是"喜新厌旧"的，新的值会覆盖旧的值！

例题1-1　圆柱体的表面积

输入底面半径 r 和高 h，输出圆柱体的表面积，保留3位小数

样例输入

3.5 9

样例输出

Area = 274.889

程序1-5　圆柱体的表面积

#include <stdio.h>
#include <math.h>
int main() {
    const double pi = acos(-1.0);
    double r, h, s1, s2, s;
    scanf("%lf%lf", &r, &h);
    s1 = pi*r*r;
    s2 = 2*pi*r*h;
    s = s1*2.0 + s2;
    printf("Area = %.3f\n", s);
    return 0;
}

竞赛规则提醒

❌ 不要加提示语

printf("请输入a:") - 错误！

✓ 直接 scanf()

scanf("%d", &a); - 正确！

✓ 输出格式要严格

题目说保留3位小数，就用 %.3f

变量交换：三变量法

int a = 3, b = 5, t;
t = a; // t = 3
a = b; // a = 5
b = t; // b = 3
