🏪 储物柜的比喻

学校储物柜大家都用过吧？

• 每个柜子有一个 编号

• 柜子里可以 放东西

• 凭编号可以 取东西

💡 变量就像储物柜！

变量名 = 柜子编号

变量值 = 柜子里的东西

什么是变量？

变量就是计算机里的一个小盒子

我们可以给它起个名字

然后往里面放数字

int age = 10;

意思是：创建一个叫 age 的盒子，里面记住数字 10

📝 声明变量

使用变量前，要先"告诉"计算机会有这个变量

int age;

int = 告诉计算机这是一个整数

age = 给这个盒子起的名字

分号 ; 表示这句话说完了

✏️ 给变量赋值

用 = 让变量"记住"一个数字

age = 10;

注意：

这里的 = 不是等于的意思

是"记住"或者"放进"的意思

把数字 10 放进叫 age 的盒子里

⚡ 声明和赋值一起

可以一步到位

int age = 10;

int age;

先声明

age = 10;

再赋值

📊 常见数据类型

类型中文名能存什么例子
int整数整数1, 100, -5
float小数小数3.14, -0.5
double双精度更精确的小数3.14159

💡 先记住 int（整数）就够了！

🎯 使用变量

可以显示变量的值

int age = 10;
printf("%d", age);

计算机显示：10

⌨️ 什么是输入？

📝 到目前为止，我们写的数字都是"写死"的

如果我们想让用户自己输入数字呢？

这就是"输入"！

输入命令：scanf

printf = 显示出去

scanf = 读进来

📖 scanf 的用法

"请用户输入一个整数"

int age;
scanf("%d", &age);

"%d" = 告诉scanf要读一个整数

&age = 存到这个变量里

& 是"取地址"，记住就行

📝 完整例子

#include <stdio.h>
int main() {
    int age;
    scanf("%d", &age);
    printf("你的年龄是 %d\n", age);
    return 0;
}

运行效果：

用户输入 18

显示：你的年龄是 18

❌ 常见错误

✗ 忘记声明

age = 10;

报错：未定义的标识符

✓ 先声明

int age;
age = 10;

✗ 忘记 &

scanf("%d", age);

可能出错！

✓ 加上 &

scanf("%d", &age);

正确！
