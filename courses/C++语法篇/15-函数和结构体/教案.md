# 15-函数和结构体 - 教案

---

## 教学目标

1. 知识目标：掌握函数和结构体的概念和使用方法
2. 能力目标：能够定义和调用函数，能够声明和使用结构体
3. 情感目标：培养模块化编程的意识

## 教学重点

- 函数的定义和调用
- 结构体的声明和使用
- 结构体与函数的结合

## 教学难点

- 理解函数的抽象思维
- 结构体成员的访问方式

## 教学过程

### 导入（5分钟）

提问：什么是模块化？为什么需要函数？

### 新课讲解（30分钟）

**函数部分**
- 函数的概念：输入→处理→输出
- 函数的定义格式
- 函数的调用方式
- void类型的函数

**结构体部分**
- 结构体的概念：打包相关数据
- 结构体的定义
- 结构体成员的访问
- 结构体与函数

### 实践操作（10分钟）

学生动手编写函数和结构体代码。

### 课堂小结（5分钟）

总结函数和结构体的特点和使用方法。

---

## 提示与答案

### 提示

1. 函数名通常用动词或动词短语
2. 结构体成员用点号(.)访问
3. void表示没有返回值

### 参考答案

**选择题：**
1. C (int max(int a, int b))
2. B (s.age)

**编程题答案：**

```c
// 题目1：编写求和函数
int sum(int a, int b) {
    return a + b;
}

// 题目2：编写打印学生信息的函数
void printStudent(struct Student s) {
    printf("%s %d %.1f\n", s.name, s.age, s.score);
}

// 题目3：计算两点距离
double distance(struct Point p1, struct Point p2) {
    int dx = p1.x - p2.x;
    int dy = p1.y - p2.y;
    return sqrt(dx*dx + dy*dy);
}
```
