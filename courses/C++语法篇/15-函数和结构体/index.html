📦 函数是什么？

函数就像一个小型工厂：

📥

输入

→
🏭

函数

处理加工

→
📤

输出

💡 生活类比

就像微波炉：放入食物（输入），设置时间（参数），微波炉工作（函数执行），得到热腾腾的食物（输出）

📝 函数定义

// 函数定义的基本格式
返回类型 函数名(参数列表) {
  // 函数体：具体的处理步骤
  return 结果;
}

例：计算两个整数的最大值

int max(int a, int b) {
  if (a > b)
    return a;
  else
    return b;
}

💡 提示

函数名通常是动词或动词短语，如 max, sum, print 等

📞 函数调用

#include <stdio.h>

int max(int a, int b) {
  return a > b ? a : b;
}

int main() {
  int x = 10, y = 20;
  int m = max(x, y);  // 调用max函数
  printf("max = %d\n", m);
  return 0;
}

运行结果：max = 20

📵 无返回值的函数

如果函数不需要返回结果，用 void 表示返回类型：

void printHello() {
  printf("Hello!\n");
}

int main() {
  printHello();  // 调用，不需要接收返回值
  return 0;
}

💡 提示

void 表示"空"，即没有返回值

🏠 结构体是什么？

结构体就像一个自定义的"数据包"：

📋

学生信息

姓名: "小明"

年龄: 15

分数: 95.5

💡 生活类比

就像学校的成绩单：把姓名、年龄、分数等信息打包在一起

📝 结构体定义

struct Student {
  char name[50];  // 姓名
  int age;          // 年龄
  double score;      // 分数
};

声明结构体变量：

struct Student s1, s2;

🖊️ 访问结构体成员

使用点号(.)访问结构体的成员：

struct Student s1;

// 给成员赋值
strcpy(s1.name, "小明");
s1.age = 15;
s1.score = 95.5;

// 访问成员
printf("%s %d %.1f\n", s1.name, s1.age, s1.score);

输出：小明 15 95.5

🔧 结构体与函数

结构体作为函数的参数和返回值：

struct Point {
  int x, y;
};

// 结构体作为参数
int getDistance(struct Point p) {
  return p.x * p.x + p.y * p.y;
}

// 结构体作为返回值
struct Point move(struct Point p, int dx, int dy) {
  struct Point result = {p.x + dx, p.y + dy};
  return result;
}
