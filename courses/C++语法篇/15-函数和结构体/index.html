什么是结构体？

📋 生活类比：学生档案

一个学生有很多不同的信息

  • 👤 姓名（字符串）
  • 🎂 年龄（整数）
  • 📏 身高（小数）

结构体就是把这些不同类型的数据打包在一起！

定义结构体

struct Student {
    char name[20]; // 姓名
    int age; // 年龄
    float height; // 身高
};

struct 是关键字，Student 是类型名

使用结构体变量

// 定义结构体变量
struct Student stu;

// 给成员赋值
stu.age = 15;
stu.height = 1.65;
strcpy(stu.name, "Tom");

// 访问成员
printf("%s %d %.2f", stu.name, stu.age, stu.height);

练习：结构体赋值

以下代码输出什么？

struct Point {
  int x;
  int y;
};
struct Point p;
p.x = 10;
p.y = 20;
printf("%d,%d", p.x, p.y);
A 10,20
B 10,20（正确）
C x,y

函数返回结构体

// 返回一个Point结构体
struct Point createPoint(int x, int y) {
    struct Point p;
    p.x = x;
    p.y = y;
    return p;
}

// 使用
struct Point p1 = createPoint(3, 4);

函数与结构体：计算距离

// 计算两点距离
float distance(struct Point a, struct Point b) {
    int dx = a.x - b.x;
    int dy = a.y - b.y;
    return sqrt(dx*dx + dy*dy);
}

结构体可以作为函数的参数和返回值！

  • 用下标访问
