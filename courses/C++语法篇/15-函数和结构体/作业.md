## 编程题

### 简单题

**题目1：** 求最大值函数

**描述：** 编写一个函数 `max_of_two`，接收两个整数，返回较大的那个。在 main 函数中读入两个整数，调用该函数并输出结果。

**示例输入：**
```
5 8
```

**示例输出：**
```
8
```

---

**题目2：** 判断素数函数

**描述：** 编写一个函数 `is_prime`，接收一个整数n，如果n是素数返回1，否则返回0。在 main 函数中读入一个整数，调用该函数并输出 "YES" 或 "NO"。

**示例输入1：**
```
7
```

**示例输出1：**
```
YES
```

**示例输入2：**
```
8
```

**示例输出2：**
```
NO
```

**题目3：** 结构体：点坐标

**描述：** 定义一个结构体 `Point`，包含两个int成员 x 和 y。读入两个点的坐标，计算并输出它们的中点坐标（x1+x2）/2, (y1+y2)/2。

```
1 3
```

### 编程题参考答案

```cpp
// 题目1：求最大值函数
#include <stdio.h>
int max_of_two(int a, int b) {
    if (a > b) {
        return a;
    } else {
        return b;
    }
}
int main() {
    int x, y;
    scanf("%d %d", &x, &y);
    printf("%d\n", max_of_two(x, y));
    return 0;
}
```

```cpp
// 题目2：判断素数函数
#include <stdio.h>
int is_prime(int n) {
    if (n < 2) return 0;
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) return 0;
    }
    return 1;
}
int main() {
    int n;
    scanf("%d", &n);
    if (is_prime(n)) {
        printf("YES\n");
    } else {
        printf("NO\n");
    }
    return 0;
}
```

```cpp
// 题目3：结构体：点坐标
#include <stdio.h>
struct Point {
    int x;
    int y;
};
int main() {
    struct Point p1, p2;
    scanf("%d %d", &p1.x, &p1.y);
    scanf("%d %d", &p2.x, &p2.y);
    printf("%d %d\n", (p1.x + p2.x) / 2, (p1.y + p2.y) / 2);
    return 0;
}
```

```cpp
// 题目4：结构体数组排序
#include <stdio.h>
#include <string.h>
struct Student {
    char name[25];
    int score;
};
int main() {
    struct Student s[5];
    struct Student temp;
    for (int i = 0; i < 5; i++) {
        scanf("%s %d", s[i].name, &s[i].score);
    }
    for (int i = 0; i < 4; i++) {
        for (int j = i + 1; j < 5; j++) {
            if (s[i].score < s[j].score) {
                temp = s[i];
                s[i] = s[j];
                s[j] = temp;
            }
        }
    }
    for (int i = 0; i < 5; i++) {
        printf("%s %d\n", s[i].name, s[i].score);
    }
    return 0;
}
```
