## 编程题

### 简单题

**题目1：** 递归阶乘

**描述：** 编写递归函数 `factorial`，计算n的阶乘。在 main 函数中读入一个整数n（0≤n≤12），输出n!的值。

**示例输入：**
```
5
```

**示例输出：**
```
120
```

---

**题目2：** 递归斐波那契

**描述：** 编写递归函数 `fib`，计算斐波那契数列第n项（从0开始计数：fib(0)=0, fib(1)=1, fib(n)=fib(n-1)+fib(n-2)）。在 main 函数中读入n，输出fib(n)。

**示例输入：**
```
10
```

**示例输出：**
```
55
```

---

**题目3：** 递归求和

**描述：** 编写递归函数 `sum`，计算1+2+...+n的和。在 main 函数中读入n，输出sum(n)。

**示例输入：**
```
100
```

**示例输出：**
```
5050
```

### 中等题

**题目4：** 递归指数计算

**题目来源：** 自出

**描述：** 编写递归函数 `power`，计算 a 的 b 次幂（a^b），其中a和b都是非负整数，且a^b = a × a^(b-1)，a^0 = 1。

**示例输入：**
```
2 10
```

**示例输出：**
```
1024
```

**提示：** 注意处理b=0的情况。当b为0时应该返回1，否则返回 a * power(a, b-1)。

## 参考答案

### 选择题答案
1. B, 2. C, 3. C, 4. A, 5. B, 6. C

### 编程题参考答案

```cpp
// 题目1：递归阶乘
#include <stdio.h>
int factorial(int n) {
    if (n == 0 || n == 1) {
        return 1;
    }
    return n * factorial(n - 1);
}
int main() {
    int n;
    scanf("%d", &n);
    printf("%d\n", factorial(n));
    return 0;
}
```

```cpp
// 题目2：递归斐波那契
#include <stdio.h>
int fib(int n) {
    if (n == 0) return 0;
    if (n == 1) return 1;
    return fib(n - 1) + fib(n - 2);
}
int main() {
    int n;
    scanf("%d", &n);
    printf("%d\n", fib(n));
    return 0;
}
```

```cpp
// 题目3：递归求和
#include <stdio.h>
int sum(int n) {
    if (n == 1) {
        return 1;
    }
    return n + sum(n - 1);
}
int main() {
    int n;
    scanf("%d", &n);
    printf("%d\n", sum(n));
    return 0;
}
```

```cpp
// 题目4：递归指数计算
#include <stdio.h>
int power(int a, int b) {
    if (b == 0) {
        return 1;
    }
    return a * power(a, b - 1);
}
int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d\n", power(a, b));
    return 0;
}
```
