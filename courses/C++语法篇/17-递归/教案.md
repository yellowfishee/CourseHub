# 17-递归 - 教案

---

## 教学目标

1. 知识目标：理解递归的概念和原理
2. 能力目标：能够编写简单的递归函数
3. 情感目标：培养分解问题的思维方式

## 教学重点

- 递归的两个要素：终止条件和递归公式
- 递归调用栈的理解
- 递归与循环的关系

## 教学难点

- 递归终止条件的确定
- 递归调用过程的理解

## 教学过程

### 导入（5分钟）

用俄罗斯套娃引出递归的概念。

### 新课讲解（30分钟）

**递归概念**
- 函数调用自身
- 必须有终止条件
- 必须有递归公式

**递归示例**
- 阶乘：factorial(n) = n × factorial(n-1)
- 斐波那契数列
- 数字各位求和

**递归调用栈**
- 后进先出的调用管理
- 每一层调用都有独立的变量

### 实践操作（10分钟）

学生动手编写递归函数。

---

## 提示与答案

### 提示

1. 递归必须有终止条件，否则会无限递归
2. 递归公式将大问题分解为小问题
3. 可以用数学归纳法理解递归

### 参考答案

**编程题答案：**

```c
// 阶乘
int factorial(int n) {
    if(n == 1) return 1;
    return n * factorial(n-1);
}

// 斐波那契
int fib(int n) {
    if(n <= 2) return 1;
    return fib(n-1) + fib(n-2);
}

// 数字各位求和
int digitSum(int n) {
    if(n < 10) return n;
    return n % 10 + digitSum(n / 10);
}
```
