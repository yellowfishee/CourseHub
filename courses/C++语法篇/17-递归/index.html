🎯 递归的两个要素

✅

终止条件

递归结束的条件
（最小的娃娃）

🔁

递归公式

大问题分解为小问题
（打开一个娃娃）

💡 重要

没有终止条件的递归会无限调用，直到程序崩溃！

📝 例题：阶乘

n! = n × (n-1) × (n-2) × ... × 1

int factorial(int n) {
  if (n == 1) return 1;  // 终止条件
  return n * factorial(n - 1);  // 递归公式
}

factorial(5) = 5 × factorial(4)

= 5 × 4 × factorial(3) = 5 × 4 × 3 × 2 × factorial(1) = 120

🧠 递归调用栈

factorial(3) 的调用过程：

factorial(3) → 调用 factorial(2)
factorial(2) → 调用 factorial(1)
factorial(1) → 返回 1 (终止)
factorial(2) → 返回 2×1=2
factorial(3) → 返回 3×2=6

调用栈（后进先出）

factorial(1) - 返回值: 1
factorial(2) - 返回值: 2
factorial(3) - 返回值: 6

📝 例题：斐波那契数列

1, 1, 2, 3, 5, 8, 13, 21, ...

int fib(int n) {
  if (n <= 2) return 1;  // 终止条件
  return fib(n-1) + fib(n-2);  // 递归公式
}

💡 注意

斐波那契数列有两个终止条件：fib(1)=1, fib(2)=1

📝 例题：求和

计算 1+2+3+...+n 的和

int sum(int n) {
  if (n == 1) return 1;  // 终止条件
  return n + sum(n-1);  // 递归公式
}

sum(5) = 5 + sum(4) = 5 + 4 + 3 + 2 + 1 = 15
