for 循环是什么？🤔

📋

for 循环就像一个计数员
帮你数着"第1次、第2次、第3次..."
直到数到你想要的数量为止！ 

for (int i = 1; i <= 5; i++) {
    // 重复做的事情
}

for 循环的三部分 ⚙️

i++ 是什么意思？➕

🔢

i++ 等于 i = i + 1

每次循环结束后，i 的值增加 1
就像跑完一圈后，计圈器 +1 

i++   // i 的值 +1（加1）
i--   // i 的值 -1（减1）
i+=2  // i 的值 +2（加2）

第一个 for 循环 👶

数到 3，显示"第X次"

int main() {
    for (int i = 1; i <= 3; i++) {
        cout << "第" << i << "次" << endl;
    }
}

显示结果：

第1次

第2次

第3次

测验：这个循环显示几次？🧠

for (int i = 1; i <= 5; i++) {
    cout << "Hello";

会显示几次 "Hello"？

A 5 次
B 4 次
C 6 次

< 和 <= 有什么区别？⚖️

两种写法都能执行 5 次，但有点不同：

写法 1：i = 1 到 i <= 5

i = 1, 2, 3, 4, 5

共 5 次 ✓

写法 2：i = 0 到 i < 5

i = 0, 1, 2, 3, 4

共 5 次 ✓

💡 注意看清是从 0 还是 1 开始！

例题：1+2+3+...+100 📝

计算 1 到 100 的所有数字之和

int main() {
    int sum = 0;
    for (int i = 1; i <= 100; i++) {
        sum = sum + i;
    }
    cout << sum;
}

结果：5050

常见错误 ⚠️

❌ 错误1：忘了分号

for (int i = 1 i <= 10; i++)

三个部分之间必须用分号分隔！

❌ 错误2：分号放错位置

for (int i = 1; i <= 10; i++);

for 后面不能加分号！

❌ 错误3：死循环

for (int i = 1; i <= 10;)

i 永远不会变，永远停不下来！

测验：哪个会死循环？🧠

哪个循环会永远停不下来？

A

for (i=1; i<=10; i++)

正常循环 ✓

B

for (i=1; i<=10;)

i 不会变化 → 死循环！

C

for (i=10; i>=1; i--)

倒序循环 ✓

倒序循环：i-- ⬇️

⏰

有时候我们需要倒数：

for (int i = 5; i >= 1; i--) {
    cout << i << endl;
}

显示：5, 4, 3, 2, 1

倒计时发射！🚀

跳着数：i += 2 🦘

有时候不需要每次 +1，可以跳着数：

for (int i = 0; i < 10; i += 2) {
    cout << i << endl;
}

显示：0, 2, 4, 6, 8

每次 +2，偶数！

💡 i += 2 等于 i = i + 2

嵌套循环：循环里面还有循环 🔁🔁

📦

就像时钟：

当时针走 1 格，分针走完一圈（60格）
外层循环走 1 次，内层循环走 n 次 

for (int hour = 0; hour < 3; hour++) {
    for (int min = 0; min < 5; min++) {
        // 做事
    }
}

例题：打印乘法表 ✨

for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        cout << i << "x" << j << "=" << i*j;
    }
    cout << endl;
}

1x1=1 1x2=2 1x3=3

2x1=2 2x2=4 2x3=6

3x1=3 3x2=6 3x3=9
