## 内容

### 一、引入：什么是循环？

**生活例子**：
```
运动员跑操场：
一圈 → 一圈 → 一圈 → ... → 直到跑了10圈
```

**计算机中的循环**：
```
printf("Hello!\n");  → 第1次
printf("Hello!\n");  → 第2次
printf("Hello!\n");  → 第3次
...一直重复，直到完成任务
```

**类比**：
```
循环 = 跑圈
跑10圈 = 循环执行10次
```

---

### 二、for循环的基本结构

**C++ 代码**：

```cpp
for (int i = 1; i <= 10; i++) {
    printf("Hello!\n");
}
```

**拆解**：
```
for (初始化; 条件; 更新) {
    循环体（要重复执行的代码）
}
```

| 部分 | 例子 | 含义 |
|------|------|------|
| 初始化 | int i = 1 | 循环变量 i 从 1 开始 |
| 条件 | i <= 10 | i 小于等于 10 时继续 |
| 更新 | i++ | 每次循环后 i 加 1 |

---

### 三、循环变量 i

**i 是什么？**

i 是一个"计数器"，记录循环执行到第几次了。

```cpp
for (int i = 1; i <= 5; i++) {
    printf("第 %d 次\n", i);
}
```

**输出**：
```
第 1 次
第 2 次
第 3 次
第 4 次
第 5 次
```

---

### 四、i++ 的含义

**i++ 等价于 i = i + 1**

```cpp
i++  // i 的值增加 1
```

**生活类比**：
```
跑完一圈后，圈数 +1
i++ = 跑完一次循环后，计数器 +1
```

**逆向**：i-- 表示 i 的值减 1

---

### 五、循环范围

**从 1 开始，到 10 结束**：

```cpp
for (int i = 1; i <= 10; i++) {
    // 执行 10 次
}
```

**从 0 开始，到 9 结束**：

```cpp
for (int i = 0; i < 10; i++) {
    // 也执行 10 次
}
```

> ⚠️ **提示**：`i <= 10` 和 `i < 10` 的区别！两者都能执行10次，但起始值不同。

---

### 六、例题：求 1+2+3+...+100

**问题**：计算 1 + 2 + 3 + ... + 100 的和

**思路**：
```
1. 准备一个"盒子"存总和，初始为 0
2. 每次循环，把当前的 i 加到总和中
3. 循环结束后，输出总和
```

**代码**：

```cpp
#include <stdio.h>

int main() {
    int sum = 0;
    for (int i = 1; i <= 100; i++) {
        sum = sum + i;
    }
    printf("%d\n", sum);
    return 0;
}
```

**结果**：5050

### 七、循环的三个关键

1. **起始值**：从哪开始？
2. **结束条件**：到什么为止？
3. **更新方式**：怎么变化？

```
           开始
             ↓
    ┌─────────────────┐
    │  i = 1          │  ← 初始化
    └─────────────────┘
             ↓
    ┌─────────────────┐
    │  i <= 10 ?      │  ← 条件判断
    └─────────────────┘
         ↓ Yes
    ┌─────────────────┐
    │  执行循环体      │
    └─────────────────┘
             ↓
    ┌─────────────────┐
    │  i++            │  ← 更新
    └─────────────────┘
             ↓
         回到条件判断
```

### 八、常见错误

**错误 1**：忘了分号

```cpp
for (int i = 1 i <= 10; i++)  // 错！
for (int i = 1; i <= 10; i++)  // 对！
```

**错误 2**：分号位置错误

```cpp
for (int i = 1; i <= 10; i++);  // 错！多了一个分号
{
    printf("%d\n", i);  // 这段代码只会执行一次
}
```

> ⚠️ **提示**：循环体外面的大括号不需要分号，但 for 语句后面不能加分号！

**错误 3**：死循环

```cpp
for (int i = 1; i <= 10;) {  // 错！i 永远不会增加
    printf("%d\n", i);
}
```

> ⚠️ **提示**：检查循环的三个部分是否都正确：初始化、条件、更新。如果忘记更新，就会变成死循环。

### 九、竞赛中的常见问题

> ⚠️ **提示**：竞赛题目中要注意以下几点：
> 1. 看清循环范围，是从0开始还是从1开始
> 2. 看清循环条件，是 `<` 还是 `<=`
> 3. 注意输出格式，是否需要换行
> 4. 变量类型是否足够大（防止溢出）

## 总结

| 概念 | C++ 写法 | 作用 |
|------|----------|------|
| for循环 | for (初始化; 条件; 更新) { } | 重复执行代码 |
| 循环变量 | int i = 1 | 计数器从 1 开始 |
| 自增 | i++ | 每次循环后 +1 |
| 自减 | i-- | 每次循环后 -1 |

**记住**：循环就是"数圈"，数到目标圈数就停下来！
