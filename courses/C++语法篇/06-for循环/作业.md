## 编程题

### 简单题

**题目1：** 输出1到N

**描述：** 输入一个整数 N，输出 1 到 N 的所有整数，每个数后面跟一个空格。

```
示例输入：
5
示例输出：
1 2 3 4 5
```

---

**题目2：** 计算连续和

**描述：** 输入正整数 N，计算 1 + 2 + 3 + ... + N 的和并输出。

```
示例输入：
10
示例输出：
55
```

---

**题目3：** 输出偶数

**描述：** 输入正整数 N，输出 1 到 N 之间的所有偶数，每个数后面跟一个空格。

```
示例输入：
8
示例输出：
2 4 6 8
```

---

### 中等题

**题目4：** 计算阶乘

**描述：** 输入正整数 N，输出 N!（N的阶乘）。
阶乘：N! = 1 × 2 × 3 × ... × N
约定：0! = 1

```
示例输入：
5
示例输出：
120
```

**提示：** 用一个变量存放累积的结果，初始值设为1，然后循环乘以每个数。

## 参考答案

### 选择题答案

1. B, 2. B, 3. B, 4. B, 5. B, 6. C

### 编程题参考答案

```cpp
// 题目1：输出1到N
#include <stdio.h>
int main() {
    int n;
    scanf("%d", &n);
    for (int i = 1; i <= n; i++) {
        printf("%d ", i);
    }
    printf("\n");
    return 0;
}

// 题目2：计算连续和
#include <stdio.h>
int main() {
    int n, sum = 0;
    scanf("%d", &n);
    for (int i = 1; i <= n; i++) {
        sum += i;
    }
    printf("%d\n", sum);
    return 0;
}

// 题目3：输出偶数
#include <stdio.h>
int main() {
    int n;
    scanf("%d", &n);
    for (int i = 2; i <= n; i += 2) {
        printf("%d ", i);
    }
    printf("\n");
    return 0;
}

// 题目4：计算阶乘
#include <stdio.h>
int main() {
    int n;
    long long fact = 1;
    scanf("%d", &n);
    for (int i = 1; i <= n; i++) {
        fact *= i;
    }
    printf("%lld\n", fact);
    return 0;
}
```
