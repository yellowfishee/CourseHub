就像"上海自来水来自海上"这句话

正着读、反着读完全一样！

判断一个字符串是不是回文

回文判断：思路分析

✓ 判断方法

把字符串首尾对比

"ABA"

↑ ↑ → 相同

↓ → 相同

💻 关键代码

int isPalindrome(char s[]) {
  int len = strlen(s);
  for(int i = 0; i < len/2; i++) {
    if(s[i] != s[len-1-i])
      return 0;
  }
  return 1;
}

例题2：字符串反转

🔄 生活类比：翻卡片

把一排卡片从前往后翻到后面

比如："ABC" → "CBA"

就像照镜子时左右颠倒的效果

编写程序把输入的字符串反过来

字符串反转：代码实现

#include <stdio.h>
#include <string.h>
int main() {
    char s[100];
    gets(s);
    int len = strlen(s);
    for(int i = 0; i < len/2; i++) {
      // 交换s[i]和s[len-1-i]
      char temp = s[i];
      s[i] = s[len-1-i];
      s[len-1-i] = temp;
    }
    puts(s);
    return 0;
}

例题3：找最大值及其位置

🏆 生活类比：运动会比赛

在一群运动员中找到跑得最快的人

需要记住两件事：

  • 🏅 最好成绩是多少
  • 📍 第几号运动员

在数组中找出最大值和它的下标

找最大值：代码实现

int main() {
    int a[10], maxVal, maxIdx;
    for(int i = 0; i < 10; i++)
      scanf("%d", &a[i]);
    maxVal = a[0];
    maxIdx = 0;
    for(int i = 1; i < 10; i++) {
      if(a[i] > maxVal) {
        maxVal = a[i];
        maxIdx = i;
      }
    }
    printf("%d %d\n", maxVal, maxIdx);
}

例题4：数组元素循环移动

🔄 生活类比：旋转椅子

把一排座位上的每个人往右移动一个位置

最右边的人移动到最左边

就像"报数"游戏，最后一个人变成第一个

把数组中的元素向右循环移动一位

循环移动：代码实现

int main() {
    int a[100], n, temp;
    scanf("%d", &n);
    for(int i = 0; i < n; i++)
      scanf("%d", &a[i]);
    // 保存最后一个元素
    temp = a[n-1];
    // 从右往左移动
    for(int i = n-1; i > 0; i--)
      a[i] = a[i-1];
    // 第一个位置放原来的最后一个
    a[0] = temp;
}

练习：排列循环移动的步骤

把正确的执行顺序拖到框里

原始数组: [1, 2, 3, 4]

→ 移动后: [4, 1, 2, 3]

① 保存最后元素
② 从右往左移动
③ 第一个=保存的值

例题5：字符串连接

🔗 生活类比：接绳子

把两根绳子头尾接在一起

比如："Hello" + "World" = "HelloWorld"

就像把两句话连起来说

把两个字符串连接成一个

字符串连接：代码实现

#include <stdio.h>
#include <string.h>
int main() {
    char s1[100], s2[100];
    gets(s1); gets(s2);
    int len1 = strlen(s1);
    int i;
    for(i = 0; s2[i] != '\0'; i++) {
      s1[len1 + i] = s2[i];
    }
    s1[len1 + i] = '\0';
    puts(s1);
}

思考：字符串连接的关键是什么？

A 直接用 s1 + s2
B 找到s1的末尾，再逐个复制s2
C 把s2放在s1的最前面
