📝 题目一：单词逆序

题目描述

输入一个字符串（包含空格），将每个单词逆序输出。

示例输入：

hello world

示例输出：

olleh dlrow

💡 解题思路

逐字符读取，遇到空格就输出之前积累的字符（逆序）。

📝 题目一：代码实现

#include <stdio.h>
#include <string.h>

char s[1000];

int main() {
  fgets(s, sizeof(s), stdin);
  int len = strlen(s);
  int start = 0;
  for(int i = 0; i <= len; i++) {
    if(s[i] == ' ' || s[i] == '\0' || s[i] == '\n') {
      for(int j = i - 1; j >= start; j--)
        printf("%c", s[j]);
      if(s[i] == ' ') printf(" ");
      start = i + 1;
    }
  }
  printf("\n");
  return 0;
}

💡 关键点

用 start 记录每个单词的起始位置，遇到空格时逆序输出该单词。

📝 题目二：字母统计

题目描述

输入一个字符串，统计每个字母出现的次数（不区分大小写）。

示例输入：

Hello World

示例输出：

d:1 e:1 h:1 l:3 o:2 r:1 w:1

💡 解题思路

使用26个元素的数组统计每个字母的出现次数。

📝 题目二：代码实现

#include <stdio.h>
#include <ctype.h>

int cnt[26];

int main() {
  char c;
  while((c = getchar()) != '\n' && c != EOF) {
    if(isalpha(c)) {
      c = tolower(c);
      cnt[c - 'a']++;
    }
  }
  for(int i = 0; i < 26; i++) {
    if(cnt[i] > 0)
      printf("%c:%d ", i + 'a', cnt[i]);
  }
  printf("\n");
  return 0;
}

💡 关键点

用 isalpha 判断字母，tolower 转换大小写，cnt['a'-'a'] 即 cnt[0] 统计 'a' 的出现次数。

📝 题目三：螺旋矩阵

题目描述

输入n，输出n×n的螺旋矩阵，从1开始填数，顺时针旋转。

示例输入：

3

示例输出：

1
2
3
8
9
4
7
6
5

💡 解题思路

控制四条边：右→下→左→上，依次填数，每填完一条边边界收缩。

📝 题目三：代码实现

#include <stdio.h>
int a[20][20];

int main() {
  int n;
  scanf("%d", &n);
  int top = 0, bottom = n-1, left = 0, right = n-1;
  int num = 1;
  while(num <= n*n) {
    for(int j = left; j <= right; j++) a[top][j] = num++;
    top++;
    for(int i = top; i <= bottom; i++) a[i][right] = num++;
    right--;
    for(int j = right; j >= left; j--) a[bottom][j] = num++;
    bottom--;
    for(int i = bottom; i >= top; i--) a[i][left] = num++;
    left++;
  }
  for(int i = 0; i < n; i++) {
    for(int j = 0; j < n; j++)
      printf("%3d", a[i][j]);
    printf("\n");
  }
  return 0;
}
