# 13-竞赛题目选讲 - 教案

---

## 教学目标

1. 知识目标：综合运用数组和字符串知识解决实际问题
2. 能力目标：培养分析问题、设计算法的能力
3. 情感目标：激发对算法竞赛的兴趣

## 教学重点

- 题目理解与问题分解
- 数组和字符串的综合应用
- 代码调试技巧

## 教学难点

- 将文字描述转化为程序逻辑
- 边界条件的处理

## 教学过程

### 导入（5分钟）

回顾数组和字符串的知识，引出综合应用的重要性。

### 新课讲解（30分钟）

**题目一：单词逆序**
- 思路：逐字符读取，用start标记单词起始位置
- 关键点：遇到空格时逆序输出

**题目二：字母统计**
- 思路：26个元素的数组，按字母索引存储计数
- 关键点：大小写转换，isalpha和tolower函数

**题目三：螺旋矩阵**
- 思路：控制四条边，依次收缩边界
- 关键点：边界变量的更新

### 实践操作（10分钟）

学生动手调试代码，理解算法逻辑。

### 课堂小结（5分钟）

总结三道题目的共同特点：分析问题→设计算法→实现代码→调试优化。

## 提示与答案

### 提示

1. 单词逆序：使用start变量记录每个单词的起始位置
2. 字母统计：cnt['a'-'a'] 统计 'a' 的出现次数
3. 螺旋矩阵：top++, bottom--, left++, right-- 收缩边界

### 参考答案

**题目一：**
```c
#include <stdio.h>
#include <string.h>
char s[1000];

int main() {
    fgets(s, sizeof(s), stdin);
    int len = strlen(s);
    int start = 0;
    for(int i = 0; i <= len; i++) {
        if(s[i] == ' ' || s[i] == '\0' || s[i] == '\n') {
            for(int j = i - 1; j >= start; j--)
                printf("%c", s[j]);
            if(s[i] == ' ') printf(" ");
            start = i + 1;
        }
    }
    printf("\n");
    return 0;
}
```

**题目二：**
```c
#include <stdio.h>
#include <ctype.h>
int cnt[26];

int main() {
    char c;
    while((c = getchar()) != '\n' && c != EOF) {
        if(isalpha(c)) {
            c = tolower(c);
            cnt[c - 'a']++;
        }
    }
    for(int i = 0; i < 26; i++) {
        if(cnt[i] > 0)
            printf("%c:%d ", i + 'a', cnt[i]);
    }
    printf("\n");
    return 0;
}
```

**题目三：**
```c
#include <stdio.h>
int a[20][20];

int main() {
    int n;
    scanf("%d", &n);
    int top = 0, bottom = n-1, left = 0, right = n-1;
    int num = 1;
    while(num <= n*n) {
        for(int j = left; j <= right; j++) a[top][j] = num++;
        top++;
        for(int i = top; i <= bottom; i++) a[i][right] = num++;
        right--;
        for(int j = right; j >= left; j--) a[bottom][j] = num++;
        bottom--;
        for(int i = bottom; i >= top; i--) a[i][left] = num++;
        left++;
    }
    for(int i = 0; i < n; i++) {
        for(int j = 0; j < n; j++)
            printf("%3d", a[i][j]);
        printf("\n");
    }
    return 0;
}
```
