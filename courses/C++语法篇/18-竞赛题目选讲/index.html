📝 题目一：数字各位求和

📋 题目描述

输入一个正整数，输出各位数字之和。

示例输入：

12345

示例输出：

15

🔍 数字各位求和 - 思路分析

💡 生活类比：数钱

有一堆零钱：1元、2元、5元、10元...

要算总数？一个个数，加起来！

对于数字 12345：

最后一位 = 12345 % 10 = 5

前面的 = 12345 / 10 = 1234

digitSum(12345) = 5 + digitSum(1234)

📐 数字各位求和 - 递归公式

递归公式

digitSum(n) = n%10 + digitSum(n/10)

终止条件

digitSum(n) = n（当 n < 10 时）

因为一位数就是它自己！

💻 数字各位求和 - 代码实现

int digitSum(int n) {
    if (n < 10) { // 终止条件：一位数
        return n;
    }
    return n % 10 + digitSum(n / 10);
}

验证：digitSum(12345)

= 5 + digitSum(1234)

= 5 + 4 + digitSum(123)

= 5 + 4 + 3 + digitSum(12)

= 5 + 4 + 3 + 2 + digitSum(1)

= 5 + 4 + 3 + 2 + 1 = 15 ✓

🎯 练习：digitSum(789) = ?

按照递归公式算一算：

digitSum(789) = 9 + digitSum(78)

= 9 + 8 + digitSum(7)

= 9 + 8 + 7

A 7
B 20
C 24
D 15

📝 题目二：计算幂次

📋 题目描述

计算 a 的 n 次方（aⁿ）。

示例输入：

2 10

示例输出：

1024

📐 幂次计算 - 递归公式

💡 生活类比：翻倍

2⁴ = 2 × 2 × 2 × 2

翻4次，每次都乘以2

递归公式

power(a, n) = a × power(a, n-1)

终止条件

power(a, 0) = 1

💻 幂次计算 - 代码实现

double power(double a, int n) {
    if (n == 0) { // 终止条件
        return 1;
    }
    return a * power(a, n - 1);
}

验证：power(2, 10)

= 2¹⁰ = 1024 ✓

🎯 练习：power(3, 4) = ?

按照递归公式：

power(3, 4) = 3 × power(3, 3)

= 3 × 3 × power(3, 2)

= 3 × 3 × 3 × power(3, 1)

= 3 × 3 × 3 × 3 × power(3, 0)

= 3 × 3 × 3 × 3 × 1

A 64
B 81
C 27
D 12

📝 题目三：判断回文数

📋 题目描述

判断一个正整数是否是回文数（正读反读都一样）。

示例输入：

12321

示例输出：

YES

示例输入：

12345

示例输出：

NO

🔍 回文数 - 生活类比

💡 什么是回文？

上海自来水来自海上

山西运煤运西山

12321 正看反看都一样！

判断方法：

比较第一位和最后一位

如果相同，去掉这两位继续判断

12321 → 比较 1 和 1 → 232 → 比较 2 和 2 → 3

💻 回文数 - 代码实现

int getWidth(int n) { // 计算位数
    int len = 0;
    do { len++; n /= 10; } while (n > 0);
    return len;
}

bool isPalindrome(int n) {
    int w = getWidth(n);
    if (w == 1) return true;
    int left = n / pow(10, w-1);
    int right = n % 10;
    if (left != right) return false;
    return isPalindrome(n % pow(10, w-1) / 10);
}

📝 题目四：最大公约数

📋 题目描述

求两个正整数的最大公约数（GCD）。

示例输入：

12 18

示例输出：

6

📐 最大公约数 - 欧几里得算法

💡 生活类比：找共同的朋友

你和朋友有一堆糖果

你的糖果能整除朋友的糖果吗？

如果有剩余，继续这个过程！

公式

gcd(a, b) = gcd(b, a % b)

终止条件

gcd(a, 0) = a

💻 最大公约数 - 代码实现

int gcd(int a, int b) {
    if (b == 0) { // 终止条件
        return a;
    }
    return gcd(b, a % b);
}

验证：gcd(12, 18)

= gcd(18, 12%18=12)

= gcd(12, 18%12=6)

= gcd(6, 12%6=0)

= 6 ✓

🎯 练习：gcd(24, 36) = ?

按照欧几里得算法：

gcd(24, 36) = gcd(36, 24)

= gcd(36, 24) = gcd(24, 12)

= gcd(24, 12) = gcd(12, 0)

A 6
B 8
C 12

🏛️ 汉诺塔 - 再次挑战

💡 三步曲

第1步：把上面的 n-1 个盘子从 A 借助 C 移到 B

第2步：把最底下的大盘子从 A 直接移到 C

第3步：把 B 上的 n-1 个盘子从 B 借助 A 移到 C

3个盘子需要 2³-1 = 7 步

A→B, A→C, B→C, A→B, C→A, C→B, A→B

💻 汉诺塔 - 代码实现

void hanoi(int n, char from, char to, char temp) {
    if (n == 0) return; // 终止条件
    hanoi(n-1, from, temp, to); // 第1步
    printf("%c→%c ", from, to);
    hanoi(n-1, temp, to, from); // 第3步
}

调用 hanoi(3, 'A', 'C', 'B') 输出：

A→B A→C B→C A→B C→A C→B A→B

2⁴ = 16

🔄 递归 vs 循环 总结

🔁 循环

适合简单重复

执行效率高

digitSum 用循环简单

🪆 递归

适合分治问题

代码简洁优雅

hanoi 非递归莫属

💡 选择技巧：能用循环的优先用循环，复杂问题再用递归
