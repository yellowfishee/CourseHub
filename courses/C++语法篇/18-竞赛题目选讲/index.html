📝 题目一：数字各位求和

题目描述

输入一个正整数，输出各位数字之和。

示例输入：

12345

示例输出：

15

💡 思路：递归

sum(n) = n%10 + sum(n/10)

📝 题目一：代码实现

#include <stdio.h>

int digitSum(int n) {
  if (n < 10) return n;  // 终止条件：一位数
  return n % 10 + digitSum(n / 10);
}

int main() {
  int n;
  scanf("%d", &n);
  printf("%d\n", digitSum(n));
  return 0;
}

digitSum(12345)

= 5 + digitSum(1234) = 5 + 4 + 3 + 2 + 1 = 15

📝 题目二：计算幂次

题目描述

计算 a 的 n 次方（a^n）。

示例输入：

2 10

示例输出：

1024

💡 思路：递归

pow(a,n) = a × pow(a,n-1)，pow(a,0) = 1

📝 题目二：代码实现

#include <stdio.h>

double power(double a, int n) {
  if (n == 0) return 1;  // 终止条件
  return a * power(a, n - 1);
}

int main() {
  double a; int n;
  scanf("%lf %d", &a, &n);
  printf("%.0f\n", power(a, n));
  return 0;
}

power(2, 10)

= 2 × power(2,9) = 2^10 = 1024

📝 题目三：汉诺塔

题目描述

将n个盘子从A柱移动到C柱，规则：每次移动一个盘子，大盘子不能放在小盘子上。

示例输入：

3

示例输出：

A->C A->B C->B A->C B->A B->C A->C

💡 经典递归问题

汉诺塔的移动次数是 2^n - 1

📝 题目三：代码实现

#include <stdio.h>

void hanoi(int n, char from, char to, char temp) {
  if (n == 0) return;
  hanoi(n-1, from, temp, to);
  printf("%c->%c ", from, to);
  hanoi(n-1, temp, to, from);
}

int main() {
  int n;
  scanf("%d", &n);
  hanoi(n, 'A', 'C', 'B');
  printf("\n");
  return 0;
}
