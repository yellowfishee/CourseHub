# 18-竞赛题目选讲 - 教案

---

## 教学目标

1. 知识目标：综合运用函数和递归解决问题
2. 能力目标：分析问题、设计递归算法的能力
3. 情感目标：提高对算法竞赛的兴趣

## 教学重点

- 问题分析和分解
- 递归函数的编写
- 边界条件处理

## 教学难点

- 将文字描述转化为递归公式
- 边界条件的确定

## 教学过程

### 导入（5分钟）

回顾递归的两个要素。

### 新课讲解（30分钟）

**题目一：数字各位求和**
- 递归公式：sum(n) = n%10 + sum(n/10)
- 终止条件：n < 10

**题目二：计算幂次**
- 递归公式：power(a,n) = a × power(a,n-1)
- 终止条件：n == 0

**题目三：汉诺塔**
- 经典递归问题
- 三步法：移动n-1个、移动最大、移动n-1个

### 实践操作（10分钟）

学生理解汉诺塔的递归过程。

---

## 提示与答案

### 提示

1. 递归的关键是找到终止条件和递归公式
2. 汉诺塔的移动次数是 2^n - 1
3. 递归问题都可以用数学归纳法验证

### 参考答案

**编程题答案：**

```c
// 数字各位求和
int digitSum(int n) {
    if(n < 10) return n;
    return n % 10 + digitSum(n / 10);
}

// 计算幂次
double power(double a, int n) {
    if(n == 0) return 1;
    return a * power(a, n - 1);
}

// 汉诺塔
void hanoi(int n, char from, char to, char temp) {
    if(n == 0) return;
    hanoi(n-1, from, temp, to);
    printf("%c->%c ", from, to);
    hanoi(n-1, temp, to, from);
}
```
