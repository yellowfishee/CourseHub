**描述：** 编写一个函数 `is_leap_year`，接收一个年份，如果是闰年返回1，否则返回0。闰年规则：能被4整除但不能被100整除，或者能被400整除。在 main 函数中读入一个年份，调用该函数并输出 "YES" 或 "NO"。

```
2000
```

### 中等题

**题目4：** 递归：求最大公约数

**题目来源：** 自出

**描述：** 编写递归函数 `gcd`，使用辗转相除法求两个正整数a和b的最大公约数。辗转相除法：如果a除以b余数为r，则 gcd(a,b) = gcd(b,r)，当b为0时返回a。

**示例输入：**
```
36 24
```

**示例输出：**
```
12
```

**提示：** 
- 辗转相除法：gcd(a, b) = gcd(b, a % b)，当 b == 0 时返回 a
- 例如 gcd(36, 24) = gcd(24, 12) = gcd(12, 0) = 12

## 参考答案

### 选择题答案
1. B, 2. C, 3. C, 4. B, 5. B, 6. B

### 编程题参考答案

```cpp
// 题目1：函数：判断闰年
#include <stdio.h>
int is_leap_year(int year) {
    if ((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)) {
        return 1;
    }
    return 0;
}
int main() {
    int year;
    scanf("%d", &year);
    if (is_leap_year(year)) {
        printf("YES\n");
    } else {
        printf("NO\n");
    }
    return 0;
}
```

```cpp
// 题目2：递归：数字各位求和
#include <stdio.h>
int digit_sum(int n) {
    if (n < 10) {
        return n;
    }
    return n % 10 + digit_sum(n / 10);
}
int main() {
    int n;
    scanf("%d", &n);
    printf("%d\n", digit_sum(n));
    return 0;
}
```

```cpp
// 题目3：函数：回文判断
#include <stdio.h>
#include <string.h>
int is_palindrome(char s[]) {
    int len = strlen(s);
    for (int i = 0; i < len / 2; i++) {
        if (s[i] != s[len - 1 - i]) {
            return 0;
        }
    }
    return 1;
}
int main() {
    char s[105];
    scanf("%s", s);
    if (is_palindrome(s)) {
        printf("YES\n");
    } else {
        printf("NO\n");
    }
    return 0;
}
```

```cpp
// 题目4：递归：求最大公约数
#include <stdio.h>
int gcd(int a, int b) {
    if (b == 0) {
        return a;
    }
    return gcd(b, a % b);
}
int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d\n", gcd(a, b));
    return 0;
}
```
