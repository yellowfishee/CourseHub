## 参考答案

**选择题：** 1-B, 2-C, 3-C, 4-B

**填空题：**

8. 相同类型的更小

**编程题答案：**

```c
// 题目1：回文数判断
int isPalindrome(int n) {
    char s[20];
    sprintf(s, "%d", n);
    int len = strlen(s);
    for(int i = 0; i < len/2; i++)
        if(s[i] != s[len-1-i]) return 0;
    return 1;
}

// 题目2：斐波那契数列
int fib(int n) {
    if(n <= 2) return 1;
    return fib(n-1) + fib(n-2);
}

// 题目3：最大公约数
int gcd(int a, int b) {
    if(b == 0) return a;
    return gcd(b, a % b);
}
```
