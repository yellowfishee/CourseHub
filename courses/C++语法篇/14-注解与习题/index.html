什么是数组？

🏠 生活类比：学生宿舍

想象一排编好号的储物柜

每个柜子都能放东西

可以通过编号快速找到柜子

数组就是一排编好号的"盒子"！

数组的声明

// 声明一个能放5个整数的数组
int a[5];

int

数据类型

a

数组名字

[5]

5个格子

数组的下标

🔢 生活类比：电影院座位

电影院座位从1号开始编号

但C++数组从0开始编号！

a[0]

第1个元素

a[1]

第2个元素

a[2]

第3个元素

...

以此类推

数组的使用

int a[5];

a[0] = 10; // 第1个格子放10
a[1] = 20; // 第2个格子放20
a[2] = 30; // 第3个格子放30

printf("%d", a[1]); // 输出20

练习：数组赋值

以下代码输出什么？

int b[4];
b[0] = 5;
b[1] = b[0] * 2;
b[2] = b[1] + 3;
printf("%d", b[2]);
A 5
B 13
C 10

字符数组（字符串）

🔤 生活类比：字母饼干

字符数组就像一排字母饼干

每个格子里放一个字符

最后一个位置必须是'\0'表示结束

char name[] = "Tom"; // 实际是 {'T','o','m','\0'}

字符数组的输入输出

char s[100];

// 输入字符串
gets(s); // 读取一行
// 或者
scanf("%s", s); // 读取一个单词

// 输出字符串
puts(s); // 输出并换行
// 或者
printf("%s", s);

常见错误

✗ 下标越界

int a[5];
a[5] = 10; // 错误！最大是a[4]

✗ 忘记\0

char s[4];
s[0]='H'; s[1]='i';
// 忘记加 '\0' 结束符

思考：哪个下标是错误的？

数组 int arr[5]; 哪个访问是错的？

A arr[0]
B arr[4]
C arr[5]
D arr[3]

strlen() - 求字符串长度

📏 生活类比：量绳子

strlen()就像量绳子的长度

从绳子头到结束标记的长度

char s[] = "hello";
int len = strlen(s); // len = 5

strcmp() - 比较字符串

⚖️ 生活类比：比身高

strcmp()比较两个字符串

就像比谁更高、更矮、一样高

strcmp(s1, s2)
// 返回值：0=相等，>0=s1更大，<0=s2更大

strcpy() - 复制字符串

📋 生活类比：复印文件

strcpy()复制字符串

就像复印一份一模一样的

注意：目标字符串要足够大！

strcpy(dest, src); // 把src复制到dest

练习：字符串函数

以下代码输出什么？

char a[] = "cat";
char b[] = "dog";
printf("%d", strcmp(a, b));
A 正数
B 负数（c < d）
C 0

二维数组

📊 生活类比：电影院座位

电影院座位有行和列

比如第3排第5座

二维数组就是这样的表格

int seat[5][10]; // 5行10列的座位
seat[2][4] = 1; // 第3行第5列（从0开始）

练习：二维数组访问

数组 int m[3][2]; 最大的合法下标是？

A m[3][2]
B m[2][1]
C m[2][2]
D m[0][1]

本节知识总结

数组

  • 一排编好号的盒子
  • 下标从0开始
  • 注意不要越界

字符数组

  • 存放字符的数组
  • 以'\0'结尾
  • 可以用gets/scanf输入

字符串函数

  • strlen() - 求长度
  • strcmp() - 比较
  • strcpy() - 复制

二维数组

  • 像表格一样的结构
  • 两个下标：行和列
  • 注意行列下标范围
