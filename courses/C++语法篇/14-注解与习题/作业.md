## 编程题

### 简单题

**题目1：** 字符串拷贝

**描述：** 读入一个字符串（长度不超过100），使用 `strcpy` 将它拷贝到另一个数组中，然后输出拷贝后的字符串。

**示例输入：**
```
Hello World
```

**示例输出：**
```
Hello World
```

---

**题目2：** 字符串连接

**描述：** 读入两个字符串，用 `strcat` 将第二个字符串连接到第一个字符串后面并输出。

**示例输入：**
```
Hello 
World
```

**示例输出：**
```
HelloWorld
```

---

**题目3：** 字符串比较

**描述：** 读入两个字符串，如果它们相等则输出 "YES"，否则输出 "NO"。

**示例输入1：**
```
abc
abc
```

**示例输出1：**
```
YES
```

**示例输入2：**
```
abc
def
```

**示例输出2：**
```
NO
```

---

### 中等题

**题目4：** 字符串数组排序

**题目来源：** 自出

**描述：** 读入5个字符串（每个长度不超过50），将它们按字典序排序后输出。

**示例输入：**
```
banana
apple
cherry
date
berry
```

**示例输出：**
```
apple
banana
berry
cherry
date
```

**提示：** 可以用 strcmp 比较字符串，用 strcpy 交换字符串。

## 参考答案

### 选择题答案
1. B, 2. C, 3. C, 4. D, 5. B, 6. C

### 编程题参考答案

```cpp
// 题目1：字符串拷贝
#include <stdio.h>
#include <string.h>
int main() {
    char s1[105];
    char s2[105];
    scanf("%s", s1);
    strcpy(s2, s1);
    printf("%s\n", s2);
    return 0;
}
```

```cpp
// 题目2：字符串连接
#include <stdio.h>
#include <string.h>
int main() {
    char s1[105];
    char s2[105];
    scanf("%s", s1);
    scanf("%s", s2);
    strcat(s1, s2);
    printf("%s\n", s1);
    return 0;
}
```

```cpp
// 题目3：字符串比较
#include <stdio.h>
#include <string.h>
int main() {
    char s1[105];
    char s2[105];
    scanf("%s", s1);
    scanf("%s", s2);
    if (strcmp(s1, s2) == 0) {
        printf("YES\n");
    } else {
        printf("NO\n");
    }
    return 0;
}
```

```cpp
// 题目4：字符串数组排序
#include <stdio.h>
#include <string.h>
int main() {
    char s[5][55];
    char temp[55];
    for (int i = 0; i < 5; i++) {
        scanf("%s", s[i]);
    }
    for (int i = 0; i < 4; i++) {
        for (int j = i + 1; j < 5; j++) {
            if (strcmp(s[i], s[j]) > 0) {
                strcpy(temp, s[i]);
                strcpy(s[i], s[j]);
                strcpy(s[j], temp);
            }
        }
    }
    for (int i = 0; i < 5; i++) {
        printf("%s\n", s[i]);
    }
    return 0;
}
```
