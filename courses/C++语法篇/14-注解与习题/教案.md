## 教学目标

1. 知识目标：掌握数组和字符串的核心概念
2. 能力目标：能够正确使用数组和字符串函数
3. 情感目标：培养认真细致的编程习惯

## 教学重点

- 数组下标的正确使用
- 字符串函数的正确使用
- gets和fgets的区别

## 教学难点

- 理解字符串结束符 '\0' 的作用
- 避免缓冲区溢出

**数组部分**
- 下标从0开始，最大为size-1

**字符串部分**
- 字符串以'\0'结尾
- strlen不包含结束符
- fgets比gets更安全

## 提示与答案

### 提示

1. 数组下标从0开始，a[10]表示第11个元素
2. 字符串结束符'\0'占用一个字符位置
3. fgets可以指定读取的字符数上限

### 参考答案

**选择题：**
1. C (int a[100])
2. B (5)
3. B (fgets可以指定读取的字符数)
4. C (3×4=12)

**编程题答案：**

```c
// 题目1：数组元素交换
#include <stdio.h>
int main() {
    int a[10] = {3,1,4,1,5,9,2,6,5,3};
    int maxIdx = 0;
    for(int i = 1; i < 10; i++)
        if(a[i] > a[maxIdx]) maxIdx = i;
    int temp = a[0];
    a[0] = a[maxIdx];
    a[maxIdx] = temp;
    return 0;
}

// 题目2：字符串连接
#include <stdio.h>
#include <string.h>
int main() {
    char s1[100] = "Hello";
    char s2[100] = "World";
    strcat(s1, s2);
    printf("%s\n", s1);
    return 0;
}
```
