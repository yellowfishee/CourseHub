# 程序设计的目标和流程

## 打开ide
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "I love cpp" << endl;
    return 0;
}
```

我们来详细的拆分一下，这个程序是什么意思。

首先在devcpp中打开文件，输入代码并运行后，我们会发现，弹出了一个窗口，窗口中显示了我们输入的代码。也就是 i love cpp

我们来看第一行代码，这是一个头文件，用于包含iostream库，用于输入输出。
```cpp
#include <iostream>
using namespace std;
```

i o stream 是 input output stream 的缩写，名字是输入输出流，
这个流非常的形象，我们可以把数据想象成河流一样，数据具备一个基本的流向，也就是，人输入数据到电脑，电脑处理数据，最后输出数据到人。

第二行，using namespace std;

这一行的作用是，把程序中使用到的std::cout和std::endl，都直接使用，而不需要写std::。

我们来尝试这么一段代码

```cpp
#include <iostream>
int main() {
    cout << "I love cpp" << endl;
    return 0;
}
```

这个时候你会发现，程序报错了，因为我们cout的全称是std::cout，而我们没有使用std::，所以程序报错了。

那么如果我们加上 std::cout，程序就会正常运行了。

所以我们有两种方法，第一种是使用命名空间，第二种是使用全称。


那么，为什么输入输出一定要使用iostream呢。

这是因为标准输入输出的功能，是定义在iostream库中的。
我们可以把iostream想象成一个工具包，里面包含了输入和输出的功能。如果我们不使用iostream，那么我们就不能使用输入和输出的功能。

请删除掉 #include <iostream>, 看看程序会报错吗？

我们可以这么理解，新建的程序是一张空白的白纸，我们#include <iostream> ，就想当于把iostream这个文件的内容，提前复制到了我们的程序中，所以我们才可以使用。

就像你找朋友借铅笔一样，你先要和朋友说，我需要一个铅笔，然后朋友就会给你一个铅笔。如果没有给你，你是使用不了的。

## 变量
什么是变量，为什么我们需要变量？

在程序的计算中，经常需要存储一些数据。

比如说：我们需要统计一个公交站台这一天的人流量。那么这个人流量就是变量的名字，有多少人通过这个站台，就是变量的值。

所以，当我们需要一个长期存储的数据时，我们就需要使用变量。或者说，只要是需要在程序中使用到的数据，我们就需要使用变量。

那么怎么创建一个变量呢？

在创建变量之前我们先来看看。

身高：1.88
体重：70
性别：男

我们看这三个变量，有哪三种类型？

整数，小数和字

我们一个变量能不能同时存储多个数据？
答案：不能。

存身高的变量，能不能同时存储整数？
答案：不能。

很显然，也不能。

所以我们的变量有一个类型。

也就是说，不同的杯子，要装不同的东西。


我们来看前面阶段要学习到的数据类型：

整数：int
小数：float

首先，我们需要先告诉电脑，我们需要一个什么类型的变量。

int 

然后我们要告诉电脑，这个变量要叫什么名字

int height;

那么在创建这个变量的时候，我们可以同时告诉电脑，这个变量的初始值是多少。

int height = 1.88;

也可以在创建变量的时候，不告诉初始值，然后在程序中创建

int height;
height = 1.88;

。

float是同样的道理。

小测试: 猜一猜下面的代码会输出什么？

```cpp
float a = 1.88;
int b = 1.88;
cout << a << endl;
cout << b << endl;
```


## 算法小题目
先来看第一个题目

例 2-1 跑步。小A的跑步速度为5m/s,八尾勇跑步的速度为8m/s,八尾勇在小A后面100m,他们同时起跑。请问:需要多长时间八尾勇可以追上小A?输出一个数字表示答案。

分析:八尾勇和小A之间的相对速度是(8-5)m/s=3m/s。也就是说,每秒钟他们的相对距离就会缩短 3m,因此100/3就是答案。

最开始定义了v_a、v_yao、distance 这3个int类型的变量并赋初始值。然后定义了delta、ans 这两个 double类型的变量,但是没有给它们赋初始值。double类型的意思是双精度浮点数，

设想一下,指定了一个变量碗,但是这个碗是之前程序用过的然后被废弃,里面不知道有什么奇怪的东西,如果直接拿来用搞不好就会食物中毒。因此,在使用这个变量前,必须给这个变量赋值,进行初始化。

正常情况下,赋值符号的右边的数据类型(整数或者是浮点数)应该和左边的变量数据类型相匹配,但是有些情况也可以不一致。正如程序第6行,赋值右边是两个整数的差,结果还是整数;但是左边的变量类型是浮点数,所以右边的整数类型会被自动转换成左边的浮点数的类型。相反,如果右边是浮点数类型,左边是整数类型,C++在自动转换的时候会舍去浮点数的小数部分,这说明跨类型的自动转换可能会造成一些精度损失。计算答案ans的时候,distance 是整数类型,delta也是浮点数类型,所以它们相除会变为浮点数类型,刚好可以存在ans变量里。这种现象被称为自动类型转换。还有另外一种写法可以不使用delta变量,就像注释行那样。因为右边的变量全部都是整数,所以需要乘上1.0将distance变成一个浮点数;否则,计算机就将其当作整除来处理造成答案错误。
