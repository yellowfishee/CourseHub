核心原则

🎯 1. 交互优先（最重要！）

学生必须参与，不能只是看！每个知识点都要设计交互环节：

拖拽题 → 选择题 → 填空题 → 排序题
设计检查清单：
  • • 这个知识点有交互吗？
  • • 学生能动手操作吗？
  • • 操作后有即时反馈吗？
  • • 有得分/连胜等激励机制吗？
📝 2. 语言通俗易懂
应该用的词 避免用的词
命令 函数（除非解释清楚）
显示出来 / 打印 输出、打印
让计算机记住 赋值、存储
拆解问题 拆分问题
📋 3. 每页都要有的元素
大问题/标题 → 生活类比 → 代码示例 → 交互练习 → 一句话总结

学习追踪

得分、连胜等激励机制

⭐ 得分: 0
🔥 连胜: 0

答对题目得分，连续答对增加连胜数

const learningData = {
    score: 0,
    streak: 0
};

function updateScore(points) {
    learningData.score += points;
    document.getElementById('scoreDisplay').textContent = learningData.score;
}

function updateStreak() {
    learningData.streak++;
    if (learningData.streak >= 3) {
        showBadge('连胜 ' + learningData.streak);
    }
}

function resetStreak() {
    learningData.streak = 0;
}
