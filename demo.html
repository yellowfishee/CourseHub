🔄 变量交换：如何交换两个变量的值？

现在 a = 3, b = 5，如何变成 a = 5, b = 3？

3

a

↔️
5

b

-

t（临时）

t = a; // t = 3
a = b; // a = 5
b = t; // b = 3
🎉

交换完成！

a = 5, b = 3

💡 为什么需要临时变量？

🍶

想象你有两个杯子 a（酱油）和 b（醋），想把它们的内容交换。

你需要一个空杯子 t 先把 a 的东西倒出来。

t = a（备份）→ a = b（覆盖）→ b = t（恢复）
