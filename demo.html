🔄 变量交换

现在 a = 3, b = 5，如何变成 a = 5, b = 3？

3

a

↔️
5

b

-

t（临时）

t = a; // t = 3
a = b; // a = 5
b = t; // b = 3
🎉

交换完成！

a = 5, b = 3
