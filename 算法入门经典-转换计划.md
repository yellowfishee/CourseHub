# 算法入门经典 - 转换计划

## 目标
- 原书：《算法入门经典》
- 面向：小学生、初中生
- 语言：Python + C++
- 格式：每个二级标题 = 主课程，三级标题归入主课程
- 要求：保留书中所有内容，不遗漏
