# 教案生成规范

## 模板

```markdown
# 课时名称

## 教学目标
- 目标1
- 目标2

## 内容

### 引入
生活例子/问题引入

### 知识点1
- 概念
- 代码示例

### 知识点2
- 概念
- 代码示例

## 常见错误
- ❌ 错误 → 正确

## 练习
1. 练习1
2. 练习2
```

## 语言规范

| 用这个词 | 不用 |
|---------|------|
| 显示 | 输出 |
| 记住/存放 | 存储 |
| 读入 | 输入 |
| 计算 | 处理 |

## 生活类比库

| 概念 | 类比 |
|------|------|
| 程序 | 食谱 |
| 变量 | 便利贴 |
| 循环 | 重复做操 |
| 判断 | 岔路口选择 |
| 函数 | 微波炉 |

## 输出路径
`courses/课程名/课时名/教案.md`
