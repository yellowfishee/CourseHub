PPT 生成规范

面向零基础编程学习者的交互式幻灯片制作指南

🎉 开始学习
左侧导航可切换不同章节，点击各页面卡片查看详细说明和交互演示。

这是什么？

📚 CourseHub PPT 规范

这是一套为 零基础编程教学 设计的 PPT 制作规范。 不同于传统幻灯片，这里的每一页都可以互动展示， 支持键盘翻页、逐步显示动画，让学生在课堂上跟随老师的节奏一步步理解知识。

用户触发式动画 键盘导航 生活类比 问题拆解

核心特性

🎯 通俗易懂

避免专业术语，用生活类比解释每个概念。 如：变量 = 便利贴、函数 = 遥控器按钮。

✨ 逐步显示

每页内容分步展示，点击才出现下一个元素， 让学生跟得上老师的讲解节奏。

⌨️ 键盘控制

空格/右箭头显示下一元素或翻页， 左箭头回退。全程无需鼠标。

🎨 深色主题

投影仪友好，亮色文字 + 深色背景， 配合模糊光斑动画，现代感十足。

快速开始

1. 创建 HTML 文件
courses/课程名/课时名/index.html
2. 引入 TailwindCSS
<script src="https://cdn.tailwindcss.com"></script>
3. 编写幻灯片页面
<div class="slide active">
nbsp; <h1>标题</h1>
  <p>副标题</p>
</div>

核心原则

面向零基础，每页都要问自己这三个问题

灵魂三问

🤔 能看懂吗？

零基础的人能看懂这页吗？ 如果有术语，有解释清楚吗？

🌰 有例子吗？

有没有用生活例子解释？ 用表情/图片辅助理解了吗？

📖 术语解释了吗？

有没有专业词汇没解释？ 避免让学生产生困惑。

问题拆解流程

🔄 贯穿整个 PPT 的流程
遇到问题
↓
拆解问题（拆成小问题）
↓
再拆解（直到能用一个命令解决）
↓
✅ 写出代码解决

拆解示例

问题：让计算机显示 "Hello"
步骤思考结果
1显示在哪里？→ 屏幕
2怎么让屏幕显示？→ 用一个"命令"
3什么命令？→ print() = "显示出来"
答案print("Hello")

完整模板

可复制使用的完整 HTML PPT 模板

<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>课时名称</title>
  <script src="https://cdn.tailwindcss.com"></script>
  <style>
    body { font-family: 'Inter', sans-serif; background: #0a0a0a; overflow: hidden; }
    .slide { position: absolute; inset: 0; display: flex; flex-direction: column; justify-content: center; align-items: center; padding: 4rem; opacity: 0; transition: all 0.5s ease; }
    .slide.active { opacity: 1; }
    .animate-item { opacity: 0; transform: translateY(20px); transition: opacity 0.3s ease; }
    .animate-item.visible { opacity: 1; transform: translateY(0); }
  </style>
</head>
<body>
  <div id="slides-container">
    <div class="slide active">
      <h1 class="text-5xl font-bold text-white">标题</h1>
      <p class="text-xl text-gray-400">副标题</p>
    </div>
  </div>
  <div class="fixed bottom-0 left-0 h-1 bg-gradient-to-r from-blue-500 to-purple-500" id="progress"></div>
  <script>
    let current = 0;
    const slides = document.querySelectorAll('.slide');
    document.addEventListener('keydown', (e) => {
      if (e.key === 'ArrowRight' || e.key === ' ') next();
      if (e.key === 'ArrowLeft') prev();
    });
    function next() { if (current < slides.length - 1) { slides[current].classList.remove('active'); slides[++current].classList.add('active'); } }
    function prev() { if (current > 0) { slides[current].classList.remove('active'); slides[--current].classList.add('active'); } }
  </script>
</body>
</html>
