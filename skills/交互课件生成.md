# 交互课件生成规范

## 什么是交互课件？

交互课件是增强版 PPT，支持：
- **拖拽操作** — 可视化赋值、交换等概念
- **代码执行可视化** — 逐行高亮执行过程
- **随堂测验** — 即时反馈的选择题
- **编程练习** — 代码输入与验证
- **学习追踪** — 分数、连胜、进度

## 触发方式

- 命令: `/生成交互课件 课时名`
- 对话: "帮我生成一个交互式课件讲变量赋值"
- 对话: "生成一个可以拖拽操作的课件"

## 适用场景

推荐对以下知识点生成交互课件：
- 变量赋值与交换
- if/else 条件判断
- for/while 循环
- 数组遍历与操作
- 函数调用与返回值

对于简单概念，可用标准 PPT；对于需要动手操作的知识点，用交互课件。

## 核心原则

### 1. 语言通俗易懂

面向零基础初学者，用生活类比解释每个概念。

### 2. 交互驱动学习

通过拖拽、点击、输入等操作，让学生在"做"中学。

### 3. 即时反馈

每个操作都有即时反馈（正确/错误提示 + 动画效果）。

## 交互元素设计

### 1. 拖拽赋值（变量赋值概念）

```html
<!-- 拖拽源：数值方块 -->
<div class="var-box" draggable="true" data-value="5">
    <span class="text-2xl">5</span>
    <span class="text-xs">数值</span>
</div>

<!-- 放置槽：变量 -->
<div class="slot" data-var="a">
    <span class="text-gray-400">a</span>
    <span class="text-xs text-gray-500">变量</span>
</div>
```

```javascript
// 拖拽逻辑
slot.addEventListener('dragover', e => {
    slot.classList.add('highlight');
});

slot.addEventListener('drop', e => {
    const value = e.dataTransfer.getData('text/plain');
    slot.innerHTML = `<span class="text-2xl">${value}</span><span class="text-xs">${value}</span>`;
    slot.classList.add('filled');
});
```

### 2. 代码执行可视化

```html
<div class="code-block">
    <div class="code-line" data-line="1">
        <span class="code-keyword">int</span> a = <span class="code-number">5</span>;
    </div>
    <div class="code-line" data-line="2">
        a = a + <span class="code-number">1</span>;
    </div>
</div>
```

```javascript
// 逐行高亮执行
const codeLines = document.querySelectorAll('.code-line');
codeLines.forEach((line, i) => {
    setTimeout(() => {
        line.classList.add('executed');
    }, i * 1000);
});
```

### 3. 随堂测验

```html
<div class="quiz-option" data-correct="true">
    <span class="letter">A</span>
    <span>6</span>
</div>
<div class="quiz-option" data-correct="false">
    <span class="letter">B</span>
    <span>5</span>
</div>
```

```javascript
// 测验逻辑
quizOptions.forEach(option => {
    option.addEventListener('click', () => {
        if (option.dataset.correct === 'true') {
            option.classList.add('correct');
            showParticles(option);
            updateScore(10);
        } else {
            option.classList.add('wrong');
        }
    });
});
```

### 4. 编程练习

```html
<div class="code-input" id="codeInput" contenteditable="true" spellcheck="false">
    <!-- 用户输入代码 -->
</div>
<button class="btn btn-primary" onclick="checkCode()">验证</button>
```

```javascript
function checkCode() {
    const code = codeInput.innerText.trim();
    const expected = 'a = 5;';
    if (code === expected) {
        codeInput.classList.add('correct');
        updateScore(20);
    } else {
        codeInput.classList.add('wrong');
    }
}
```

## 学习追踪系统

```javascript
const learningData = {
    score: 0,
    streak: 0,
    completedSteps: [],
    quizResults: []
};

function updateScore(points) {
    learningData.score += points;
    document.getElementById('scoreDisplay').textContent = learningData.score;
}

function updateStreak() {
    learningData.streak++;
    document.getElementById('streakDisplay').textContent = learningData.streak;
    if (learningData.streak >= 3) {
        showBadge('连胜达成');
    }
}

function resetStreak() {
    learningData.streak = 0;
    document.getElementById('streakDisplay').textContent = '0';
}
```

## 动画效果库

```css
/* 浮动动画 */
@keyframes float { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-6px); } }
.animate-float { animation: float 3s ease-in-out infinite; }

/* 弹入动画 */
@keyframes popIn { 0% { transform: scale(0.8); opacity: 0; } 100% { transform: scale(1); opacity: 1; } }
.animate-pop { animation: popIn 0.3s ease; }

/* 上滑动画 */
@keyframes slideUp { 0% { transform: translateY(10px); opacity: 0; } 100% { transform: translateY(0); opacity: 1; } }
.animate-slide { animation: slideUp 0.3s ease; }

/* 粒子效果 */
.particle {
    position: fixed;
    pointer-events: none;
    font-size: 1.25rem;
    animation: confetti 0.8s ease forwards;
}
@keyframes confetti {
    0% { opacity: 1; transform: scale(1); }
    100% { opacity: 0; transform: scale(2) translateY(-50px); }
}
```

```javascript
// 创建粒子效果
function createParticles(element) {
    const rect = element.getBoundingClientRect();
    const emojis = ['⭐', '✨', '🎉', '💯'];
    for (let i = 0; i < 6; i++) {
        const particle = document.createElement('div');
        particle.className = 'particle';
        particle.textContent = emojis[Math.floor(Math.random() * emojis.length)];
        particle.style.left = rect.left + rect.width / 2 + 'px';
        particle.style.top = rect.top + 'px';
        document.body.appendChild(particle);
        setTimeout(() => particle.remove(), 800);
    }
}
```

## 典型交互流程

```
步骤1: 概念引入（卡片展示 + 生活类比）
    ↓ 点击/空格
步骤2: 拖拽赋值（拖动数值方块到变量槽）
    ↓ 正确完成
步骤3: 代码执行（逐行高亮 + 变量变化动画）
    ↓ 点击"下一步"
步骤4: 变量交换（三变量法演示）
    ↓ 点击
步骤5: 随堂测验（选择题，即时反馈）
    ↓
步骤6: 编程练习（输入代码，验证正确性）
    ↓
步骤7: 总结报告（学习数据回顾）
```

| 项目 | 规范 |

## 技术栈

- TailwindCSS（CDN）
- 原生 HTML5 Drag and Drop API
- 原生 JavaScript
- CSS Animations
- 无需 Vue/React，保持轻量

---

## 文件输出

输出路径: `courses/课程名/课时名/index.html`

注意：交互课件和标准 PPT 输出到同一文件，通过内容区分。生成时根据知识点类型决定是否加入交互元素。
